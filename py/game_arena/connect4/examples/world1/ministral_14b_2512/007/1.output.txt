### Strategy for this Policy

The strategy for this policy is based on a **hybrid approach** combining **minimax with alpha-beta pruning** (for depth-limited lookahead) and **pattern-based heuristics** (to evaluate board positions efficiently). Here's the breakdown:

1. **Immediate Win Check**: First, the policy checks if there's a move that would immediately win the game (i.e., create a Connect 4 line). If so, it prioritizes that move.
2. **Block Opponent's Win**: If no immediate win exists, the policy checks if the opponent can win in their next move. If so, it blocks that move.
3. **Heuristic-Based Lookahead**: If neither an immediate win nor a forced block exists, the policy uses a **minimax with alpha-beta pruning** approach to evaluate the best move up to a certain depth (e.g., 5 plies). The heuristic function assigns scores based on:
   - **Potential lines**: Counts how many of your discs are in sequences of 2, 3, or 4 (horizontally, vertically, or diagonally).
   - **Opponent's potential lines**: Similarly counts the opponent's sequences but penalizes them.
   - **Center control**: Discs in the center columns (3-4) are weighted more heavily.
   - **Column height**: Preferentially selects columns with higher empty cells (to keep options open).
   - **Mobility**: Favors moves that leave the opponent with fewer options (i.e., fewer empty columns).
4. **Random Tiebreaker**: If multiple moves have the same heuristic score, the policy randomly selects one to avoid predictability.
5. **Fallback**: If the minimax search is too slow (due to depth or branching), the policy defaults to a **simple heuristic** (e.g., prioritizing center columns and potential lines).

This approach balances **strategic depth** (via minimax) with **computational efficiency** (via heuristics), ensuring the policy can react quickly to opponent threats while still planning ahead for optimal moves.

def find_immediate_win(board: np.ndarray) -> int:

def drop_disc(board: np.ndarray, col: int, player: int) -> np.ndarray:

def check_win(board: np.ndarray, player: int) -> bool:

    # Check horizontal
    for row in range(rows):
        for col in range(cols - 3):
            if all(board[row][col + i] == player for i in range(4)):
                return True

    # Check vertical
    for row in range(rows - 3):

def minimax(board: np.ndarray, depth: int, alpha: float, beta: float, maximizing_player: bool) -> int:
    """Minimax with alpha-beta pruning to find the best move."""
    if depth == 0 or check_win(board, 1) or check_win(board, -1):
        return None

    legal_moves = get_legal_moves(board)
    if not legal_moves:
        return None

    if maximizing_player:
        best_score = -float('inf')
        best_move = None
        for move in legal_moves:
            new_board = drop_disc(board, move, 1)
            score = evaluate_board(new_board)
            if score > best_score:
                best_score = score
                best_move = move
            # Alpha-beta pruning
            if best_score >= beta:
                return best_move
            alpha = max(alpha, best_score)
        return best_move
    else:
        best_score = float('inf')
        best_move = None
        for move in legal_moves:
            new_board = drop_disc(board, move, -1)
            score = evaluate_board(new_board)
            if score < best_score:
                best_score = score
                best_move = move
            # Alpha-beta pruning
            if best_score <= alpha:
                return best_move
            beta = min(beta, best_score)
        return best_move

def simple_heuristic(board: np.ndarray, legal_moves: List[int]) -> int:
    """Fallback heuristic: prioritizes center columns and potential lines."""

    scores = []

    for col in legal_moves:
        # Simulate dropping a disc in this column
        new_board = drop_disc(board, col, 1)
        score = evaluate_potential(new_board, 1) + center_weight[col]
        scores.append(score)

    # Randomly select among top moves to avoid predictability
    max_score = max(scores)
    best_moves = [i for i, s in enumerate(scores) if s == max_score]
    return random.choice(best_moves)
