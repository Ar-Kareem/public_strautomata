To solve this problem, we need to create a policy function for a Connect Four game that can make optimal moves to win or prevent the opponent from winning. The function should prioritize moves that either lead to an immediate win or block the opponent's potential winning moves, and if no such moves are available, it should use a heuristic to make the best possible move.

### Approach
1. **Immediate Win Check**: First, check all possible columns to see if placing a disc there would result in a win. If such a move exists, return it immediately.
2. **Block Opponent's Win Check**: If there are no immediate winning moves, check each possible move to see if it prevents the opponent from winning in their next turn. If there are such moves, collect them and choose the best one.
3. **Heuristic Selection**: If no blocking moves are necessary, evaluate the remaining moves using a heuristic that prioritizes central columns, which are generally more advantageous in Connect Four.

### Solution Code

```python
def policy(board: list[list[int]]) -> int:
    # Check for an immediate winning move
    for col in range(7):
        if board[0][col] != 0:
            continue
        row = get_next_available_row(board, col)
        new_board = [row.copy() for row in board]
        new_board[row][col] = 1
        if has_four_in_row(new_board, 1):
            return col
    
    # Check for moves that block the opponent's potential win
    safe_cols = []
    for col in range(7):
        if board[0][col] != 0:
            continue
        row = get_next_available_row(board, col)
        new_board = [r.copy() for r in board]
        new_board[row][col] = 1
        if not any_opponent_can_win(new_board):
            safe_cols.append(col)
    
    if safe_cols:
        return select_best_move(board, safe_cols)
    
    # If all moves are losing, pick any legal move
    for col in range(7):
        if board[0][col] == 0:
            return col
    return -1

def get_next_available_row(board, col):
    for row in range(5, -1, -1):
        if board[row][col] == 0:
            return row
    return -1

def has_four_in_row(board, player):
    # Check rows
    for row in range(6):
        for col in range(4):
            if (board[row][col] == player and board[row][col+1] == player and
                board[row][col+2] == player and board[row][col+3] == player):
                return True
    # Check columns
    for col in range(7):
        for row in range(3):
            if (board[row][col] == player and board[row+1][col] == player and
                board[row+2][col] == player and board[row+3][col] == player):
                return True
    # Check diagonals (top-left to bottom-right)
    for row in range(3):
        for col in range(4):
            if (board[row][col] == player and board[row+1][col+1] == player and
                board[row+2][col+2] == player and board[row+3][col+3] == player):
                return True
    # Check diagonals (top-right to bottom-left)
    for row in range(3):
        for col in range(3, 7):
            if (board[row][col] == player and board[row+1][col-1] == player and
                board[row+2][col-2] == player and board[row+3][col-3] == player):
                return True
    return False

def any_opponent_can_win(board):
    for col in range(7):
        if board[0][col] != 0:
            continue
        new_board = [row.copy() for row in board]
        row = get_next_available_row(new_board, col)
        new_board[row][col] = -1
        if has_four_in_row(new_board, -1):
            return True
    return False

def select_best_move(board, possible_cols):
    # Prioritize columns in this order: 3, 2, 4, 5, 1, 6, 0
    priority_order = {3: 0, 2: 1, 4: 2, 5: 3, 1: 4, 6: 5, 0: 6}
    best_col = None
    best_priority = 7  # Initialize with highest priority value (worst)
    for col in possible_cols:
        current_priority = priority_order.get(col, priority_order.get(None))
        if current_priority < best_priority:
            best_priority = current_priority
            best_col = col
    return best_col
```

### Explanation
1. **Immediate Win Check**: The function first checks each column to see if placing a disc there would result in a win for the current player. This is done by simulating the move and using the `has_four_in_row` function to check for a winning condition.
2. **Blocking Opponent's Win**: If no immediate win is found, the function checks each possible move to see if it would allow the opponent to win in their next turn. Moves that do not allow the opponent to win are collected.
3. **Heuristic Selection**: Among the safe moves (those that block the opponent's potential win), the function uses a heuristic that prioritizes central columns (3, 2, 4, 5, 1, 6, 0), which are generally more advantageous in Connect Four due to their central position.
4. **Default Move**: If no safe moves are available (i.e., every possible move allows the opponent to win), the function defaults to selecting the first available column.

This approach ensures that the AI prioritizes winning moves, blocks the opponent's wins, and makes strategic moves to control the game effectively.
