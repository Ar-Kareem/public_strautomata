The strategy implemented here focuses on three main priorities to determine the optimal move in a Connect 4 game:

1. **Immediate Win:** First, check if there's a column where placing the current player's disc results in an immediate win. If found, select that column to win the game.
2. **Block Opponent's Win:** If no winning move exists, check if the opponent can win in their next move by simulating their disc placement in each column. Block their winning move by selecting that column.
3. **Strategic Placement:** If neither player has an immediate win, prioritize columns closer to the center (column 3 first) since they offer more opportunities for creating future winning lines.

The approach uses a helper function `is_win` to determine if placing a disc at a specific position leads to a win by checking all possible directions (horizontal, vertical, and diagonals) for four consecutive discs. The columns are evaluated in a center-first order to maximize strategic advantage when no immediate threats or wins are present.

This ensures the AI plays optimally by capitalizing on winning opportunities, preventing the opponent from winning, and maintaining a strong board position.

def policy(board: list[list[int]]) -> int:
    def is_win(b, player, row, col):
        directions = [(0, 1), (1, 0), (1, 1), (1, -1)]
        for dx, dy in directions:
            count = 1
            x, y = row + dx, col + dy
            while 0 <= x < 6 and 0 <= y < 7:
                if (x == row and y == col) or b[x][y] == player:
                    count += 1
                else:
                    break
                x += dx
                y += dy
            x, y = row - dx, col - dy
            while 0 <= x < 6 and 0 <= y < 7:
                if (x == row and y == col) or b[x][y] == player:
                    count += 1
                else:
                    break
                x -= dx
                y -= dy
            if count >= 4:
                return True
        return False

    valid_columns = [col for col in range(7) if board[0][col] == 0]
    center_order = [3, 2, 4, 1, 5, 0, 6]

    # Check for immediate win
    for col in center_order:
        if col not in valid_columns:
            continue
        row = 5
        while row >= 0 and board[row][col] != 0:
            row -= 1
        if row < 0:
            continue
        if is_win(board, 1, row, col):
            return col

    # Check opponent's win
    for col in center_order:
        if col not in valid_columns:
            continue
        row = 5
        while row >= 0 and board[row][col] != 0:
            row -= 1
        if row < 0:
            continue
        if is_win(board, -1, row, col):
            return col

    # Choose best remaining column
    for col in center_order:
        if col in valid_columns:
            return col

    # Fallback (shouldn't reach here as valid_columns is non-empty)
    return valid_columns[0]
