The goal is to implement a strong policy for Lines of Action. Key considerations:
1. The move must be legal: a piece moves exactly N squares in a line (row/col/diag), where N is the total number of pieces (both players) in that line.
2. Pieces can jump over friendly pieces, but not enemy pieces.
3. The objective is to connect all your pieces into one contiguous group (8-directional connectivity).

Strategy:
- We want to prioritize moves that lead toward connecting our pieces.
- We also want to prevent the opponent from connecting their pieces.
- We should prefer captures when possible (removing an opponent's piece reduces their connectivity and total piece count, making future moves harder for them).
- We should evaluate moves by simulating their effect on our connectivity and opponent's connectivity.

Algorithm:
1. Generate all legal moves for current player.
2. For each legal move, simulate it and compute:
   a. The new connectivity of our pieces (number of connected components).
   b. The new connectivity of opponent's pieces.
   c. A score based on: 
        - Reducing our own connected components (good)
        - Increasing opponent's connected components (bad)
        - Capturing an opponent piece (bonus)
        - Moving toward the center (spatial control)
3. Choose the move with the highest score.

We'll use BFS/DFS to compute connected components of our pieces and the opponent's pieces.

Since the board is small (8x8) and we have 1 second, we can compute all legal moves and evaluate them exhaustively.

- Generate all possible moves from every piece (8 directions).
- For each piece and direction, compute the number of pieces in that line, then check if a move of that distance is possible (without jumping over opponent pieces).

- Use a scoring function that favors:
   * Fewer connected components in our group (ideally 1)
   * More connected components in opponent's group
   * Captures (bonus 10 points)

def simulate_move(board, from_row, from_col, to_row, to_col):
    # Create a copy of the board
    new_board = np.array(board)
    # Move piece from source to target
    piece = new_board[from_row, from_col]
    new_board[from_row, from_col] = 0
    new_board[to_row, to_col] = piece
    return new_board

def count_connected_components(board, player):
    """Count connected components for player (8-directional) using BFS"""
    n = 8
    visited = [[False]*n for _ in range(n)]
    components = 0
    
    for r in range(n):
        for c in range(n):
            if board[r][c] == player and not visited[r][c]:
                components += 1
                # BFS
                queue = deque([(r, c)])
                visited[r][c] = True
                while queue:
                    cr, cc = queue.popleft()
                    # 8 directions
                    for dr in [-1,0,1]:
                        for dc in [-1,0,1]:
                            if dr == 0 and dc == 0:
                                continue
                            nr, nc = cr + dr, cc + dc
                            if 0 <= nr < n and 0 <= nc < n and board[nr][nc] == player and not visited[nr][nc]:
                                visited[nr][nc] = True
                                queue.append((nr, nc))
    
    return components
