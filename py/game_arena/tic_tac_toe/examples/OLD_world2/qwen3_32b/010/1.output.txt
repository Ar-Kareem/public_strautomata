Here's a full Python implementation of the `policy` function for the 4x4 Tic Tac Toe game. This function prioritizes immediate wins, blocks threats, and selects strategic positions on the board in a fallback order if no urgent move exists.

```python
def policy(board):
    def is_winning_move(r, c, player):
        # Check if placing the player at (r,c) completes a winning line
        # Row check
        row_win = True
        for i in range(4):
            if board[r][i] != player and i != c:
                row_win = False
                break
        if row_win:
            return True

        # Column check
        col_win = True
        for i in range(4):
            if board[i][c] != player and i != r:
                col_win = False
                break
        if col_win:
            return True

        # Main diagonal check (r == c)
        if r == c:
            diag1_win = True
            for i in range(4):
                if board[i][i] != player and i != r:
                    diag1_win = False
                    break
            if diag1_win:
                return True

        # Anti-diagonal check (r + c == 3)
        if r + c == 3:
            diag2_win = True
            for i in range(4):
                j = 3 - i
                if not (i == r and j == c) and board[i][j] != player:
                    diag2_win = False
                    break
            if diag2_win:
                return True

        return False

    # 1. Check for an immediate win
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0 and is_winning_move(r, c, 1):
                return (r, c)

    # 2. Check if the opponent can win in the next move and block it
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0 and is_winning_move(r, c, -1):
                return (r, c)

    # 3. Fallback: select center, then corners, then edges in fixed order
    for (r, c) in [(1, 1), (1, 2), (2, 1), (2, 2)]:
        if board[r][c] == 0:
            return (r, c)

    for (r, c) in [(0, 0), (0, 3), (3, 0), (3, 3)]:
        if board[r][c] == 0:
            return (r, c)

    for (r, c) in [(0, 1), (0, 2), (1, 0), (1, 3), (2, 0), (2, 3), (3, 1), (3, 2)]:
        if board[r][c] == 0:
            return (r, c)

    # Fallback to the top-left corner if all else fails
    return (0, 0)
```

### Explanation:

- **Immediate Win Check**: The function scans the board for an empty cell where placing the AI's symbol (`1`) would result in a win. This includes rows, columns, and full diagonals.

- **Opponent Win Blocking**: If the AI does not have an immediate win, the function checks if the opponent (`-1`) can win on their next move. If so, the AI blocks by taking that cell.

- **Strategic Fallback**: If no urgent action is needed, the AI chooses the optimal positions in priority order:
  - Prefer **center positions** for maximum influence.
  - Then **corners**, which are stable and strategic.
  - Finally, **edges**, prioritized in a consistent order.

This approach balances tactical urgency with strategic positioning and runs efficiently under the time constraint.
