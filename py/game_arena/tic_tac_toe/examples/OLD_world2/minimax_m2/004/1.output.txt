To solve this problem, we need to determine the next optimal move for a 4x4 Tic Tac Toe game where the player is represented by `1` and the opponent by `-1`. The goal is to either win immediately if possible, block the opponent's immediate win, or make a move that maximizes the player's advantage while minimizing the opponent's potential gain over the next two moves.

### Approach
1. **Precompute Winning Lines**: Identify all possible winning lines in a 4x4 grid (4 rows, 4 columns, and 2 diagonals).
2. **Check Immediate Win**: If the player can win in one move, choose that move.
3. **Check Immediate Block**: If the opponent can win in one move, block it by placing the player's piece in the opponent's winning cell.
4. **Evaluate Potential Moves**: For each remaining empty cell:
   - **Simulate Player's Move**: Place the player's piece in the cell.
   - **Simulate Opponent's Response**: For each possible opponent move, evaluate the resulting board state. The opponent is assumed to play optimally to minimize the player's score.
   - **Score Calculation**: Use an evaluation function that scores the board from the player's perspective, considering the number of `1`s (player) and `-1`s (opponent) in each winning line. The player wants to maximize this score.
5. **Select Best Move**: Choose the move that results in the highest score after considering the opponent's optimal response.

### Solution Code
```python
def policy(board):
    lines = [
        [(i, j) for j in range(4)] for i in range(4)
    ] + [
        [(i, j) for i in range(4)] for j in range(4)
    ] + [
        [(i, i) for i in range(4)],
        [(i, 3-i) for i in range(4)]
    ]
    
    empty_cells = []
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                empty_cells.append((i, j))
                
    if not empty_cells:
        return (0, 0)
        
    def check_win(player):
        for line in lines:
            count = 0
            for (i, j) in line:
                if board[i][j] == player:
                    count += 1
            if count == 4:
                return True
        return False
        
    for (i, j) in empty_cells:
        board[i][j] = 1
        if check_win(1):
            board[i][j] = 0
            return (i, j)
        board[i][j] = 0
        
    for (i, j) in empty_cells:
        board[i][j] = -1
        if check_win(-1):
            board[i][j] = 0
            return (i, j)
        board[i][j] = 0
        
    def evaluate():
        score = 0
        for line in lines:
            count_1 = 0
            count_minus1 = 0
            for (i, j) in line:
                if board[i][j] == 1:
                    count_1 += 1
                elif board[i][j] == -1:
                    count_minus1 += 1
            score += count_1 * 10
            score -= count_minus1 * 10
        return score
        
    best_move = None
    best_score = -10**9
    
    for (i, j) in empty_cells:
        board[i][j] = 1
        new_empty = []
        for x in range(4):
            for y in range(4):
                if board[x][y] == 0:
                    new_empty.append((x, y))
        min_score = 10**9
        
        for (k, l) in new_empty:
            board[k][l] = -1
            if check_win(-1):
                score = -1000
            else:
                score = evaluate()
            board[k][l] = 0
            if score < min_score:
                min_score = score
        board[i][j] = 0
        if min_score > best_score:
            best_score = min_score
            best_move = (i, j)
            
    return best_move
```

### Explanation
1. **Winning Lines**: The solution first computes all possible winning lines (rows, columns, diagonals) in the 4x4 grid.
2. **Immediate Win Check**: The algorithm checks if placing a piece in any empty cell results in an immediate win for the player. If so, it selects that move.
3. **Immediate Block Check**: If the opponent can win on their next move, the algorithm blocks by placing the player's piece in the opponent's winning cell.
4. **Move Evaluation**: For each remaining empty cell:
   - **Simulate Player's Move**: The player's piece is placed in the cell.
   - **Simulate Opponent's Response**: The opponent's optimal move is simulated for each possible response.
   - **Score Calculation**: The board is evaluated by summing the number of player pieces and subtracting the number of opponent pieces in all winning lines, scaled by 10. Higher scores indicate better positions for the player.
5. **Best Move Selection**: The move that results in the highest score after the opponent's optimal response is selected as the next move.

This approach efficiently balances immediate gains (wins) and strategic positioning (blocking and optimizing future moves) using a lookahead strategy and an evaluation function.
