To solve this problem, we need to determine the next move for a 4x4 Tic Tac Toe game. The board is represented as a 4x4 grid where each cell can be empty (0), occupied by the player (1), or occupied by the opponent (-1). The goal is to choose a move that maximizes the chances of winning while preventing the opponent from winning in the next move.

### Approach
1. **Check for Immediate Win**: If placing the player's piece in any empty cell results in a win (three pieces in a row), return that cell.
2. **Block Opponent's Immediate Win**: If the opponent can win in their next move by placing a piece in any empty cell, block that move by placing the player's piece there.
3. **Heuristic Evaluation**: If neither immediate win nor blocking is possible, use a heuristic to evaluate each empty cell. The heuristic considers:
   - **Winning Lines**: For each empty cell, check all lines (rows, columns, diagonals) that include this cell. Assign points based on the number of existing player or opponent pieces in these lines.
   - **Position Value**: Assign additional points to cells based on their position (center cells are more valuable than corners, which are more valuable than edges).
4. **Choose Best Move**: Select the empty cell with the highest heuristic score. If multiple cells have the same highest score, any can be chosen.

### Solution Code
```python
def policy(board):
    lines = []
    for i in range(4):
        lines.append([(i, j) for j in range(4)])
    for j in range(4):
        lines.append([(i, j) for i in range(4)])
    lines.append([(i, i) for i in range(4)])
    lines.append([(i, 3 - i) for i in range(4)])
    
    for line in lines:
        ones = 0
        empty_cell = None
        for (i, j) in line:
            if board[i][j] == 1:
                ones += 1
            elif board[i][j] == 0:
                empty_cell = (i, j)
        if ones == 3 and empty_cell is not None:
            return empty_cell
    
    for line in lines:
        opps = 0
        empty_cell = None
        for (i, j) in line:
            if board[i][j] == -1:
                opps += 1
            elif board[i][j] == 0:
                empty_cell = (i, j)
        if opps == 3 and empty_cell is not None:
            return empty_cell
    
    empty_cells = [(i, j) for i in range(4) for j in range(4) if board[i][j] == 0]
    moves_scores = []
    for (i, j) in empty_cells:
        score = 0
        for line in lines:
            if (i, j) not in line:
                continue
            count_ours = 0
            count_opp = 0
            for (x, y) in line:
                if (x, y) == (i, j):
                    continue
                if board[x][y] == 1:
                    count_ours += 1
                elif board[x][y] == -1:
                    count_opp += 1
            if count_opp >= 2:
                score += 100
            if count_ours >= 2:
                score += 20
            if count_ours == 1:
                score += 5
            if count_opp == 1:
                score += 3
            if count_ours == 0 and count_opp == 0:
                score += 1
        if (i, j) in [(1, 1), (1, 2), (2, 1), (2, 2)]:
            score += 10
        elif (i, j) in [(0, 0), (0, 3), (3, 0), (3, 3)]:
            score += 5
        moves_scores.append((score, (i, j)))
    
    moves_scores.sort(key=lambda x: x[0], reverse=True)
    return moves_scores[0][1]
```

### Explanation
1. **Immediate Win Check**: The function first checks all possible winning lines (rows, columns, diagonals). If three player pieces are found in a line with one empty cell, it returns that empty cell to complete the line and win.
2. **Opponent Block Check**: Similarly, it checks for lines where the opponent has three pieces. If found, it places the player's piece in the remaining empty cell to block the opponent's win.
3. **Heuristic Evaluation**: For each empty cell not used in the above checks, the function calculates a heuristic score based on:
   - **Winning Line Influence**: Points are awarded based on how many player or opponent pieces are already in the lines that include the cell.
   - **Position Value**: Center cells receive the highest points, followed by corners and edges.
4. **Move Selection**: The cell with the highest heuristic score is selected as the next move. If multiple cells have the same score, the first one encountered is chosen.

This approach efficiently balances immediate defensive and offensive strategies with a positional heuristic to make optimal moves in a 4x4 Tic Tac Toe game.
