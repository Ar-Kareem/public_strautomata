To address the challenge of determining the next move in a 4x4 Tic Tac Toe game, we can utilize a heuristic-based approach that prioritizes immediate winning moves, blocking the opponent's potential winning moves, and finally selecting positions based on strategic positioning.

### Approach
1. **Check for Immediate Winning Moves**: For each empty cell, check if placing a move in that cell would result in a line of four 1s (my marks). If found, return that cell.
2. **Check for Blocking Moves**: If no winning move is found, check for cells that would block the opponent from completing four in a row, column, or diagonal.
3. **Heuristic Selection**: When no immediate threats are present, use a heuristic based on the contribution of the cell to the player's lines (rows, columns, diagonals) and positional bonuses for strategic cells like the center and corners.

### Solution Code
```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    empty_cells = [(i, j) for i in range(4) for j in range(4) if board[i][j] == 0]
    
    # Step 1: Check for winning moves (my three marks in a line)
    for i, j in empty_cells:
        # Check row i
        if board[i].count(1) == 3:
            return (i, j)
        # Check column j
        if sum(board[x][j] == 1 for x in range(4)) == 3:
            return (i, j)
        # Check main diagonal (i == j)
        if i == j:
            diag = [board[k][k] for k in range(4)]
            if sum(diag) == 3:
                return (i, j)
        # Check other diagonal (i + j == 3)
        if i + j == 3:
            diag = [board[k][3 - k] for k in range(4)]
            if sum(diag) == 3:
                return (i, j)
    
    # Step 2: Check for blocking moves (opponent's three marks)
    for i, j in empty_cells:
        # Check row
        if board[i].count(-1) == 3:
            return (i, j)
        # Check column
        if sum(board[x][j] == -1 for x in range(4)) == 3:
            return (i, j)
        # Check main diagonal
        if i == j:
            diag = [board[k][k] for k in range(4)]
            if sum(diag) == -3:
                return (i, j)
        # Check other diagonal
        if i + j == 3:
            diag = [board[k][3 - k] for k in range(4)]
            if sum(diag) == -3:
                return (i, j)
    
    # Step 3: Heuristic-based selection for strategic moves
    best_score = float('-inf')
    best_cell = None
    
    for cell in empty_cells:
        i, j = cell
        score = 0
        
        # Check row contribution
        row = board[i]
        my_row = row.count(1)
        opp_row = row.count(-1)
        row_diff = my_row - opp_row
        if row_diff > 0:
            score += row_diff * row_diff
        elif row_diff == 0:
            score += 1
        
        # Check column contribution
        col = [board[x][j] for x in range(4)]
        my_col = col.count(1)
        opp_col = col.count(-1)
        col_diff = my_col - opp_col
        if col_diff > 0:
            score += col_diff * col_diff
        elif col_diff == 0:
            score += 1
        
        # Evaluate diagonals
        if i == j:
            diag = [board[k][k] for k in range(4)]
            my_diag = diag.count(1)
            opp_diag = diag.count(-1)
            diag_diff = my_diag - opp_diag
            if diag_diff > 0:
                score += diag_diff * diag_diff
            elif diag_diff == 0:
                score += 1
        if i + j == 3:
            diag = [board[k][3 - k] for k in range(4)]
            my_diag = diag.count(1)
            opp_diag = diag.count(-1)
            diag_diff = my_diag - opp_diag
            if diag_diff > 0:
                score += diag_diff * diag_diff
            elif diag_diff == 0:
                score += 1
        
        # Position bonuses
        if i in {1, 2} and j in {1, 2}:
            score += 2  # Center cells bonus
        if (i, j) in {(0, 0), (0, 3), (3, 0), (3, 3)}:
            score += 1  # Corners bonus
        
        if score > best_score:
            best_score = score
            best_cell = cell
        elif score == best_score and cell < best_cell:
            best_cell = cell
    
    return best_cell
```

### Explanation
1. **Immediate Winning Moves**: The code first checks each empty cell to see if placing a mark would complete a line of four 1s. It does this by counting the number of 1s in the row, column, and diagonals of the cell.
2. **Blocking Opponent Moves**: If no winning move is found, the code checks for moves that would prevent the opponent from completing four in a row, column, or diagonal by placing in the necessary cell.
3. **Heuristic Scoring**: If neither winning nor blocking moves are available, the code uses a heuristic to evaluate each empty cell. The score is derived from the contribution of the cell to the player's lines, prioritizing positive differentials in rows, columns, and diagonals. Additionally, positional bonuses are given to center cells and corners to guide strategic decisions.

This approach ensures efficient decision-making within the given constraints, balancing between immediate threats and long-term strategic positioning.
