To solve this problem, I need to write a Python function that determines the optimal next move for a Tic Tac Toe game. The function should return a tuple representing the row and column of the chosen move. 

**Approach:**
1. **Immediate Win Check:** First, the AI checks if it can win immediately by placing its mark (1) in any empty cell. This is done by simulating placing a 1 in each empty cell and checking if that results in a win.
2. **Block Opponent's Win:** If the AI can't win immediately, it checks if the opponent (-1) is about to win by simulating the opponent's move in each empty cell. If so, the AI blocks that move.
3. **Strategic Placement:** If no immediate win or block is needed, the AI chooses the best strategic position:
   - Prefer the center (1,1) as it's part of the most winning lines.
   - Then choose corners (0,0), (0,2), (2,0), (2,2) as they are part of three lines.
   - Finally, pick edges (0,1), (1,0), (1,2), (2,1) which are part of two lines.

**Helper Function:**
- `is_win(board, player, i, j)`: Checks if placing `player` at position (i,j) results in a win. This is done by checking the row, column, and relevant diagonals for a sum equal to 3 times the player's value.

**Edge Cases:**
- The first move should take the center.
- If the opponent is about to win in multiple spots, the first occurrence is blocked.
- When neither player can win immediately, the AI follows the priority order to maximize its chances.

**Efficiency:**
- The algorithm checks each empty cell a maximum of twice (once for win, once for block) and then iterates through a fixed list of prioritized positions. This is efficient enough for a 3x3 grid within the time constraints.

def policy(board: list[list[int]]) -> tuple[int, int]:
    def is_win(b, player, i, j):
        # Simulate placing the player's mark at (i,j) and check for a win
        temp = [row.copy() for row in b]
        temp[i][j] = player
        # Check row
        if sum(temp[i]) == player * 3:
            return True
        # Check column
        if sum(temp[r][j] for r in range(3)) == player * 3:
            return True
        # Check main diagonal
        if i == j and sum(temp[k][k] for k in range(3)) == player * 3:
            return True
        # Check anti-diagonal
        if i + j == 2 and sum(temp[k][2 - k] for k in range(3)) == player * 3:
            return True
        return False
    
    # Collect all empty cells
    empty = [(i, j) for i in range(3) for j in range(3) if board[i][j] == 0]
    
    # Check for immediate win
    for i, j in empty:
        if is_win(board, 1, i, j):
            return (i, j)
    
    # Check for opponent's immediate win and block
    for i, j in empty:
        if is_win(board, -1, i, j):
            return (i, j)
    
    # Prefer center, then corners, then edges
    preferred_order = [(1, 1), (0, 0), (0, 2), (2, 0), (2, 2),
                       (0, 1), (1, 0), (1, 2), (2, 1)]
    
    for move in preferred_order:
        if board[move[0]][move[1]] == 0:
            return move
    
    # Fallback (shouldn't be reached in a valid game state)
    return empty[0]
