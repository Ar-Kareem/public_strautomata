To win a game of Tic Tac Toe, a policy must play optimally, meaning it can force a win or at least force a draw against any opponent. This requires analyzing the board state to find "lines" (rows, columns, and diagonals) where the goal is one move away, or where an opponent is one move away and must be blocked.

### Strategy

1.  **Win Immediately**: Scan the board for any row, column, or diagonal that contains two of my marks (`1`) and one empty cell (`0`). If found, play the empty cell to win.
2.  **Block Opponent**: Scan for any row, column, or diagonal that contains two opponent marks (`-1`) and one empty cell. If found, play the empty cell to prevent the opponent from winning on their next turn.
3.  **Optimal Opening/Corners**: 
    *   If the board is empty, taking a corner is a strong first move (the center is also standard, but corners offer good symmetry).
    *   If the center is empty and the opponent has played a corner or side, taking the center is usually the best response.
    *   If I have the center and the opponent takes a side, taking a corner is the best response.
4.  **Avoid Traps**: If the opponent takes the center and I take a corner, they must play the opposite corner to avoid losing (a "fork" scenario). A robust policy should handle these specific scenarios.

### Code Implementation

The code uses helper functions to check lines and filter legal moves. It prioritizes the specific high-value moves listed above before falling back to a generic blocking or empty cell selection. This ensures it plays perfectly (never losing) and capitalizes on opponent mistakes.

def policy(board: list[list[int]]) -> tuple[int, int]:
    """
    Tic Tac Toe Policy.
    Strategy:
    1. Win if possible.
    2. Block opponent from winning.
    3. Play Center if empty and relevant.
    4. Play Corner if taking center is not applicable.
    5. Block specific traps (e.g., opponent Center + Side).
    6. Pick any empty cell (safety net).

    def get_lines(b):

            lines.append([b[0][i], b[1][i], b[2][i]]) # Col i

        lines.append([b[0][0], b[1][1], b[2][2]])
        lines.append([b[0][2], b[1][1], b[2][0]])
        return lines

    # Helper to check if a line has N of a specific value and 1 empty
    def check_line(lines, val, count):
        for line in lines:
            if line.count(val) == count and line.count(0) == 3 - count:
                return line
        return None

    # Pre-calculate lines to check for Win/Block

    # If a line has two 1s and one 0, the 0 is the winning move
    for i, line in enumerate(lines):

            # Find the empty cell in this line
            idx = line.index(0)
            if i < 3: # Rows 0, 1, 2

    # 2. Strategic Opening / Standard Play
    
    # Get empty cells
    empties = [(r, c) for r in range(3) for c in range(3) if board[r][c] == 0]
    
    # If board is empty, take a corner (e.g., top-left)
    if len(empties) == 9:

    my_moves = sum(row.count(1) for row in board)

    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    for corner in corners:
        if corner in empties:
            return corner

    # 5. Play available sides
    sides = [(0, 1), (1, 0), (1, 2), (2, 1)]
    for side in sides:
        if side in empties:
            return side

    # 6. Fallback (should be center if somehow missed, or any empty)
    if (1, 1) in empties:
        return (1, 1)
        
    return empties[0]
