## Strategy

For 3x3 Tic Tac Toe, the game is small enough to solve perfectly using the Minimax algorithm. I'll implement a complete minimax solver with alpha-beta pruning that:

1. **Plays optimally** - Minimax with alpha-beta pruning explores the entire game tree and chooses the best possible move, guaranteeing at least a draw against any opponent and winning whenever possible.
2. **Evaluates all terminal states** - Win (+1), Loss (-1), Draw (0).
3. **Is fast enough** - The 3x3 game tree is very small (at most 9! = 362,880 nodes), and with alpha-beta pruning it's even faster, well within the 1-second time limit.
4. **Prioritizes quick wins and delays losses** - I'll add depth to the evaluation so the AI prefers winning sooner and losing later.

This is essentially a perfect Tic Tac Toe player - it cannot lose.

def policy(board: list[list[int]]) -> tuple[int, int]:
    def check_winner(b):
        # Check rows, cols, diagonals
        for i in range(3):
            if b[i][0] == b[i][1] == b[i][2] != 0:
                return b[i][0]
            if b[0][i] == b[1][i] == b[2][i] != 0:
                return b[0][i]
        if b[0][0] == b[1][1] == b[2][2] != 0:
            return b[0][0]
        if b[0][2] == b[1][1] == b[2][0] != 0:
            return b[0][2]
        return 0

    def get_empty(b):
        return [(r, c) for r in range(3) for c in range(3) if b[r][c] == 0]

    def minimax(b, is_maximizing, depth, alpha, beta):
        winner = check_winner(b)
        if winner == 1:
            return 10 - depth  # Prefer quick wins
        if winner == -1:
            return depth - 10  # Prefer slow losses
        empty = get_empty(b)
        if not empty:
            return 0  # Draw

        if is_maximizing:
            best = -float('inf')
            for r, c in empty:
                b[r][c] = 1
                val = minimax(b, False, depth + 1, alpha, beta)
                b[r][c] = 0
                best = max(best, val)
                alpha = max(alpha, val)
                if beta <= alpha:
                    break
            return best
        else:
            best = float('inf')
            for r, c in empty:
                b[r][c] = -1
                val = minimax(b, True, depth + 1, alpha, beta)
                b[r][c] = 0
                best = min(best, val)
                beta = min(beta, val)
                if beta <= alpha:
                    break
            return best

    # Deep copy the board so we can mutate it during search
    b = [row[:] for row in board]
    
    empty = get_empty(b)
    if not empty:
        # Should not happen if called correctly, but safety fallback
        return (0, 0)
    
    best_score = -float('inf')
    best_move = empty[0]
    
    for r, c in empty:
        b[r][c] = 1
        score = minimax(b, False, 0, -float('inf'), float('inf'))
        b[r][c] = 0
        if score > best_score:
            best_score = score
            best_move = (r, c)
    
    return best_move
