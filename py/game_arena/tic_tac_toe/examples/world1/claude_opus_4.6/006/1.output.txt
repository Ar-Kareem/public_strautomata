## Strategy

For 3x3 Tic Tac Toe, I'll implement a perfect minimax algorithm with alpha-beta pruning. Since the game tree for 3x3 Tic Tac Toe is very small (at most 9! = 362,880 terminal nodes, much less with pruning), this will easily run within the 1-second time limit.

The minimax algorithm will:
1. Evaluate all possible moves recursively
2. Choose the move that maximizes our score (win = +1, draw = 0, loss = -1)
3. Assume the opponent plays optimally (minimizes our score)
4. Use alpha-beta pruning to skip unnecessary branches

This guarantees perfect play - it will never lose, and will win whenever the opponent makes a mistake.

<code>
def policy(board: list[list[int]]) -> tuple[int, int]:
    def check_winner(b):
        # Check rows, cols, diags
        for i in range(3):
            if b[i][0] == b[i][1] == b[i][2] != 0:
                return b[i][0]
            if b[0][i] == b[1][i] == b[2][i] != 0:
                return b[0][i]
        if b[0][0] == b[1][1] == b[2][2] != 0:
            return b[0][0]
        if b[0][2] == b[1][1] == b[2][0] != 0:
            return b[0][2]
        return 0

    def get_empty(b):
        return [(r, c) for r in range(3) for c in range(3) if b[r][c] == 0]

    def minimax(b, is_maximizing, alpha, beta):
        winner = check_winner(b)
        if winner == 1:
            return 1
        if winner == -1:
            return -1
        empty = get_empty(b)
        if not empty:
            return 0

        if is_maximizing:
            best = -2
            for r, c in empty:
                b[r][c] = 1
                val = minimax(b, False, alpha, beta)
                b[r][c] = 0
                best = max(best, val)
                alpha = max(alpha, val)
                if beta <= alpha:
                    break
            return best
        else:
            best = 2
            for r, c in empty:
                b[r][c] = -1
                val = minimax(b, True, alpha, beta)
                b[r][c] = 0
                best = min(best, val)
                beta = min(beta, val)
                if beta <= alpha:
                    break
            return best

    # Make a deep copy of the board
    b = [row[:] for row in board]
    empty = get_empty(b)

    best_score = -2
    best_move = empty[0]

    for r, c in empty:
        b[r][c] = 1
        score = minimax(b, False, -2, 2)
        b[r][c] = 0
        if score > best_score:
            best_score = score
            best_move = (r, c)
            if best_score == 1:
                break

    return best_move
