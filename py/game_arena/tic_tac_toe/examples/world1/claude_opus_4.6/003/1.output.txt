## Strategy

For 3x3 Tic Tac Toe, the game is fully solvable. I'll implement a perfect minimax algorithm with alpha-beta pruning. Since the board is only 3x3 (9 cells max), the search space is tiny and minimax will complete well within the time limit.

The strategy:
1. Use minimax with alpha-beta pruning to evaluate all possible future game states
2. The algorithm assumes the opponent plays optimally (minimizing our score)
3. We maximize our score, choosing the move that leads to the best guaranteed outcome
4. This results in perfect play - we will never lose, and will win if the opponent makes any mistake

Key details:
- Score +10 minus depth for wins (preferring faster wins)
- Score -10 plus depth for losses (preferring slower losses)
- Score 0 for draws
- Alpha-beta pruning speeds things up, though it's already fast for 3x3

def policy(board: list[list[int]]) -> tuple[int, int]:
    def check_winner(b):
        # Check rows, cols, diagonals
        for i in range(3):
            if b[i][0] == b[i][1] == b[i][2] != 0:
                return b[i][0]
            if b[0][i] == b[1][i] == b[2][i] != 0:
                return b[0][i]
        if b[0][0] == b[1][1] == b[2][2] != 0:
            return b[0][0]
        if b[0][2] == b[1][1] == b[2][0] != 0:
            return b[0][2]
        return 0

    def get_empty(b):
        return [(r, c) for r in range(3) for c in range(3) if b[r][c] == 0]

    def minimax(b, is_maximizing, depth, alpha, beta):
        winner = check_winner(b)
        if winner == 1:
            return 10 - depth
        if winner == -1:
            return -10 + depth
        
        empty = get_empty(b)
        if not empty:
            return 0

        if is_maximizing:
            best = -float('inf')
            for r, c in empty:
                b[r][c] = 1
                score = minimax(b, False, depth + 1, alpha, beta)
                b[r][c] = 0
                best = max(best, score)
                alpha = max(alpha, score)
                if beta <= alpha:
                    break
            return best
        else:
            best = float('inf')
            for r, c in empty:
                b[r][c] = -1
                score = minimax(b, True, depth + 1, alpha, beta)
                b[r][c] = 0
                best = min(best, score)
                beta = min(beta, score)
                if beta <= alpha:
                    break
            return best

    # Deep copy the board so we can mutate it during search
    b = [row[:] for row in board]
    
    empty = get_empty(b)
    if not empty:
        # Should not happen if called correctly
        return (0, 0)
    
    best_score = -float('inf')
    best_move = empty[0]
    
    for r, c in empty:
        b[r][c] = 1
        score = minimax(b, False, 0, -float('inf'), float('inf'))
        b[r][c] = 0
        if score > best_score:
            best_score = score
            best_move = (r, c)
    
    return best_move
