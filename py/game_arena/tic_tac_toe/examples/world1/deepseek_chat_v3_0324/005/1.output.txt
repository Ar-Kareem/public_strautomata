To create a strong Tic Tac Toe AI, we need to implement a strategy that can both win when possible and block the opponent from winning. Here's the approach:

1. First, check if there's an immediate winning move (where we can place our mark to complete a line of three 1s). If found, take that move.
2. If no immediate win, check if the opponent has an immediate winning move (two -1s in a line). If found, block that move.
3. If neither, follow a strategic approach:
   - Prefer the center if it's open (best strategic position)
   - Then prefer corners (they offer more winning opportunities)
   - Finally take any remaining edges

This approach ensures we never lose while maximizing our chances to win. The implementation will:
- Check all rows, columns, and diagonals for potential wins/blocks
- Use the center/corners/edges priority when no immediate threats exist
- Return the first valid move found in this priority order

The solution is efficient (O(1) operations since the board is fixed size) and will always complete well within the 1 second limit.

def policy(board: list[list[int]]) -> tuple[int, int]:
    # Check for immediate winning move
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                # Test if this move would win
                board[i][j] = 1
                if is_winner(board, 1):
                    board[i][j] = 0  # undo test move
                    return (i, j)
                board[i][j] = 0  # undo test move
    
    # Check for opponent's winning move to block
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                # Test if opponent would win here
                board[i][j] = -1
                if is_winner(board, -1):
                    board[i][j] = 0  # undo test move
                    return (i, j)
                board[i][j] = 0  # undo test move
    
    # Strategic moves when no immediate win/block
    # Prefer center
    if board[1][1] == 0:
        return (1, 1)
    
    # Then prefer corners
    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    for i, j in corners:
        if board[i][j] == 0:
            return (i, j)
    
    # Finally take any edge
    edges = [(0, 1), (1, 0), (1, 2), (2, 1)]
    for i, j in edges:
        if board[i][j] == 0:
            return (i, j)
    
    # Should never reach here if board isn't full
    return (0, 0)

def is_winner(board, player):
    # Check rows
    for row in board:
        if all(cell == player for cell in row):
            return True
    # Check columns
    for col in range(3):
        if all(board[row][col] == player for row in range(3)):
            return True
    # Check diagonals
    if (board[0][0] == player and board[1][1] == player and board[2][2] == player):
        return True
    if (board[0][2] == player and board[1][1] == player and board[2][0] == player):
        return True
    return False
