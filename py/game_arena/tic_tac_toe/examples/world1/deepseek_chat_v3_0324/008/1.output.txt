1. **Check for Immediate Wins**: First, scan the board to see if there's any row, column, or diagonal where the AI can place a mark to win immediately.
2. **Block Opponent Wins**: If the opponent has two in a row and can win on their next move, block that position.
3. **Take Center**: The center is the most strategic position in Tic Tac Toe. If it's open, take it.
4. **Take Corners**: Corners are strong positions. If any corner is open, take one.
5. **Take Edges**: As a last resort, take any remaining edge position.

This strategy ensures the AI plays optimally, never losing when it's possible to force a draw or win.

The implementation will:
- Iterate through all empty cells, checking if placing a mark there would complete a line for the AI (win).
- If no win, check if the opponent can win next turn and block that.
- Otherwise, follow the priority: center > corners > edges.

def policy(board: list[list[int]]) -> tuple[int, int]:
    # Check for immediate win
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                # Simulate placing a 1 here
                board[i][j] = 1
                # Check if this move wins
                if is_winner(board, 1):
                    board[i][j] = 0  # Undo simulation
                    return (i, j)
                board[i][j] = 0  # Undo simulation

    # Check for opponent's immediate win and block
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                # Simulate opponent placing a -1 here
                board[i][j] = -1
                # Check if this move would make opponent win
                if is_winner(board, -1):

                    return (i, j)

    # Take center if available
    if board[1][1] == 0:

    # Take any corner
    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    for i, j in corners:
        if board[i][j] == 0:
            return (i, j)

    # Take any edge
    edges = [(0, 1), (1, 0), (1, 2), (2, 1)]
    for i, j in edges:
        if board[i][j] == 0:
            return (i, j)

    # Fallback (shouldn't reach here if board isn't full)
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                return (i, j)
    return (0, 0)  # Should never happen

def is_winner(board, player):
    # Check rows
    for row in board:
        if all(cell == player for cell in row):
            return True
    # Check columns
    for col in range(3):
        if all(board[row][col] == player for row in range(3)):
            return True
    # Check diagonals
    if board[0][0] == player and board[1][1] == player and board[2][2] == player:
        return True
    if board[0][2] == player and board[1][1] == player and board[2][0] == player:
        return True
    return False
