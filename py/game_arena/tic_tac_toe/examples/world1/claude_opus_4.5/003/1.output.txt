I need to write a Tic Tac Toe AI that plays optimally. For a 3x3 Tic Tac Toe game, the optimal strategy can be implemented using the minimax algorithm, which will guarantee at least a draw against any opponent.

Let me think about the approach:

1. I am player `1` and opponent is `-1`
2. I need to find the best move using minimax
3. For Tic Tac Toe, minimax is computationally feasible since the game tree is small

Key components:
- Check for win condition
- Minimax algorithm with alpha-beta pruning (though for 3x3, even without pruning it's fast enough)
- Return the best move

Let me implement this:

```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    # Check if someone has won
    def check_winner(b):
        # Check rows
        for row in b:
            if sum(row) == 3:
                return 1
            if sum(row) == -3:
                return -1
        # Check columns
        for col in range(3):
            s = b[0][col] + b[1][col] + b[2][col]
            if s == 3:
                return 1
            if s == -3:
                return -1
        # Check diagonals
        d1 = b[0][0] + b[1][1] + b[2][2]
        d2 = b[0][2] + b[1][1] + b[2][0]
        if d1 == 3 or d2 == 3:
            return 1
        if d1 == -3 or d2 == -3:
            return -1
        return 0
    
    def get_empty_cells(b):
        cells = []
        for i in range(3):
            for j in range(3):
                if b[i][j] == 0:
                    cells.append((i, j))
        return cells
    
    def minimax(b, is_maximizing, alpha, beta):
        winner = check_winner(b)
        if winner == 1:
            return 10
        if winner == -1:
            return -10
        
        empty = get_empty_cells(b)
        if not empty:
            return 0  # Draw
        
        if is_maximizing:
            best = -float('inf')
            for (r, c) in empty:
                b[r][c] = 1
                score = minimax(b, False, alpha, beta)
                b[r][c] = 0
                best = max(best, score)
                alpha = max(alpha, score)
                if beta <= alpha:
                    break
            return best
        else:
            best = float('inf')
            for (r, c) in empty:
                b[r][c] = -1
                score = minimax(b, True, alpha, beta)
                b[r][c] = 0
                best = min(best, score)
                beta = min(beta, score)
                if beta <= alpha:
                    break
            return best
    
    # Find the best move for player 1 (me)
    empty_cells = get_empty_cells(board)
    best_move = empty_cells[0]
    best_score = -float('inf')
    
    for (r, c) in empty_cells:
        board[r][c] = 1
        score = minimax(board, False, -float('inf'), float('inf'))
        board[r][c] = 0
        if score > best_score:
            best_score = score
            best_move = (r, c)
    
    return best_move
```
