The board is a 3x3 list of lists with 0 (empty), 1 (AI), and -1 (opponent).
I need to implement a Tic-Tac-Toe AI. A common and effective approach for Tic-Tac-Toe is the Minimax algorithm, possibly with Alpha-Beta pruning for efficiency, although for a 3x3 board, simple Minimax without pruning is usually fast enough.

Let's break down the Minimax algorithm for Tic-Tac-Toe:

1.  **Terminal States**:
    *   Win: Current player has 3 in a row, column, or diagonal.
    *   Loss: Opponent has 3 in a row, column, or diagonal.
    *   Draw: Board is full, and no one has won.

2.  **Scoring**:
    *   Win: +10 (for the current player)
    *   Loss: -10 (for the current player)
    *   Draw: 0

3.  **Minimax Function**:
    *   Takes the current `board` and `player` (1 for AI, -1 for opponent).
    *   If the state is terminal, return its score.
    *   If `player` is AI (maximizing player):
        *   Initialize `best_score = -infinity`.
        *   For each empty cell:
            *   Make a move for `player`.
            *   Recursively call Minimax for the opponent (`-player`).
            *   `score = minimax(new_board, -player)`
            *   `best_score = max(best_score, score)`
            *   Undo the move (reset the cell to 0).
        *   Return `best_score`.
    *   If `player` is opponent (minimizing player):
        *   Initialize `best_score = +infinity`.
        *   For each empty cell:
            *   Make a move for `player`.
            *   Recursively call Minimax for the AI (`-player`).
            *   `score = minimax(new_board, -player)`
            *   `best_score = min(best_score, score)`
            *   Undo the move.
        *   Return `best_score`.

4.  **Main `policy` Function**:
    *   Iterate through all empty cells on the current `board`.
    *   For each empty cell:
        *   Make a temporary move for AI (`player = 1`).
        *   Call the Minimax function for the opponent (`-1`) to evaluate this move's outcome.
        *   Store the score returned by Minimax and the corresponding move.
        *   Undo the temporary move.
    *   Choose the move that results in the *maximum* score for the AI.
    *   Return that move.

**Helper Functions needed**:

*   `check_winner(board)`: Returns 1 if AI wins, -1 if opponent wins, 0 if draw, None if game is not over.
*   `get_empty_cells(board)`: Returns a list of (row, col) tuples for empty cells.

*   The `policy` function itself will be the entry point and will initiate the Minimax search by trying all possible first moves for itself and selecting the one that leads to the best outcome.

**`minimax(board, current_player)` function:**

`current_player` will be `1` for the AI and `-1` for the opponent.
The value returned by minimax is from the perspective of the *initial* AI player (maximizing player). So, if `current_player` is the AI, it wants to maximize the score. If `current_player` is the opponent, it wants to minimize the score for the AI.

```python
def _minimax(board, current_player):
    winner = _check_winner(board)

    if winner is not None:
        if winner == 1:
            return 10 # AI wins
        elif winner == -1:
            return -10 # Opponent wins
        else: # winner == 0 (draw)
            return 0

    if current_player == 1: # AI's turn (Maximizing player)
        best_score = -math.inf
        for r in range(3):
            for c in range(3):
                if board[r][c] == 0:
                    board[r][c] = 1 # Make the move
                    score = _minimax(board, -1) # Recurse for opponent
                    board[r][c] = 0 # Undo the move
                    best_score = max(best_score, score)
        return best_score
    else: # Opponent's turn (Minimizing player for AI's score)
        best_score = math.inf
        for r in range(3):
            for c in range(3):
                if board[r][c] == 0:
                    board[r][c] = -1 # Make the move
                    score = _minimax(board, 1) # Recurse for AI
                    board[r][c] = 0 # Undo the move

def policy(board: list[list[int]]) -> tuple[int, int]:
    best_move = (-1, -1)
    best_score = -math.inf

    # Iterate through all empty cells
    for r in range(3):
        for c in range(3):
            if board[r][c] == 0:
                board[r][c] = 1 # Try making AI's move
                # After AI makes a move, it's the opponent's turn.
                # So, we call minimax with -1 (opponent) as the current_player
                # But the score returned is still from AI's perspective.
                score = _minimax(board, -1)
                board[r][c] = 0 # Undo the move

                if score > best_score:
                    best_score = score
                    best_move = (r, c)
    return best_move
```
