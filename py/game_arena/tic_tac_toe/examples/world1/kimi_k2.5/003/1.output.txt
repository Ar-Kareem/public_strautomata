 The strategy for this policy is to implement **perfect play** using the minimax algorithm with alpha-beta pruning. Tic Tac Toe is a solved game where optimal play guarantees at least a draw. The algorithm works as follows:

1. **Immediate Win Check**: Scan for any move that completes three in a row for the agent (player 1) and take it immediately.
2. **Threat Block**: Scan for any move where the opponent (player -1) could win on their next turn and block it.
3. **Minimax Search**: For all other positions, recursively evaluate the game tree to depth 9 (maximum moves). The agent maximizes its score while assuming the opponent minimizes it. Scoring assigns +10 for a win (minus depth to prefer quicker victories), -10 for a loss (plus depth to delay defeat), and 0 for a draw.
4. **Alpha-Beta Pruning**: Optimizes the search by eliminating branches that cannot possibly influence the final decision, ensuring the solution runs in well under 1 millisecond.

This policy will never lose; it wins against any suboptimal opponent and draws against another optimal player.

        for i in range(3):
            if b[i][0] == b[i][1] == b[i][2] != 0:
                return b[i][0]
        # Check columns
        for i in range(3):
            if b[0][i] == b[1][i] == b[2][i] != 0:
                return b[0][i]

        if b[0][0] == b[1][1] == b[2][2] != 0:
            return b[0][0]
        if b[0][2] == b[1][1] == b[2][0] != 0:
            return b[0][2]
        return 0

        """Check if board is full."""
        for row in b:
            for cell in row:
                if cell == 0:
                    return False
        return True

    for i in range(3):

            return depth - 10
