

To solve this problem, we need to create an AI that can play Tic Tac Toe optimally. The AI should be able to determine the best move to make on a given 3x3 board by either winning immediately, blocking an opponent's potential win, or choosing the most strategic position available.

### Approach
The AI follows a prioritized approach to determine the next move:

1. **Immediate Win Check**: The AI first checks if it can win in the next move by placing its marker (1) in any of the empty cells. If such a move exists, it is immediately chosen.
2. **Block Opponent's Win**: If no immediate win is possible, the AI checks if the opponent (-1) can win in the next move. If such a move exists, the AI blocks it by placing its marker in the corresponding cell.
3. **Strategic Position Selection**: If neither an immediate win nor a block is possible, the AI selects the best available position based on a predefined priority list. The priority order is:
   - Center (1,1)
   - Corners (0,0), (0,2), (2,0), (2,2)
   - Edges (0,1), (1,0), (1,2), (2,1)

This approach ensures that the AI plays optimally, maximizing its chances of winning while minimizing the opponent's opportunities.

### Solution Code
```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    def check_win(b, player):
        # Check rows for a win
        for row in b:
            if row[0] == row[1] == row[2] == player:
                return True
        # Check columns for a win
        for col in range(3):
            if b[0][col] == b[1][col] == b[2][col] == player:
                return True
        # Check diagonals for a win
        if (b[0][0] == b[1][1] == b[2][2] == player) or (b[0][2] == b[1][1] == b[2][0] == player):
            return True
        return False

    # Collect all empty cells
    empty = [(i, j) for i in range(3) for j in range(3) if board[i][j] == 0]

    # Check for a winning move
    for i, j in empty:
        new_board = [row.copy() for row in board]
        new_board[i][j] = 1
        if check_win(new_board, 1):
            return (i, j)

    # Check for opponent's winning move to block
    for i, j in empty:
        new_board = [row.copy() for row in board]
        new_board[i][j] = -1
        if check_win(new_board, -1):
            return (i, j)

    # Select the best available position based on priority
    priority_cells = [
        (1, 1),  # Center
        (0, 0), (0, 2), (2, 0), (2, 2),  # Corners
        (0, 1), (1, 0), (1, 2), (2, 1)   # Edges
    ]
    for cell in priority_cells:
        if cell in empty:
            return cell

    # Fallback (should never be reached as all cells are checked)
    return empty[0]
```

### Explanation
1. **Win Check**: The `check_win` function evaluates if a player has three markers in a row, column, or diagonal. This is used to determine if a move results in an immediate win.
2. **Empty Cells Collection**: The AI starts by identifying all empty cells on the board.
3. **Immediate Win**: The AI simulates placing its marker in each empty cell and checks if it results in a win. If so, that cell is chosen.
4. **Block Opponent**: If no immediate win is found, the AI checks if placing the opponent's marker in any empty cell would result in a win for the opponent. If found, the AI blocks that move.
5. **Strategic Position**: If neither a win nor a block is needed, the AI chooses the best available cell based on the predefined priority list, ensuring optimal positioning.

This method ensures that the AI always makes the best possible move, leading to an optimal game strategy.
