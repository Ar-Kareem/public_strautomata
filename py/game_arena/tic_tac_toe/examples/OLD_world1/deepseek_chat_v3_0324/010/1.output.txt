from typing import Optional, Tuple

def minimax(board: np.ndarray, is_maximizing: bool, alpha: float, beta: float) -> float:

                    best_score = min(score, best_score)

def check_winner(board: np.ndarray) -> Optional[float]:
    # Check rows
    for i in range(3):
        if np.all(board[i, :] == 1):
            return 1
        if np.all(board[i, :] == -1):
            return -1
    
    # Check columns
    for j in range(3):
        if np.all(board[:, j] == 1):
            return 1
        if np.all(board[:, j] == -1):
            return -1

    if board[0, 0] == board[1, 1] == board[2, 2] != 0:
        return board[0, 0]

    if np.all(board != 0):

    return None

This implementation:

1. Uses minimax with alpha-beta pruning to find the optimal move
2. Always plays optimally (either wins or draws)
3. Makes the best first move (center) if the board is empty
4. Evaluates all possible moves recursively
5. Returns the move coordinates as a tuple (row, column)

The algorithm evaluates all possible future game states and chooses the move that maximizes the chance of winning while minimizing the chance of losing. The alpha-beta pruning optimizes the search by cutting off branches that can't possibly affect the final decision.
