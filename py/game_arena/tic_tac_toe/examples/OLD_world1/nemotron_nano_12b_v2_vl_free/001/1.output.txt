To solve this problem, we need to develop a strategy for an AI player in a Tic Tac Toe game that plays optimally. The goal is to ensure the AI maximizes its chances of winning by identifying immediate winning moves, blocking the opponent's winning moves, and using a heuristic-based strategy for suboptimal board states.

### Approach
The approach to solve this problem involves three main steps:
1. **Check for Winning Move**: The AI first checks if it can place its symbol in any cell to win the game immediately.
2. **Check for Blocking Move**: If no immediate win is possible, the AI then checks if the opponent can win in their next turn and blocks that move.
3. **Heuristic Move Selection**: If neither a winning nor a blocking move is possible, the AI selects the best available move based on a predefined priority order (center first, then corners, then edges).

### Solution Code
```python
def policy(board):
    def is_winning(row, col, player):
        # Check row
        count_row = sum(board[row][j] == player for j in range(3) if j != col)
        if count_row == 2:
            return True
        # Check column
        count_col = sum(board[i][col] == player for i in range(3) if i != row)
        if count_col == 2:
            return True
        # Check main diagonal
        if row == col:
            count_diag = sum(board[i][i] == player for i in (0, 1, 2) if i != row)
            if count_diag == 2:
                return True
        # Check anti-diagonal
        if row + col == 2:
            count_anti_diag = sum(board[i][2 - i] == player for i in range(3) if i != row)
            if count_anti_diag == 2:
                return True
        return False

    # Check for winning moves (player 1)
    for r in range(3):
        for c in range(3):
            if board[r][c] == 0 and is_winning(r, c, 1):
                return (r, c)
    # Check for blocking moves (opponent -1)
    for r in range(3):
        for c in range(3):
            if board[r][c] == 0 and is_winning(r, c, -1):
                return (r, c)
    # Heuristic selection
    priority_positions = [(1,1), (0,0), (0,2), (2,0), (2,2), (0,1), (1,0), (1,2), (2,1)]
    for pos in priority_positions:
        r, c = pos
        if board[r][c] == 0:
            return (r, c)
    # Fallback (shouldn't be needed as board is assumed to have a move)
    return (0, 0)
```

### Explanation
1. **Check for Winning Move**: The AI iterates through each empty cell and checks if placing its symbol (1) there results in a win. This is done by checking all rows, columns, and diagonals that include the cell to see if the other two cells in the line are already occupied by the AI's symbol.

2. **Check for Blocking Move**: If no winning move is found, the AI checks for any cells where the opponent's symbol (-1) would result in a win if placed there. If such a cell is found, the AI blocks that move.

3. **Heuristic Move Selection**: If no immediate win or block is possible, the AI uses a priority-based heuristic to choose the best available move. The center is prioritized first, followed by the corners (in a specific order), and then the edges. This heuristic is designed to maximize the AI's chances of creating winning opportunities or forcing a draw.

This approach ensures that the AI plays optimally by always taking or blocking immediate winning opportunities before resorting to a well-structured heuristic strategy.
