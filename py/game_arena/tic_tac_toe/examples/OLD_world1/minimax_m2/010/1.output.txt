

To solve this problem, we need to determine the optimal next move for a Tic Tac Toe game where the AI is playing against an opponent. The board is represented as a 3x3 grid with values `0` (empty), `1` (AI's piece), and `-1` (opponent's piece). The goal is to select the move that maximizes the AI's chance of winning, minimizes the opponent's chance, or results in a draw.

### Approach
1. **Immediate Win Check**: First, the AI checks if placing its piece (`1`) in any empty cell results in a win. If such a move exists, it is chosen immediately.
2. **Immediate Block Check**: If no immediate win is possible, the AI checks if placing the opponent's piece (`-1`) in any empty cell would result in a win for the opponent. If so, the AI places its piece in that cell to block the opponent.
3. **Minimax Algorithm with Memoization**: For all other scenarios, the AI uses the minimax algorithm. This algorithm explores all possible moves recursively, simulating the opponent's best responses. It assigns utility values to outcomes: `1` for an AI win, `-1` for a loss, and `0` for a draw. The AI selects the move that maximizes the utility value (from its perspective).
4. **Memoization**: To optimize the minimax algorithm, we use memoization to cache results of previously computed board states and player turns. This avoids redundant calculations and significantly reduces computational overhead, especially for deeper game states.

### Solution Code
```python
def check_winner(board_tuple):
    lines = [
        (0, 1, 2),
        (3, 4, 5),
        (6, 7, 8),
        (0, 3, 6),
        (1, 4, 7),
        (2, 5, 8),
        (0, 4, 8),
        (2, 4, 6)
    ]
    for a, b, c in lines:
        if board_tuple[a] != 0 and board_tuple[a] == board_tuple[b] == board_tuple[c]:
            return board_tuple[a]
    if 0 not in board_tuple:
        return 0
    return None

from functools import lru_cache

@lru_cache(maxsize=None)
def minimax(board_tuple, current_player):
    result = check_winner(board_tuple)
    if result is not None:
        return result

    if current_player == 1:
        best_score = -2
        for i in range(9):
            if board_tuple[i] == 0:
                new_board = list(board_tuple)
                new_board[i] = 1
                new_board_tuple = tuple(new_board)
                score = minimax(new_board_tuple, -1)
                if score > best_score:
                    best_score = score
                if best_score == 1:
                    break
        return best_score
    else:
        best_score = 2
        for i in range(9):
            if board_tuple[i] == 0:
                new_board = list(board_tuple)
                new_board[i] = -1
                new_board_tuple = tuple(new_board)
                score = minimax(new_board_tuple, 1)
                if score < best_score:
                    best_score = score
                if best_score == -1:
                    break
        return best_score

def policy(board):
    board_list = []
    for row in board:
        board_list.extend(row)
    board_tuple = tuple(board_list)
    
    for i in range(9):
        if board_tuple[i] == 0:
            new_board = list(board_tuple)
            new_board[i] = 1
            new_board_tuple = tuple(new_board)
            if check_winner(new_board_tuple) == 1:
                return (i // 3, i % 3)
    
    for i in range(9):
        if board_tuple[i] == 0:
            new_board = list(board_tuple)
            new_board[i] = -1
            new_board_tuple = tuple(new_board)
            if check_winner(new_board_tuple) == -1:
                return (i // 3, i % 3)
    
    best_move = None
    best_score = -2
    for i in range(9):
        if board_tuple[i] == 0:
            new_board = list(board_tuple)
            new_board[i] = 1
            new_board_tuple = tuple(new_board)
            score = minimax(new_board_tuple, -1)
            if score > best_score:
                best_score = score
                best_move = (i // 3, i % 3)
    
    return best_move
```

### Explanation
1. **check_winner Function**: This function checks if there is a winner by examining all possible winning lines (rows, columns, diagonals). It returns `1` if the AI wins, `-1` if the opponent wins, `0` for a draw, or `None` if the game is ongoing.
2. **minimax Function with Memoization**: This function uses the minimax algorithm to evaluate the board state from the perspective of the current player. It returns the utility value (1, -1, or 0) for the best possible outcome. The `lru_cache` decorator memoizes results to optimize performance.
3. **policy Function**: This function processes the board, checks for immediate wins and blocks, and uses the minimax algorithm to determine the best move. It converts the 2D board into a 1D tuple for easier manipulation and uses memoization to avoid redundant calculations.
The approach efficiently combines immediate win/block checks with minimax and memoization to ensure optimal performance and correct decision-making in all game scenarios.
