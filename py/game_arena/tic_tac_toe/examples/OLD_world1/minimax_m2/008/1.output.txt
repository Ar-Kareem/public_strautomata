To solve this problem, we need to determine the optimal move for a Tic Tac Toe game using the Minimax algorithm. The algorithm evaluates all possible moves and their consequences, considering the opponent's best responses, to choose the move that maximizes the player's chances of winning while minimizing the opponent's.

### Approach
1. **Problem Analysis**: The Tic Tac Toe board is a 3x3 grid where `1` represents the player, `-1` represents the opponent, and `0` represents empty cells. The goal is to make a move that leads to a win or at least a draw if winning is impossible.
2. **Key Insight**: The Minimax algorithm is used to explore all possible game states recursively. The player aims to maximize their score, while the opponent aims to minimize it.
3. **Scoring System**:
   - **Winning**: If the player wins, the score is `10 - depth` (where depth is the number of moves made so far), encouraging quicker wins.
   - **Losing**: If the opponent wins, the score is `depth - 10`, encouraging the opponent to win as late as possible.
   - **Draw**: If the game results in a draw, the score is `0`.
4. **Algorithm Selection**: The Minimax algorithm is chosen for its ability to handle zero-sum games like Tic Tac Toe, where one player's gain is another player's loss.

### Solution Code
```python
INF = 10**9

def check_winner(board):
    for i in range(3):
        if board[i][0] == board[i][1] == board[i][2] != 0:
            return board[i][0]
    for j in range(3):
        if board[0][j] == board[1][j] == board[2][j] != 0:
            return board[0][j]
    if board[0][0] == board[1][1] == board[2][2] != 0:
        return board[0][0]
    if board[0][2] == board[1][1] == board[2][0] != 0:
        return board[0][2]
    if all(board[i][j] != 0 for i in range(3) for j in range(3)):
        return 0
    return None

def minimax(board, depth, is_maximizing_player):
    winner = check_winner(board)
    if winner is not None:
        if winner == 1:
            return 10 - depth
        elif winner == -1:
            return depth - 10
        elif winner == 0:
            return 0
    
    if is_maximizing_player:
        best = -INF
        for i in range(3):
            for j in range(3):
                if board[i][j] == 0:
                    board[i][j] = 1
                    score = minimax(board, depth + 1, False)
                    board[i][j] = 0
                    best = max(best, score)
        return best
    else:
        best = INF
        for i in range(3):
            for j in range(3):
                if board[i][j] == 0:
                    board[i][j] = -1
                    score = minimax(board, depth + 1, True)
                    board[i][j] = 0
                    best = min(best, score)
        return best

def policy(board):
    best_move = None
    best_score = -INF
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                board[i][j] = 1
                score = minimax(board, 1, False)
                board[i][j] = 0
                if score > best_score:
                    best_score = score
                    best_move = (i, j)
    return best_move
```

### Explanation
1. **check_winner Function**: This function checks if there's a winner or a draw by examining all rows, columns, and diagonals. It returns `1` if the player wins, `-1` if the opponent wins, `0` for a draw, or `None` if the game is ongoing.
2. **minimax Function**: This recursive function evaluates all possible moves. For the maximizing player (player), it chooses the move with the highest score. For the minimizing player (opponent), it chooses the move with the lowest score. The depth parameter helps in prioritizing quicker wins and delaying losses.
3. **policy Function**: This function determines the best move by simulating each possible move, evaluating the resulting game state using Minimax, and selecting the move that leads to the highest score for the player.

This approach ensures the player makes optimal moves, considering all possible opponent responses, leading to the best possible outcome in the game.
