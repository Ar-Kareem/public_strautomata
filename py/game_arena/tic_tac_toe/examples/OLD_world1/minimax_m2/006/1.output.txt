To solve this problem, we need to create a Python function that determines the optimal next move for a 3x3 Tic Tac Toe game. The function should analyze the current board state and choose the best move for the player represented by '1' (the AI), assuming the opponent is represented by '-1'. The solution uses a minimax algorithm with alpha-beta pruning to efficiently evaluate all possible moves and choose the one that maximizes the AI's chances of winning or forcing a draw.

### Approach
1. **Problem Analysis**: The Tic Tac Toe board is a 3x3 grid where each cell can be empty (0), occupied by the AI (1), or occupied by the opponent (-1). The goal is to find the optimal move for the AI that maximizes its chances of winning while minimizing the opponent's chances.
2. **Key Insight**: The minimax algorithm is suitable for this problem because it explores all possible moves in the game tree, evaluating the best outcome for the current player assuming optimal play from both sides. Alpha-beta pruning is used to reduce the number of nodes evaluated, thus optimizing performance.
3. **Algorithm Selection**: The minimax algorithm with alpha-beta pruning is implemented. The algorithm works as follows:
   - **Terminal State Check**: The algorithm first checks if the current board state is a terminal state (win, loss, or draw).
   - **Heuristic Scoring**: Terminal states are assigned scores based on the depth of the game tree and the winner:
     - **Win for AI**: `10 - depth` (faster wins are better).
     - **Loss for AI**: `depth - 10` (slower losses are better).
     - **Draw**: `0`.
   - **Minimax with Alpha-Beta Pruning**:
     - **Maximizing Player (AI)**: Tries to maximize the score.
     - **Minimizing Player (Opponent)**: Tries to minimize the score.
     - **Pruning**: Alpha-beta pruning cuts off branches that cannot improve the outcome, reducing computational overhead.

### Solution Code
```python
import math

def get_winner(board):
    for i in range(3):
        if board[i][0] == board[i][1] == board[i][2] != 0:
            return board[i][0]
    for j in range(3):
        if board[0][j] == board[1][j] == board[2][j] != 0:
            return board[0][j]
    if board[0][0] == board[1][1] == board[2][2] != 0:
        return board[0][0]
    if board[0][2] == board[1][1] == board[2][0] != 0:
        return board[0][2]
    return 0

def is_draw(board):
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                return False
    return True

def minimax(board, depth, is_maximizing, alpha, beta):
    winner = get_winner(board)
    if winner == 1:
        return 10 - depth
    elif winner == -1:
        return depth - 10
    elif is_draw(board):
        return 0

    if is_maximizing:
        max_eval = -math.inf
        for i in range(3):
            for j in range(3):
                if board[i][j] == 0:
                    board[i][j] = 1
                    eval_score = minimax(board, depth+1, False, alpha, beta)
                    board[i][j] = 0
                    if eval_score > max_eval:
                        max_eval = eval_score
                    if eval_score > alpha:
                        alpha = eval_score
                    if beta <= alpha:
                        break
            if beta <= alpha:
                break
        return max_eval
    else:
        min_eval = math.inf
        for i in range(3):
            for j in range(3):
                if board[i][j] == 0:
                    board[i][j] = -1
                    eval_score = minimax(board, depth+1, True, alpha, beta)
                    board[i][j] = 0
                    if eval_score < min_eval:
                        min_eval = eval_score
                    if eval_score < beta:
                        beta = eval_score
                    if beta <= alpha:
                        break
            if beta <= alpha:
                break
        return min_eval

def policy(board: list[list[int]]) -> tuple[int, int]:
    best_score = -math.inf
    best_move = None

    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                board[i][j] = 1
                score = minimax(board, 1, False, -math.inf, math.inf)
                board[i][j] = 0
                if score > best_score:
                    best_score = score
                    best_move = (i, j)

    return best_move
```

### Explanation
1. **get_winner**: This helper function checks all possible winning lines (rows, columns, and diagonals) to determine if there's a winner (either '1' or '-1').
2. **is_draw**: This function checks if the board is full without any winner, indicating a draw.
3. **minimax**: The core algorithm that evaluates the game tree:
   - **Terminal States**: If the game is over, it returns a score based on the outcome and the depth of the move.
   - **Maximizing Player (AI)**: For each possible move, it evaluates the score and maximizes the result, pruning branches where the opponent can force a worse outcome.
   - **Minimizing Player (Opponent)**: Similar to the maximizing player but minimizes the score to choose the most detrimental move for the AI.
4. **policy**: The main function that calls minimax for each possible move of the AI. It selects the move that results in the highest score, ensuring the AI plays optimally. The board is modified in-place and restored after each evaluation to avoid side effects. The solution efficiently computes the best move using minimax with alpha-beta pruning, ensuring optimal performance even in the worst-case scenario.
