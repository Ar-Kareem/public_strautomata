To solve this problem, we need to determine the optimal move for a player in a 3x3 Tic Tac Toe game. The solution must prioritize winning moves, block the opponent's winning moves, and then choose the best available position based on strategic priorities (center, corners, edges).

### Approach
1. **Check for Winning Move**: First, check if placing the player's mark (1) in any empty cell results in an immediate win. If so, select that cell.
2. **Block Opponent's Winning Move**: If no winning move is available, check if the opponent (-1) has a move that would result in their win. If found, block that cell.
3. **Strategic Position Selection**: If neither a winning nor a blocking move is available, choose the best available position. The priority order is:
   - **Center**: The most strategic position.
   - **Corners**: Next best positions.
   - **Edges**: Least preferred but still valid positions.

### Solution Code
```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    def is_win(b, player):
        # Check rows
        for row in b:
            if all(cell == player for cell in row):
                return True
        # Check columns
        for col in range(3):
            if all(b[row][col] == player for row in range(3)):
                return True
        # Check diagonals
        if all(b[i][i] == player for i in range(3)):
            return True
        if all(b[i][2 - i] == player for i in range(3)):
            return True
        return False
    
    # Check for winning move for player 1
    for row in range(3):
        for col in range(3):
            if board[row][col] == 0:
                new_board = [r.copy() for r in board]
                new_board[row][col] = 1
                if is_win(new_board, 1):
                    return (row, col)
    
    # Check for opponent's winning move to block
    for row in range(3):
        for col in range(3):
            if board[row][col] == 0:
                new_board = [r.copy() for r in board]
                new_board[row][col] = -1
                if is_win(new_board, -1):
                    return (row, col)
    
    # Choose center if available
    if board[1][1] == 0:
        return (1, 1)
    
    # Check corners in order
    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    for r, c in corners:
        if board[r][c] == 0:
            return (r, c)
    
    # Check edges in order
    edges = [(0, 1), (1, 0), (1, 2), (2, 1)]
    for r, c in edges:
        if board[r][c] == 0:
            return (r, c)
    
    # Fallback return (should not be reached with valid input)
    return (0, 0)
```

### Explanation
1. **Winning Move Check**: The function iterates over each empty cell, simulates placing the player's mark, and checks if this results in a win using the `is_win` helper function. If a winning move is found, it is returned immediately.
2. **Blocking Opponent's Win**: Similarly, the function checks if the opponent can win by placing their mark in any empty cell. If so, the function blocks that move by returning the corresponding cell.
3. **Strategic Position Selection**: If no immediate wins or blocks are found, the function selects the best available position starting with the center, followed by corners, and finally edges. This ensures the player adopts the most strategic positions to maximize chances of winning.

This approach efficiently balances aggressive play (seeking wins) with defensive play (blocking opponents) while adhering to optimal strategic positioning in Tic Tac Toe.
