To create a competitive AI for 4x4 Tic Tac Toe, we need a strategy that:
1. First checks for immediate winning moves (to win)
2. Then blocks opponent's winning moves
3. Otherwise makes strategic moves to create potential winning opportunities

For a 4x4 grid, the winning condition is 4 in a row/column/diagonal. We'll need to:
- Scan all rows, columns, and diagonals for potential wins/blocks
- Prioritize center and corner positions for better strategic advantage
- Implement a simple evaluation function to choose the best available move when no immediate wins/blocks exist

The algorithm will:
1. First check if AI can win in the next move
2. Then check if opponent can win next move and block
3. Otherwise choose the best available position based on:
   - Center positions first
   - Then corners
   - Then edges
   - With preference for positions that create multiple potential lines

This approach balances defensive and offensive play while being computationally efficient enough to run within 1 second.

import numpy as np

def policy(board: list[list[int]]) -> tuple[int, int]:
    board_array = np.array(board)
    size = 4
    
    # Check all possible lines for a win or block opportunity
    def check_lines(value):
        # Check rows
        for i in range(size):
            if np.sum(board_array[i, :] == value) == 3:
                for j in range(size):
                    if board_array[i, j] == 0:
                        return (i, j)
        
        # Check columns
        for j in range(size):
            if np.sum(board_array[:, j] == value) == 3:
                for i in range(size):
                    if board_array[i, j] == 0:
                        return (i, j)
        
        # Check diagonals
        if np.sum(np.diag(board_array) == value) == 3:
            for i in range(size):
                if board_array[i, i] == 0:
                    return (i, i)
        
        if np.sum(np.diag(np.fliplr(board_array)) == value) == 3:
            for i in range(size):
                if board_array[i, size-1-i] == 0:
                    return (i, size-1-i)
        
        return None
    
    # 1. Check if AI can win immediately
    move = check_lines(1)
    if move is not None:
        return move
    
    # 2. Check if opponent can win and block
    move = check_lines(-1)
    if move is not None:
        return move
    
    # 3. Strategic position selection
    # Priority: center > corners > edges
    priority_positions = [
        (1,1), (1,2), (2,1), (2,2),  # Center positions
        (0,0), (0,3), (3,0), (3,3),  # Corners
        (0,1), (0,2), (1,0), (1,3),  # Edges
        (2,0), (2,3), (3,1), (3,2)   # Edges
    ]
    
    for pos in priority_positions:
        if board_array[pos] == 0:
            return pos
    
    # Fallback (shouldn't reach here if board isn't full)
    for i in range(size):
        for j in range(size):
            if board_array[i, j] == 0:
                return (i, j)
    
    return (0, 0)  # Should never reach here for valid boards
