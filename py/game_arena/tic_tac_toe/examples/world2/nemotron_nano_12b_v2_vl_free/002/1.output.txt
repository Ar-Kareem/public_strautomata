To solve the problem of determining the next optimal move in a 4x4 Tic Tac Toe game, we need to build a function that evaluates the board state and decides the best possible move for the AI. The AI should aim to win immediately, block the opponent's winning moves, and otherwise play strategically using a heuristic.

### Approach
1. **Winning Move Check**: The AI first checks all possible moves to see if placing its mark (1) will result in a win. This is done by simulating placing the mark on each empty cell and checking if that move completes a line of four.
2. **Blocking Opponent's Win**: If no winning move is found, the AI checks if the opponent can win on their next move. If such a move exists, the AI blocks it by placing its mark in that cell.
3. **Heuristic Selection**: If neither a winning nor a blocking move is found, the AI uses a heuristic to prioritize cells with more potential lines (row, column, diagonals). The center cells (2x2 grid) are prioritized over corners and then edges.

### Solution Code
```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    def is_win_move(r: int, c: int, player: int) -> bool:
        # Check if placing player at (r,c) would result in a win.
        # Check row r
        row_win = True
        for i in range(4):
            if i != c and board[r][i] != player:
                row_win = False
                break
        if row_win:
            return True
        
        # Check column c
        col_win = True
        for i in range(4):
            cell = board[i][c]
            if i != r and cell != player:
                col_win = False
                break
        if col_win:
            return True
        
        # Check main diagonal (if cell is on it)
        if r == c:
            diag_win = True
            for i in range(4):
                cell = board[i][i]
                if i != r and cell != player:
                    diag_win = False
                    break
            if diag_win:
                return True
        
        # Check anti-diagonal (if cell is on it)
        if r + c == 3:
            anti_diag_win = True
            for i in range(4):
                cell = board[i][3 - i]
                if i != r and cell != player:
                    anti_diag_win = False
                    break
            if anti_diag_win:
                return True
        
        return False

    # Step 1: Check for AI's winning move
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0:
                if is_win_move(r, c, 1):
                    return (r, c)
    
    # Step 2: Check for opponent's winning move to block
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0:
                if is_win_move(r, c, -1):
                    return (r, c)
    
    # Step 3: Heuristic-based move selection
    empty_cells = []
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0:
                line_count = 2  # All cells have row and column
                # Check if part of main diagonal
                if r == c:
                    line_count += 1
                # Check if part of anti-diagonal
                if r + c == 3:
                    line_count += 1
                # Assign group based on cell position
                if r in {1, 2} and c in {1, 2}:
                    group = 0  # Center cells have higher priority
                elif r in {0, 3} and c in {0, 3}:
                    group = 1  # Corners next
                else:
                    group = 2  # Edges last
                # Append with negative line_count for descending sort
                empty_cells.append((-line_count, group, r, c))
    
    # Sort by line_count (desc), group (asc), then row and column
    empty_cells.sort()
    best_move = empty_cells[0]
    return (best_move[2], best_move[3])
```

### Explanation
1. **Winning Move Check**: The helper function `is_win_move` checks if placing a mark at a given cell would result in a win by examining the respective row, column, and diagonals. This avoids creating temporary boards and directly checks the current board state, improving efficiency.
2. **Blocking Opponent's Win**: If the AI cannot win immediately, it checks if the opponent can win on their next move. If such a threat exists, the AI blocks it.
3. **Heuristic Selection**: For cells that are not immediately winning or blocking, the AI prioritizes cells that are part of the most lines (center cells first, then corners, then edges). This heuristic increases the chances of winning by controlling more potential lines.

This approach ensures optimal play by prioritizing moves that lead to a win, block the opponent's win, and strategically position the AI's mark for future opportunities.
