**Rule-based approach:**

*   **Player ID:** My player is `1`, opponent is `-1`.
*   **Winning condition:** 4 in a row/column/diagonal.

**Steps:**

1.  **Iterate through all empty cells (0,0) to (3,3):**
    *   For each empty cell `(r, c)`:
        *   **Try to win:** Place my piece (`1`) at `(r, c)`. Check if `1` wins. If yes, return `(r, c)`.
        *   **Undo the move:** Reset `board[r][c]` to `0`.
        *   **Try to block opponent:** Place opponent's piece (`-1`) at `(r, c)`. Check if `-1` wins. If yes, return `(r, c)`. (This blocks the opponent)
        *   **Undo the move:** Reset `board[r][c]` to `0`.

2.  **If no immediate win or block, look for strategic moves:**
    *   **Center preference:** If any of the center squares (1,1), (1,2), (2,1), (2,2) are empty, pick one.
    *   **Corners preference:** If any corners (0,0), (0,3), (3,0), (3,3) are empty, pick one.
    *   **Any open square:** Otherwise, pick the first available empty square.

This rule-based approach is often fast enough for small boards.
