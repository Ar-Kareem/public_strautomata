The approach to determine the next move in the 4x4 Tic Tac Toe game involves several prioritized steps to ensure optimal play:
1. **Immediate Win Check**: First, the AI checks if placing its mark (1) in any empty cell results in an immediate win (four in a row, column, or diagonal). If found, it returns that winning move.
2. **Block Opponent's Win**: If no winning move is found, the AI checks if the opponent (-1) is one move away from winning and blocks them by occupying that cell.
3. **Occupy Center Cells**: If neither an immediate win nor a block is needed, the AI prioritizes occupying the central cells (1,1), (1,2), (2,1), (2,2) as they offer more strategic advantages for creating multiple winning opportunities.
4. **Choose Corners**: If all centers are occupied, the AI selects an empty corner cell.
5. **Fallback to Any Empty Cell**: Finally, if none of the above conditions are met, the AI returns the first available empty cell.

This strategy ensures the AI plays defensively by blocking opponent threats while looking for opportunities to win, leveraging the center's strategic value, and falling back to corners or other cells when necessary. The solution is efficient for the 4x4 grid and adheres to the 1-second execution time constraint.

def policy(board: list[list[int]]) -> tuple[int, int]:
    def can_win(board, i, j, player):
        # Check row
        if sum(board[i]) == player * 3:
            return True
        # Check column
        if sum(board[row][j] for row in range(4)) == player * 3:
            return True
        # Check main diagonal
        if i == j:
            diag_sum = sum(board[k][k] for k in range(4))
            if diag_sum == player * 3:
                return True
        # Check anti-diagonal
        if i + j == 3:
            anti_diag_sum = sum(board[k][3 - k] for k in range(4))
            if anti_diag_sum == player * 3:
                return True
        return False

    # Check for winning move
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                if can_win(board, i, j, 1):
                    return (i, j)
    
    # Check for blocking opponent's win
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                if can_win(board, i, j, -1):
                    return (i, j)
    
    # Prioritize center cells
    centers = [(1, 1), (1, 2), (2, 1), (2, 2)]
    for c in centers:
        if board[c[0]][c[1]] == 0:
            return c
    
    # Choose any corner
    corners = [(0, 0), (0, 3), (3, 0), (3, 3)]
    for c in corners:
        if board[c[0]][c[1]] == 0:
            return c
    
    # Choose first available cell
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                return (i, j)
    
    # If no moves are possible (shouldn't happen)
    return (0, 0)
