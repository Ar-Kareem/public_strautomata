# Strategy

The strategy for this policy is a combination of immediate win/block checks, center control, and a simple evaluation function for deeper lookahead. Here's the breakdown:

1. **Immediate Win/Block**: First check if we can win immediately or if the opponent can win next turn (which we must block).

2. **Center Control**: Prioritize the center squares (1,1), (1,2), (2,1), (2,2) as they provide the most strategic advantage in a 4x4 grid.

3. **Corner Control**: If center is taken, prioritize corners (0,0), (0,3), (3,0), (3,3) as they offer good expansion opportunities.

4. **Evaluation Function**: For remaining moves, use a simple evaluation that:
   - Counts potential lines (rows, columns, diagonals) we could complete
   - Counts opponent's potential lines to block
   - Prefers moves that create multiple threats

5. **Fallback**: If no strategic move is found, default to the first available empty cell.

This approach balances immediate tactical responses with positional play, making it competitive against other policies.

import numpy as np

def evaluate_line(line):
    """Evaluate a line (row, column, or diagonal) for potential threats."""
    count = sum(line)
    if count == 3:  # We can win
        return 100
    elif count == -3:  # Opponent can win
        return -100
    elif count == 2:  # We have two in a row
        return 10
    elif count == -2:  # Opponent has two in a row
        return -10
    return 0

def evaluate_board(board):
    """Evaluate the entire board for potential threats."""
    score = 0

    # Check rows
    for row in board:
        score += evaluate_line(row)

    # Check columns
    for col in range(4):
        column = [board[row][col] for row in range(4)]
        score += evaluate_line(column)

    # Check diagonals
    diag1 = [board[i][i] for i in range(4)]
    diag2 = [board[i][3-i] for i in range(4)]
    score += evaluate_line(diag1)
    score += evaluate_line(diag2)

    return score

def policy(board):
    # Check for immediate win
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                board[i][j] = 1
                if evaluate_board(board) >= 100:
                    board[i][j] = 0
                    return (i, j)
                board[i][j] = 0

    # Check for immediate block
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                board[i][j] = -1
                if evaluate_board(board) <= -100:
                    board[i][j] = 0
                    return (i, j)
                board[i][j] = 0

    # Center control
    center_moves = [(1,1), (1,2), (2,1), (2,2)]
    for move in center_moves:
        if board[move[0]][move[1]] == 0:
            return move

    # Corner control
    corner_moves = [(0,0), (0,3), (3,0), (3,3)]
    for move in corner_moves:
        if board[move[0]][move[1]] == 0:
            return move

    # Find best move based on evaluation
    best_score = -float('inf')
    best_move = None
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                board[i][j] = 1
                score = evaluate_board(board)
                board[i][j] = 0
                if score > best_score:
                    best_score = score
                    best_move = (i, j)

    if best_move:
        return best_move

    # Fallback: return first empty cell
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                return (i, j)

    # Should never reach here as board must have empty cells
    return (0, 0)
