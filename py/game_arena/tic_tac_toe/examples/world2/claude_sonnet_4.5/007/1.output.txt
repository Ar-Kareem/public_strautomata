For a 4x4 Tic Tac Toe game, I need to create a strong policy function. The key considerations are:

1. **Win condition**: In 4x4 Tic Tac Toe, I assume we need 4 in a row (horizontally, vertically, or diagonally) to win.

2. **Strategy priorities**:
   - If I can win immediately, take that move
   - If opponent can win on their next move, block them
   - Otherwise, make a strategic move (center positions are typically stronger, build threats)

3. **Implementation approach**:
   - Check all possible winning lines (rows, columns, diagonals)
   - For each line, count my pieces, opponent pieces, and empty spaces
   - Prioritize moves that either complete my win or block opponent's win
   - Use heuristics for other moves (prefer center, create multiple threats)

def policy(board: list[list[int]]) -> tuple[int, int]:
    # Check for immediate win or block
    # Then use strategic positioning
    
    # First, find all empty positions
    empty_positions = []
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                empty_positions.append((i, j))
    
    if not empty_positions:
        return (0, 0)  # Shouldn't happen in valid game
    
    # Define all possible winning lines
    lines = []
    
    # Rows
    for i in range(4):
        lines.append([(i, 0), (i, 1), (i, 2), (i, 3)])
    
    # Columns
    for j in range(4):
        lines.append([(0, j), (1, j), (2, j), (3, j)])
    
    # Diagonals
    lines.append([(0, 0), (1, 1), (2, 2), (3, 3)])
    lines.append([(0, 3), (1, 2), (2, 1), (3, 0)])
    
    # Function to evaluate a move
    def evaluate_move(pos):
        r, c = pos
        score = 0
        
        # Check each line containing this position
        for line in lines:
            if pos in line:
                my_count = 0
                opp_count = 0
                empty_count = 0
                
                for lr, lc in line:
                    if board[lr][lc] == 1:
                        my_count += 1
                    elif board[lr][lc] == -1:
                        opp_count += 1
                    else:
                        empty_count += 1
                
                # If I place here, my_count increases by 1, empty_count decreases by 1
                if (r, c) == pos and board[r][c] == 0:
                    my_count_after = my_count + 1
                    opp_count_after = opp_count
                    empty_count_after = empty_count - 1
                    
                    # Winning move
                    if my_count_after == 4:
                        score += 10000
                    # Block opponent's winning move
                    elif opp_count == 3 and empty_count == 1:
                        score += 5000
                    # Create threat (3 in a row)
                    elif my_count_after == 3 and opp_count_after == 0:
                        score += 100
                    # Build up (2 in a row)
                    elif my_count_after == 2 and opp_count_after == 0:
                        score += 10
                    # Block opponent's 2 in a row
                    elif opp_count == 2 and my_count == 0 and empty_count == 2:
                        score += 5
        
        # Prefer center positions
        if r in [1, 2] and c in [1, 2]:
            score += 3
        
        return score
    
    # Evaluate all empty positions
    best_score = -1
    best_move = empty_positions[0]
    
    for pos in empty_positions:
        score = evaluate_move(pos)
        if score > best_score:
            best_score = score
            best_move = pos
    
    return best_move
