For a 4x4 Tic Tac Toe game, I need to create a policy that returns good moves. The winning condition is typically 4 in a row (horizontally, vertically, or diagonally).

Key strategies:
1. Win immediately if possible (check for 3 in a row with an empty spot)
2. Block opponent from winning (check for opponent's 3 in a row)
3. Create threats (get 2 or 3 in a row)
4. Take center or strategic positions
5. Use minimax or heuristic evaluation

Given the 1-second time constraint and 4x4 board (16 positions), I can implement a minimax algorithm with alpha-beta pruning. However, for simplicity and reliability, I'll implement a strong heuristic-based approach:

1. Check if we can win in one move
2. Check if we need to block opponent's winning move
3. Look for moves that create multiple threats
4. Use a position scoring system (center is valuable, corners are good)

```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    # Check all lines (rows, cols, diagonals) for immediate win or block
    # Then use heuristic scoring
    
    def check_line_for_win(line_positions, player):
        """Check if a line has 3 of player and 1 empty, return empty position"""
        values = [board[r][c] for r, c in line_positions]
        if values.count(player) == 3 and values.count(0) == 1:
            empty_idx = values.index(0)
            return line_positions[empty_idx]
        return None
    
    def get_all_lines():
        """Get all winning lines (rows, cols, diagonals)"""
        lines = []
        # Rows
        for r in range(4):
            lines.append([(r, c) for c in range(4)])
        # Columns
        for c in range(4):
            lines.append([(r, c) for r in range(4)])
        # Diagonals
        lines.append([(i, i) for i in range(4)])
        lines.append([(i, 3-i) for i in range(4)])
        return lines
    
    all_lines = get_all_lines()
    
    # 1. Check for immediate win
    for line in all_lines:
        win_move = check_line_for_win(line, 1)
        if win_move:
            return win_move
    
    # 2. Check for blocking opponent's win
    for line in all_lines:
        block_move = check_line_for_win(line, -1)
        if block_move:
            return block_move
    
    # 3. Heuristic scoring for each empty position
    def score_position(r, c):
        if board[r][c] != 0:
            return -1000
        
        score = 0
        # Center positions are valuable
        center_dist = abs(r - 1.5) + abs(c - 1.5)
        score += (6 - center_dist) * 10
        
        # Count how many lines this position participates in
        for line in all_lines:
            if (r, c) in line:
                values = [board[row][col] for row, col in line]
                my_count = values.count(1)
                opp_count = values.count(-1)
                empty_count = values.count(0)
                
                # If line is not blocked by opponent
                if opp_count == 0:
                    score += my_count * my_count * 10
                # If we're blocking opponent
                if my_count == 0:
                    score += opp_count * opp_count * 5
        
        return score
    
    # Find best move by scoring
    best_score = -float('inf')
    best_move = None
    
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0:
                score = score_position(r, c)
                if score > best_score:
                    best_score = score
                    best_move = (r, c)
    
    if best_move:
        return best_move
    
    # Fallback: return first empty position
    for r in range(4):
        for c in range(4):
            if board[r][c] == 0:
                return (r, c)
    
    return (0, 0)
```

This approach:
- Wins immediately when possible
- Blocks opponent's wins
- Prefers central positions
- Scores positions based on offensive and defensive potential
- Has a fallback for edge cases
