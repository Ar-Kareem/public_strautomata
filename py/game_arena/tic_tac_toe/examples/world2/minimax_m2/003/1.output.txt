We are to write a policy for a 4x4 Tic Tac Toe game. The board is represented as a 4x4 list of lists with:
   0 -> empty
   1 -> our mark (we play as 1)
   -1 -> opponent's mark

We need to return a tuple (row, col) for the next move.

Strategy:
- We use a minimax algorithm with alpha-beta pruning to a depth of 4 (which covers 8 plies, meaning 4 moves for each player).
- We precompute all winning lines (10 in total: 4 rows, 4 columns, 2 diagonals).
- We check for an immediate win or block if necessary, but since we are using minimax, we rely on the search.
- The heuristic evaluation function:
   - We assign a high positive score if we win (100000) and a high negative score if we lose (-100000).
   - Otherwise, we score the board by:
        * For each line, if the opponent has any mark, we subtract (10^(number of opponent marks)) from the score.
        * If we have any mark and the opponent has none, we add (10^(number of our marks)) to the score.
   - This heuristic encourages forming our own lines and discourages the opponent's.

Steps:
1. Precompute the 10 winning lines.
2. Define a helper function `check_winner` that checks for a winner (returns 1 for our win, -1 for opponent, 0 for no winner).
3. Define a heuristic function that computes the score for the current board.
4. Define the minimax function with alpha-beta pruning.
5. In the policy function:
   a. Check if the game is over (if either player has won) and return any empty cell.
   b. If there are no moves, return (0,0) arbitrarily.
   c. Call minimax with depth=4 and maximizing player as True (since it's our turn).
   d. Return the best move found.

Note: We assume the board is passed by reference and we backtrack (reset) moves in the minimax.

We set depth=4 because the worst-case branching factor (16 for the first move) leads to 16*15*14*13 = 43680 nodes for the deepest level, and the total nodes (considering all levels) is about 47296, which is acceptable for 1 second in Python.

# Predefined winning lines for 4x4 Tic Tac Toe
lines = [
    [(0,0), (0,1), (0,2), (0,3)],
    [(1,0), (1,1), (1,2), (1,3)],
    [(2,0), (2,1), (2,2), (2,3)],
    [(3,0), (3,1), (3,2), (3,3)],
    [(0,0), (1,0), (2,0), (3,0)],
    [(0,1), (1,1), (2,1), (3,1)],
    [(0,2), (1,2), (2,2), (3,2)],
    [(0,3), (1,3), (2,3), (3,3)],
    [(0,0), (1,1), (2,2), (3,3)],
    [(0,3), (1,2), (2,1), (3,0)]
]

def check_winner(board):
    """Check if there is a winner. Returns 1 if we win, -1 if opponent wins, 0 otherwise."""
    for line in lines:
        # Check for our win (all 1)
        if all(board[i][j] == 1 for (i, j) in line):
            return 1
        # Check for opponent win (all -1)
        if all(board[i][j] == -1 for (i, j) in line):
            return -1
    return 0

def heuristic(board):
    """Evaluate the board and return a score. Positive for our advantage, negative for opponent."""
    score = 0
    for line in lines:
        count1 = 0
        count_minus1 = 0
        for (i, j) in line:
            if board[i][j] == 1:
                count1 += 1
            elif board[i][j] == -1:
                count_minus1 += 1
        # If opponent has any in the line, subtract
        if count_minus1 > 0:
            score -= 10 ** count_minus1
        # If we have any and opponent has none, add
        if count1 > 0 and count_minus1 == 0:
            score += 10 ** count1
    return score

def minimax(board, depth, alpha, beta, maximizing_player):
    """Minimax with alpha-beta pruning. Returns (best_move, score)."""
    winner = check_winner(board)
    if winner == 1:
        return (None, 100000)
    if winner == -1:
        return (None, -100000)
    if depth == 0:
        return (None, heuristic(board))
    
    # Generate possible moves
    moves = []
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                moves.append((i, j))
    
    if not moves:
        return (None, 0)  # Draw
    
    if maximizing_player:
        max_eval = -float('inf')
        best_move = None
        for move in moves:
            i, j = move
            board[i][j] = 1
            eval_score = minimax(board, depth-1, alpha, beta, False)[1]
            board[i][j] = 0  # backtrack
            
            if eval_score > max_eval:
                max_eval = eval_score
                best_move = move
            
            alpha = max(alpha, eval_score)
            if beta <= alpha:
                break  # prune
        
        return (best_move, max_eval)
    else:
        min_eval = float('inf')
        best_move = None
        for move in moves:
            i, j = move
            board[i][j] = -1
            eval_score = minimax(board, depth-1, alpha, beta, True)[1]
            board[i][j] = 0  # backtrack
            
            if eval_score < min_eval:
                min_eval = eval_score
                best_move = move
            
            beta = min(beta, eval_score)
            if beta <= alpha:
                break  # prune
        
        return (best_move, min_eval)

def policy(board):
    """Return the next move (row, col) for our player (1)."""
    # Check if game is already over
    winner = check_winner(board)
    if winner == 1 or winner == -1:
        # Return any empty cell
        for i in range(4):
            for j in range(4):
                if board[i][j] == 0:
                    return (i, j)
    
    # Generate all possible moves
    moves = []
    for i in range(4):
        for j in range(4):
            if board[i][j] == 0:
                moves.append((i, j))
    
    # If no moves left, return arbitrary (0,0)
    if not moves:
        return (0, 0)
    
    # Set depth to 4
    depth = 4
    
    # Get the best move using minimax
    best_move = minimax(board, depth, -float('inf'), float('inf'), True)[0]
    
    # If best_move is None (shouldn't happen if moves exist), return first available
    if best_move is None:
        return moves[0]
    
    return best_move
