To solve this problem, we need to develop a chess-playing policy that selects the best legal move for the current player based on the given board state. The policy must prioritize checkmate moves, then material gains from captures, followed by check moves, and finally the first available move if no other options are available. 

### Approach
1. **Checkmate Check**: First, scan through all legal moves to find any move that results in checkmate (indicated by a '#' at the end of the move string). If found, immediately return this move as it is the most favorable outcome.
2. **Capture Evaluation**: For moves involving captures (indicated by 'x' in the move string), evaluate the net material gain. This involves:
   - **Non-promotion captures**: Calculate the net gain as the value of the captured piece minus the value of the moving piece.
   - **Promotion captures**: Calculate the net gain as the value of the promoted piece minus the value of the pawn (since only pawns can promote) minus the value of the captured piece.
   - Select the capture move with the highest net gain.
3. **Check Move Check**: If no capture moves are found, check for any non-capture moves that result in check (indicated by '+' in the move string). If found, return the first such move.
4. **Default Move**: If no checkmate, capture, or check moves are available, return the first move in the list of legal moves.

### Solution Code
```python
piece_values = {'P': 1, 'N': 3, 'B': 3, 'R': 5, 'Q': 9, 'K': 100}

def policy(pieces, to_play, legal_moves, memory):
    # Step 1: Check for checkmate
    for move in legal_moves:
        if move.endswith('#'):
            return move, {}
    
    # Step 2: Check for captures
    capture_moves = []
    for move in legal_moves:
        if 'x' not in move:
            continue
        
        move_clean = move.rstrip('+#')
        if move_clean.startswith('O-'):
            continue
        
        if '=' in move_clean:
            parts = move_clean.split('=')
            dest_part = parts[0]
            promotion = parts[1]
            dest = dest_part[-2:]
            if 'x' not in dest_part:
                continue
            captured_piece = pieces.get(dest)
            if captured_piece is None:
                continue
            moving_piece_type = 'P'
            net_gain = piece_values[promotion] - piece_values['P'] - piece_values[captured_piece[1]]
        else:
            dest = move_clean[-2:]
            if 'x' not in move_clean:
                continue
            captured_piece = pieces.get(dest)
            if captured_piece is None:
                continue
            first_char = move_clean[0]
            if first_char in 'NBRQK':
                moving_piece_type = first_char
            else:
                moving_piece_type = 'P'
            net_gain = piece_values[captured_piece[1]] - piece_values[moving_piece_type]
        
        capture_moves.append((move, net_gain))
    
    if capture_moves:
        best_move = max(capture_moves, key=lambda x: x[1])
        return best_move[0], {}
    
    # Step 3: Check for check moves (non-capture)
    for move in legal_moves:
        if '+' in move and not '#' and 'x' not in move:
            return move, {}
    
    # Step 4: Return first move if no other moves found
    return legal_moves[0], {}
```

### Explanation
1. **Checkmate Check**: The function first checks each legal move to see if it ends with '#', indicating checkmate. If such a move is found, it is immediately returned.
2. **Capture Evaluation**: For moves involving captures, the function calculates the net material gain:
   - **Non-promotion captures**: The net gain is the value of the captured piece minus the value of the moving piece.
   - **Promotion captures**: The net gain is the value of the promoted piece minus the value of the pawn (since pawns promote) minus the value of the captured piece.
   - The capture move with the highest net gain is selected.
3. **Check Move Check**: If no capture moves are found, the function checks for non-capture moves that end with '+' (indicating check). The first such move found is returned.
4. **Default Move**: If no checkmate, capture, or check moves are available, the first move in the list of legal moves is returned.

This approach ensures the policy prioritizes the most advantageous moves (checkmate, material gain, check) before resorting to any available move, thus providing a strong basis for chess move selection.
