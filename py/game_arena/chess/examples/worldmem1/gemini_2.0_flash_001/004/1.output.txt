The strategy for this policy is a simple one:

1. **Checkmate:** If there's a checkmate move, take it.
2. **Capture High-Value Piece:** Capture the highest-value opponent piece if possible (Q=9, R=5, B=3, N=3, P=1, K=0).
3. **Check:** If no captures of significant material or checkmates are present, try to give a check.
4. **Simple Move:** If none of the above are possible, pick the first legal move.

This policy aims to be fast and reliable, making it suitable for operating within the time constraints of the gaming arena. It gives priority to winning the game outright, or capturing high valued pieces, otherwise making checks.

```python
def policy(pieces: dict[str, str], to_play: str, legal_moves: list[str], memory: dict) -> tuple[str, dict]:
    """
    Chooses the next move based on a simple heuristic policy.
    """

    opponent_color = 'b' if to_play == 'white' else 'w'

    piece_values = {
        'Q': 9,
        'R': 5,
        'B': 3,
        'N': 3,
        'P': 1,
        'K': 0,
    }

    def get_piece_value(piece_code):
        return piece_values.get(piece_code[1], 0) if piece_code else 0

    def find_capture_value(move):
        if 'x' in move:
            capture_square = move.split('x')[1][:2]  # Extract target square (e.g. 'e4')
            captured_piece = pieces.get(capture_square)
            return get_piece_value(captured_piece)
        return 0

    # 1. Checkmate
    for move in legal_moves:
        if '#' in move:
            return move, memory

    # 2. Capture High-Value Piece
    best_capture_move = None
    best_capture_value = -1
    for move in legal_moves:
        capture_value = find_capture_value(move)
        if capture_value > best_capture_value:
            best_capture_value = capture_value
            best_capture_move = move

    if best_capture_move:
        return best_capture_move, memory

    # 3. Check
    for move in legal_moves:
        if '+' in move:
            return move, memory

    # 4. Simple Move
    return legal_moves[0], memory
```
