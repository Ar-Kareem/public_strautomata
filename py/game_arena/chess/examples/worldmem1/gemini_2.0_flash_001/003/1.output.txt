1. **Prioritize Checkmates:** If any legal move results in checkmate, immediately return that move.
2. **Prioritize Captures:** Evaluate all captures in `legal_moves`. Use a simple material count (Queen = 9, Rook = 5, Bishop/Knight = 3, Pawn = 1) to determine the most valuable capture. Choose the capture that wins the most material. If there are multiple captures of equal value, apply other criteria to choose between them.
3. **Avoid Blunders:** If the opponent can capture a valuable piece for free, move the piece to safety or block the capture.
4. **King Safety:** Consider King safety. If the king is in check, prioritize moves that block the check, move the king to a safe square, or capture the attacking piece.  Avoid moves that expose the king to checks.
5. **Develop Pieces:** Prioritize moving knights and bishops off the back rank to control more squares.
6. **Control the Center:** Make moves that control the center of the board (d4, e4, d5, e5).
7. **Pawn Structure:** Try to create a good pawn structure. Avoid isolated pawns, doubled pawns, and backward pawns.
8. **Castling:** If the King has not castled, and it is safe to do so, castle to improve King safety and activate a rook.
9. **Queen Activity:** Bring the queen into play early in the game when it is safe.
10. **Random Move (Fallback):** If none of the above criteria apply strongly, select a legal move at random to ensure the game continues. Prefer moves that develop pieces over pawn moves.

import random

def material_value(piece):
    if piece == 'Q':
        return 9
    elif piece == 'R':
        return 5
    elif piece == 'B':
        return 3
    elif piece == 'N':
        return 3
    elif piece == 'P':
        return 1
    else:
        return 0

def evaluate_capture(move, pieces):
    if 'x' in move:
        captured_piece_square = move[move.index('x')+1:move.index('x')+3]
        if captured_piece_square in pieces:
            captured_piece = pieces[captured_piece_square][1] # Just the piece type
            return material_value(captured_piece)
    return 0

def policy(pieces: dict[str, str], to_play: str, legal_moves: list[str], memory: dict) -> tuple[str, dict]:

    # 1. Checkmate
    for move in legal_moves:
        if '#' in move:
            return move, memory

    # 2. Captures
    best_capture_move = None

        capture_value = evaluate_capture(move, pieces)
        if capture_value > best_capture_value:
            best_capture_value = capture_value
            best_capture_move = move

    if best_capture_move:
        return best_capture_move, memory

    for move in legal_moves:

    # 4. Development
