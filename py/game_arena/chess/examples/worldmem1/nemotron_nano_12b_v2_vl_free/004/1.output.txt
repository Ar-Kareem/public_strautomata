To solve this problem, we need to create a policy for a chess-playing AI that can make strategic moves based on the current game state. The AI should prioritize checkmating moves, capturing high-value opponent pieces, castling safely, and moving valuable pieces forward when no other options are available. The solution should efficiently parse and evaluate moves to determine the best possible action under given constraints.

### Approach
1. **Capture Moves**: Prioritize capturing the highest-value opponent pieces. If multiple captures are possible, select the one with the highest value.
2. **Castling**: If capturing moves are not available, check for castling moves. Prioritize kingside castling over queenside if possible.
3. **Valuable Pieces**: If other moves are available, prioritize moving the most valuable pieces (Queen, Rook, Bishop, Knight) towards advancing forward or controlling key squares.

### Solution Code
```python
def policy(pieces, to_play, legal_moves, memory):
    if not legal_moves:
        return ("", {})

    opponent = 'b' if to_play == 'white' else 'w'
    piece_values = {'Q': 9, 'R': 5, 'B': 3, 'N': 3, 'P': 1}

    # Step 1: Check for captures and select the highest-value capture
    captures = []
    for move in legal_moves:
        if 'x' in move:
            # Split move into target parts
            parts = move.split('x', 1)
            target_sq = parts[1][:2] if len(parts) > 1 else None
            if target_sq and target_sq in pieces:
                captured_piece = pieces[target_sq]
                if captured_piece[0] == opponent:
                    captured_type = captured_piece[1]
                    if captured_type in piece_values:
                        captures.append((move, piece_values[captured_type]))
    
    # Select the best capture if any
    if captures:
        # Sort by value descending, then by move string (lex order)
        captures.sort(key=lambda x: (-x[1], x[0]))
        best_capture = captures[0][0]
        return (best_capture, {})

    # Step 2: Prioritize castling if possible
    castling_moves = ['O-O', 'O-O-O']
    possible_castling = [m for m in castling_moves if m in legal_moves]
    if possible_castling:
        return (min(possible_castling), {})  # Prefer 'O-O' over 'O-O-O'

    # Step 3: Prioritize moves involving the highest-value pieces
    move_ranking = []
    for move in legal_moves:
        move_part = move.split('=', 1)[0]  # Ignore promotion part
        if move_part == 'O-O' or move_part == 'O-O-O':
            continue  # Already handled castling

        if move_part[0] in piece_values:
            piece_type = move_part[0]
            move_ranking.append((-piece_values[piece_type], move))
        elif move_part[0] == 'O':
            # Castling moves like 'O-O-O' are handled earlier
            continue
        else:
            # Pawn moves or other promotions
            move_ranking.append((-1, move))  # Pawns have lower priority

    # Select the best move based on priority
    if move_ranking:
        move_ranking.sort(key=lambda x: x[0])
        best_move = move_ranking[0][1]
        return (best_move, {})

    # Fallback to first legal move if none of the above
    return (legal_moves[0], {})
```

### Explanation
1. **Capture Moves**: The code first checks all legal moves for captures by parsing each move string. If a move contains an 'x', it indicates a capture. The target square is extracted, and the value of the captured piece is determined. The highest-value capture is selected.
2. **Castling**: If no captures are available, the code checks for king-side or queen-side castling moves. Castling is prioritized if possible.
3. **Valuable Pieces**: If no captures or castling is possible, the code selects moves involving the most valuable pieces (Queen, Rook, Bishop, Knight) by ranking them based on their piece values.
4. **Fallback**: If no other moves are prioritized, the code defaults to the first legal move.

This approach ensures that the AI prioritizes powerful moves like capturing high-value pieces and castling, providing a robust strategy to compete effectively in the game.
