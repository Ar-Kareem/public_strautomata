# Strategy Explanation
The AI prioritizes moves as follows:
1. **Check moves**: If any move results in a check, it is preferred. Among these, moves that capture the highest-value piece are selected first.
2. **High-value captures**: Moves that capture the opponent's pieces are prioritized, starting from capturing queens, followed by rooks, knights/bishops, and pawns.
3. **Promotions**: Moves that promote a pawn to a more powerful piece (especially a queen) are next in priority.
4. **Castling**: If castling is safe (not in check), it is chosen to improve king safety.
5. **Default**: Return the first legal move if no other conditions are met.

def get_piece_value(piece_str: str) -> int:
    if piece_str.startswith('K') or piece_str.startswith('k'):
        return 0  # Kings are not captured
    type_score = {
        'P': 1, 'N': 3, 'B': 3, 'R': 5, 'Q': 9
    }.get(piece_str[1], 0)
    return type_score
