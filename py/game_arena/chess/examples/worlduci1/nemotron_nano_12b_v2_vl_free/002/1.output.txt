To solve the problem of selecting the next best move for a chess engine based on a given board state represented by a `pieces` dictionary, we need to focus on two key components:

1. **Move Generation**: Generate all possible legal moves for the current player.
2. **Move Evaluation**: Prioritize moves based on a heuristic that balances material gain, king safety, and development.

The challenge lies in generating **legal moves** from scratch and selecting the best one under time constraints (1 second). This involves parsing the current board state, evaluating all possible moves, and ensuring that the selected move does not leave the player in check.

---

### Key Assumptions
- The `pieces` dictionary gives the full board state with valid piece positions.
- The current player (`to_play`) is either `'white'` or `'black'`.
- We assume standard chess rules and ignore advanced features like en passant and castling for simplicity, due to time and complexity constraints.
- We will simulate each move and check if it results in a check condition. If it does, the move is considered illegal and discarded.

---

### Strategy
1. **Parse the board** from the `pieces` dictionary.
2. **Generate all possible moves** for the current player.
3. **Simulate each move** and check if it leaves the king in check.
4. **Select the best move** using a heuristic that prioritizes:
   - **Material gain** (capturing high-value pieces).
   - **Piece development** (moving pieces toward the center).
   - **King safety** (moving the king to safer squares).

### Explanation

- **Board Representation**: The code parses the `pieces` dictionary into an 8x8 board.
- **Move Generation**:
  - Generates all possible moves for each piece (knights, pawns, kings).
  - Checks for valid moves (e.g., no moving into own piece).
- **Check Detection**:
  - Simulates each move and checks if the king is in check afterward.
  - Discards any move that results in a king being in check.
- **Move Selection**:
  - Evaluates each legal move based on:
    - **Material value** (priority to capturing high-value pieces like queens and rooks).
    - **Development** (moving pieces toward the center of the board).

This approach provides a smart and heuristic-based selection strategy that maximizes the chance of winning the game while ensuring the returned move is always legal.
