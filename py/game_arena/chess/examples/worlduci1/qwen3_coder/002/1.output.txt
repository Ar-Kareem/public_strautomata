The strategy for this policy is to implement a simple but effective chess engine that evaluates the board using a basic heuristic. The heuristic accounts for:

1. Material advantage (using standard piece values)
2. King safety (penalizing exposed kings)
3. Piece mobility (rewarding pieces with more legal moves)
4. Center control (rewarding pieces occupying or attacking center squares)
5. Pawn structure (rewarding pawn advancement and promoting potential)

The policy looks one move ahead (a simple minimax) to find the move that maximizes the evaluation score. For pawn promotions, it always promotes to a queen (the strongest piece).

                nx, ny = x + dx, y + dy
                if is_on_board(nx, ny):
                    attacked.add(coords_to_square(nx, ny))
        elif piece_type == 'N':

                    nx += dx
                    ny += dy
        elif piece_type == 'R':

                    nx += dx
                    ny += dy

    def is_in_check(king_color, board):

    def generate_pseudo_legal_moves(board, player_color):

    def evaluate(board, player_color):
        score = 0
        opponent_color = 'b' if player_color == 'w' else 'w'
        
        # Material
        for sq, piece in board.items():

                return move  # Checkmate

        if score > best_score:
            best_score = score
            best_move = move
            
    return best_move
