### **Strategy for this Policy**
My strategy is based on a **hybrid approach** combining **material evaluation**, **king safety**, and **tactical threats** with **depth-limited minimax search** (up to depth 3) and **quiescence search** to handle checkmate and capture sequences efficiently.

Key principles:
1. **Checkmate detection**: If a move leads to an immediate checkmate, prioritize it.
2. **Material advantage**: Prefer moves that improve material balance (e.g., capturing opponent pieces, promoting pawns).
3. **King safety**: Avoid moves that expose the king to threats (e.g., castling if safe, improving pawn structure).
4. **Threat prevention**: Block or capture attacking pieces if they threaten critical squares.
5. **Quiescence**: Evaluate only "quiet" positions (no captures or checks) to avoid horizon effect in minimax.
6. **Depth-limited search**: Use a 3-ply lookahead to assess future threats/opportunities.
7. **Pawn promotion**: Prioritize pawn promotions to queens (or other high-value pieces) if possible.

The policy will:
- Use **static evaluation** (material + king safety) for depth-limited minimax.
- Apply **quiescence search** to avoid missing checkmate/capture sequences.
- Return the **best legal move** based on the highest evaluation score.

from typing import Dict, List

# --- Piece Values ---
PIECE_VALUES = {'P': 1, 'N': 3, 'B': 3, 'R': 5, 'Q': 9, 'K': 0}  # King has no material value
PAWN_PROMOTION_BONUS = {'q': 2, 'r': 1, 'b': 1, 'n': 1}  # Bonus for promoting to queen/rook

    'b1': 5, 'b8': 5, 'b2': 0, 'b7': 0,

# --- Helper Functions ---
def get_piece_value(piece: str) -> int:
    """Returns the material value of a piece (0 for king)."""
    return PIECE_VALUES.get(piece[1], 0)

def is_pawn_promotion(move: str) -> bool:
    """Checks if a move is a pawn promotion."""
    return len(move) > 4 and move[3].islower()

def evaluate_pawn_promotion(move: str) -> int:
    """Evaluates the bonus for pawn promotions."""
    if is_pawn_promotion(move):
        return PAWN_PROMOTION_BONUS.get(move[3], 0)
    return 0

def get_king_position(pieces: Dict[str, str], color: str) -> str:
    """Returns the king's position for a given color."""
    king_code = f"{color}K"
    for square, piece in pieces.items():
        if piece == king_code:
            return square
    return None

def evaluate_king_safety(pieces: Dict[str, str], to_play: str) -> int:
    """Evaluates king safety based on position."""
    king_pos = get_king_position(pieces, to_play)
    if king_pos is None:
        return 0
    return KING_SAFETY_SCORE.get(king_pos, 0)

def evaluate_material(pieces: Dict[str, str], to_play: str) -> int:
    """Evaluates material balance (to_play's perspective)."""
    material = 0
    opponent = 'b' if to_play == 'w' else 'w'
    for piece in pieces.values():
        if piece[0] == to_play:
            material += get_piece_value(piece)
        elif piece[0] == opponent:
            material -= get_piece_value(piece)
    return material

def evaluate_position(pieces: Dict[str, str], to_play: str) -> int:
    """Combined evaluation of material and king safety."""
    return evaluate_material(pieces, to_play) + evaluate_king_safety(pieces, to_play)

def is_check(pieces: Dict[str, str], to_play: str) -> bool:
    """Checks if the current player is in check."""
    king_pos = get_king_position(pieces, to_play)
    if king_pos is None:
        return False
    opponent = 'b' if to_play == 'w' else 'w'
    for square, piece in pieces.items():
        if piece[0] == opponent:
            # Simulate capture to see if king is under attack
            if simulate_capture(pieces, square, king_pos):
                return True
    return False

def simulate_capture(pieces: Dict[str, str], attacker: str, target: str) -> bool:
    """Simulates a capture and checks if the king is attacked."""
    if attacker not in pieces or target not in pieces:
        return False
    attacker_piece = pieces[attacker]
    del pieces[attacker]
    del pieces[target]
    # Check if opponent's king is now under attack
    opponent = 'b' if attacker_piece[0] == 'w' else 'w'
    king_pos = get_king_position(pieces, opponent)
    if king_pos is None:
        return False
    # Check if any remaining piece attacks the king
    for square, piece in pieces.items():
        if piece[0] == attacker_piece[0]:
            continue
        # Simulate move (oversimplified for check detection)
        if is_attacking(piece, square, king_pos):
            return True
    return False

def is_attacking(piece: str, square: str, target: str) -> bool:
    """Checks if a piece at `square` attacks `target` (simplified)."""
    file_diff = abs(ord(square[0]) - ord(target[0]))
    rank_diff = abs(int(square[1]) - int(target[1]))
    if piece[1] == 'P':
        return (piece[0] == 'w' and square[1] == '2' and target == square[0] + str(int(square[1]) + 2)) or \
               (piece[0] == 'b' and square[1] == '7' and target == square[0] + str(int(square[1]) - 2))
    elif piece[1] == 'N':
        return (file_diff == 2 and rank_diff == 1) or (file_diff == 1 and rank_diff == 2)
    elif piece[1] == 'B':
        return file_diff == rank_diff
    elif piece[1] == 'R':
        return (file_diff == 0 or rank_diff == 0) and (file_diff != 0 or rank_diff != 0)
    elif piece[1] == 'Q':
        return (file_diff == 0 or rank_diff == 0) or file_diff == rank_diff
    return False

def apply_move(pieces: Dict[str, str], move: str) -> Dict[str, str]:
    """Applies a move to the pieces dictionary (simplified)."""
    new_pieces = pieces.copy()
    if len(move) == 4:
        from_sq, to_sq = move[:2], move[2:]
        piece = new_pieces[from_sq]
        del new_pieces[from_sq]
        new_pieces[to_sq] = piece
    else:  # Pawn promotion
        from_sq, to_sq = move[:2], move[2:3]
        piece = new_pieces[from_sq]
        new_piece = piece[0] + move[3].upper()  # e.g., 'wP' -> 'wQ'
        del new_pieces[from_sq]
        new_pieces[to_sq] = new_piece
    return new_pieces

def quiescence_search(pieces: Dict[str, str], to_play: str, alpha: float, beta: float, depth: int = 0) -> float:
    """Searches only quiet positions to avoid horizon effect."""
    if depth > 0 and evaluate_position(pieces, to_play) >= beta:
        return beta
    if depth > 0 and evaluate_position(pieces, to_play) > alpha:
        alpha = evaluate_position(pieces, to_play)

    for move in captures:

        score = -quiescence_search(new_pieces, 'b' if to_play == 'w' else 'w', -beta, -alpha, depth + 1)
        if score >= beta:
            return beta
        if score > alpha:
            alpha = score

def evaluate_capture(move: str, pieces: Dict[str, str]) -> int:
    """Evaluates the material gain of a capture (simplified)."""
    if len(move) == 4:
        from_sq, to_sq = move[:2], move[2:]
        if to_sq in pieces:
            return get_piece_value(pieces[to_sq])
    elif is_pawn_promotion(move):
        return get_piece_value(pieces[move[:2]]) + PIECE_VALUES['Q']
    return 0

def is_capture(move: str) -> bool:
    """Checks if a move is a capture (simplified)."""
    return len(move) == 4 and move[2:] in pieces

# --- Main Policy ---
def policy(pieces: Dict[str, str], to_play: str, legal_moves: List[str]) -> str:
    """Selects the best move from legal_moves using minimax + quiescence."""
    if not legal_moves:
        return ""

    # Check for immediate checkmate
    for move in legal_moves:
        new_pieces = apply_move(pieces, move)
        if is_check(new_pieces, 'b' if to_play == 'w' else 'w'):
            return move

    # Evaluate all legal moves using minimax + quiescence
    best_move = None
    best_score = -float('inf')
    for move in legal_moves:
        new_pieces = apply_move(pieces, move)
        score = evaluate_position(new_pieces, to_play)

            best_score = score
            best_move = move

    return best_move
