The strategy for this policy is to implement a simple chess engine that evaluates moves based on material advantage, piece activity, and king safety. It will prioritize capturing high-value pieces, developing pieces to active squares, and castling to protect the king. It will also attempt to identify and exploit simple tactical opportunities such as forks, pins, and skewers. The evaluation function will be a linear combination of these factors. Because of runtime constraints, only a shallow search (depth 1) will be conducted. The code will always return a legal move.

def policy(pieces: dict[str, str], to_play: str) -> str:
