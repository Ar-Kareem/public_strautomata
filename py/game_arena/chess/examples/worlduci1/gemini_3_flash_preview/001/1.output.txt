The strategy for this policy is a fast and efficient evaluation-based search. It performs the following steps:
1.  **Material and Positional Evaluation**: Uses a piece-square table to value material (P=100, N=320, B=330, R=500, Q=900, K=20000) and rewards pieces for being in the center of the board.
2.  **Move Generation**: It includes a lightweight move generator that handles basic movement, captures, and pawn mechanics for all pieces.
3.  **One-Move Lookahead (Greedy)**: It evaluates the resulting board state for every legal move. It prioritizes capturing high-value targets and moving toward better positional squares.
4.  **Minimax Simulation**: It simulates the opponent's best response to ensure the chosen move doesn't lead to an immediate heavy loss (like losing a Queen for a Pawn).
5.  **Safety First**: If no clear advantage is found, it selects a move that maintains the highest possible evaluation score.

def policy(pieces: dict[str, str], to_play: str) -> str:

        p = pieces.get(sq)
        if not p: return None, None

        score = 0
        for sq, p in board.items():
            color, type = p[0], p[1]
            val = VALS.get(type, 0)

            rank, file = int(sq[1]), ord(sq[0]) - ord('a')
            dist_center = abs(3.5 - file) + abs(4.5 - rank)
            val += (5 - dist_center) * 2
            
            if (color == 'w' and to_play == 'white') or (color == 'b' and to_play == 'black'):
                score += val
            else:
                score -= val
        return score

                dirs = [(1,1),(1,-1),(-1,1),(-1,-1)]
            elif pt == 'R':

                            occ = board.get(target)
                            if not occ:
                                moves.append(sq + target)
                            elif occ[0] == enemy_char:
                                moves.append(sq + target)
                                break
                            else: break
                        else: break
            elif pt == 'P':

                    if 0 <= nf < 8:

    my_color = 'w' if to_play == 'white' else 'b'
    legal_moves = get_legal_moves(pieces, my_color)
    
    if not legal_moves:

    best_move = legal_moves[0]

    random.shuffle(legal_moves)

    for move in legal_moves:
        # Simulate move

        promo = move[4:] if len(move) > 4 else None

        # Apply move
        del pieces[start]

        # Undo move

        if saved_end_piece: pieces[end] = saved_end_piece
