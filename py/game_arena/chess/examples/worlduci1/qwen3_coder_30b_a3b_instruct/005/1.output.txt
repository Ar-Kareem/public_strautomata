# Strategy

The policy will implement a move selection strategy based on the following principles:

1. **Checkmate Priority**: First check if there's a move that delivers checkmate - this is the highest priority.
2. **Capture Priority**: Look for moves that capture pieces of higher value, prioritizing capturing high-value pieces over low-value ones.
3. **King Safety**: Avoid moves that expose the king to check or weaken king safety.
4. **Piece Development**: Prefer moves that develop pieces (especially knights and bishops) toward the center.
5. **Material Advantage**: If no immediate threats exist, aim for moves that gain material or maintain material balance.
6. **Pawn Structure**: Avoid unnecessary pawn moves that weaken pawn structure.
7. **Random Selection**: If all else fails, make a random legal move.

Since we need to be consistent and deterministic within the constraints, we will build a move scoring system that evaluates each legal move based on the above criteria. The move with the highest score will be selected.

For tackling this, I will:
- Implement move evaluation based on piece values
- Use specific heuristics for capturing, development, king safety
- Prioritize immediate tactical gains
- Generate a score for each move based on these factors
- Return the move with the highest score

This approach will be efficient and work well in tournament settings.
