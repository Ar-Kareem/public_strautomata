The strategy for this policy is to prioritize moves that lead to checkmate, capture high-value pieces, or improve the position significantly. The policy will evaluate each legal move based on the following criteria in order of importance:

1. **Checkmate**: If any move results in checkmate, choose it immediately.
2. **Captures**: Prioritize moves that capture high-value pieces (Queen > Rook > Bishop > Knight > Pawn).
3. **Promotions**: If a pawn can promote, prioritize promoting to a Queen.
4. **Checks**: Moves that put the opponent in check are preferred.
5. **Castling**: Castling is a strong move for king safety.
6. **Development**: Moves that develop pieces (e.g., moving knights or bishops to central squares) are preferred.
7. **Randomness**: If no clear best move is found, choose a random legal move to avoid predictability.

The policy will parse the move strings to identify captures, promotions, and checks, and assign scores accordingly. The move with the highest score will be selected.

import random

def policy(pieces: dict[str, str], to_play: str, legal_moves: list[str]) -> str:
    # Define piece values for evaluation
    piece_values = {
        'Q': 9,
        'R': 5,
        'B': 3,
        'N': 3,
        'P': 1,
        'K': 0  # King is not captured, so value is 0 for this purpose
    }

    # Initialize best move and score
    best_move = None
    best_score = -float('inf')

    for move in legal_moves:
        score = 0

        # Check for checkmate (highest priority)
        if '#' in move:
            return move  # Immediate return if checkmate is found

        # Check for captures
        if 'x' in move:
            # Extract the captured piece (e.g., 'Bxf5' -> 'f5' is the square, need to find the piece)
            # The captured piece is at the destination square (after 'x')
            parts = move.split('x')
            if len(parts) > 1:
                dest_square = parts[1][0:2]  # First two characters after 'x' (e.g., 'f5' in 'Bxf5')
                if dest_square in pieces:
                    captured_piece = pieces[dest_square]
                    if captured_piece[0] != to_play[0]:  # Ensure it's an opponent's piece
                        piece_type = captured_piece[1]
                        score += piece_values.get(piece_type, 0)

        # Check for promotions
        if '=' in move:
            promotion_piece = move.split('=')[1]
            if promotion_piece == 'Q':
                score += 8  # High value for promoting to Queen

        # Check for checks
        if '+' in move:
            score += 1  # Small bonus for putting opponent in check

        # Check for castling
        if move in ['O-O', 'O-O-O']:
            score += 2  # Bonus for castling (king safety)

        # Check for development (moving knights or bishops to central squares)
        if move[0] in ['N', 'B'] and len(move) >= 3:
            dest_square = move[-2:]  # Last two characters are the destination square
            if dest_square in ['e4', 'd4', 'e5', 'd5', 'c4', 'c5', 'f4', 'f5']:
                score += 0.5  # Small bonus for central development

        # Update best move if current move has higher score
        if score > best_score:
            best_score = score
            best_move = move

    # If no move is selected (unlikely), choose a random legal move
    if best_move is None:
        best_move = random.choice(legal_moves)

    return best_move
