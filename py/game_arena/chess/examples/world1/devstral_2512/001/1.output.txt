The strategy for this policy is to implement a simple yet effective heuristic-based move selection that prioritizes checkmate, material gain, and king safety. The policy will:

1. **Check for checkmate**: If any move in `legal_moves` ends with `#` (checkmate), return the first such move immediately.
2. **Prioritize captures**: Among non-checkmate moves, prioritize captures (moves containing `x`) to gain material.
3. **Promotions**: If there are pawn promotions, prioritize promoting to Queen (`=Q`) for maximum material advantage.
4. **King safety**: Avoid moves that expose the king (e.g., castling is generally good for king safety).
5. **Default to first move**: If no clear priority is found, return the first legal move.

This approach ensures the policy is fast (within 1 second) and avoids illegal moves by strictly selecting from `legal_moves`.

def policy(pieces: dict[str, str], to_play: str, legal_moves: list[str]) -> str:
    # Priority 1: Checkmate
    for move in legal_moves:
        if move.endswith('#'):
            return move

    # Priority 2: Captures (material gain)
    captures = [move for move in legal_moves if 'x' in move]
    if captures:
        return captures[0]

    # Priority 3: Promotions (prefer Queen)
    promotions = [move for move in legal_moves if '=' in move]
    if promotions:
        for move in promotions:
            if move.endswith('=Q'):
                return move
        return promotions[0]

    # Priority 4: Castling (king safety)
    castling_moves = [move for move in legal_moves if move in ['O-O', 'O-O-O']]
    if castling_moves:
        return castling_moves[0]

    # Default: Return the first legal move
    return legal_moves[0]
