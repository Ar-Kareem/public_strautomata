To solve the problem of creating a smart policy for the Kalah game, we need to define a function that selects the best possible move from a given game state. The function must return a valid move (a single integer between 0 and 5) and must always return a legal move. The evaluation strategy should prioritize moves that lead to capturing the opponent's seeds, maximize the player's store and house values, and minimize the opponent's advantages.

### Approach
The approach involves simulating each possible move and evaluating the resulting board state to determine the best move. The key steps are:

1. **Check each possible move**: For each house with seeds, simulate the move and track the resulting state of the game.
2. **Simulate the move**: Distribute the seeds from the chosen house clockwise, updating the game state accordingly.
3. **Check for captures**: If the last seed lands in a player's own house and that house was empty before the move, capture all seeds from that house and the corresponding opponent's opposite house.
4. **Evaluate the move**: Use a heuristic to evaluate the resulting state, considering the player's store value, house values, and the opponent's store and house values.
5. **Select the best move**: Choose the move with the highest evaluated value, prioritizing captures to maximize seed capture and store growth.

### Solution Code
```python
def policy(you, opponent):
    max_eval = -float('inf')
    best_move = 0
    for move in range(6):
        if you[move] == 0:
            continue
        
        # Create copies of the current state
        new_you = you.copy()
        new_opp = opponent.copy()
        seeds = new_you[move]
        new_you[move] = 0
        pos = move + 1
        remaining = seeds
        last_house = None
        
        while remaining > 0:
            if pos == 6:  # Your store
                new_you[pos] += remaining
                remaining = 0
                last_house = 6
            elif pos < len(new_you):
                # Your own house
                new_you[pos] += 1
                remaining -= 1
                last_house = pos if remaining == 0 else last_house
            else:
                # Opponent's houses (virtual positions 7-12)
                sim_op_pos = pos - 7
                new_opp[sim_op_pos] += 1
                if remaining == 0:
                    last_house = None  # Not your house, so no capture possible
                remaining -= 1
            pos += 1
        
        # Check for captures
        if last_house is not None and last_house < 6:
            if you[last_house] == 0:  # Check if house was empty before this move
                opp_house = 5 - last_house
                captured = new_opp[opp_house]
                new_you[6] += 1 + captured
                new_you[last_house] = 0
                new_opp[opp_house] = 0
        
        # Evaluate the resulting state
        eval_val = new_you[6] - new_opp[6]
        evaluator2 = sum(new_you[:6]) - sum(new_opp[:6])
        current_eval = eval_val + evaluator2
        
        if current_eval > max_eval:
            max_eval = current_eval
            best_move = move
    
    return best_move
```

### Explanation
1. **Move Simulation**: For each possible initial move, the code simulates the distribution of seeds, updating the player's and opponent's houses accordingly. The seeds are distributed clockwise, starting from the selected house.
2. **Capture Check**: After distributing all seeds, the code checks if the last seed landed in an empty house of the player's own side. If so, it captures all seeds from that house and the corresponding opponent's house.
3. **Heuristic Evaluation**: The evaluation function scores the resulting state based on the difference between the player's store and the opponent's store, along with the sum of house values.
4. **Move Selection**: The move with the highest evaluation is selected as the best move, ensuring an optimal balance between capturing seeds and maintaining a favorable board state.

This approach balances immediate gains (captures and store value increases) with the potential for future moves, making it a robust and efficient strategy for the Kalah game.
