3. **Material Advantage**: We should generally favor moves that maximize our seed count in the store.
4. **Denying Opponent Moves**: Moves that leave the opponent with few options might be advantageous.
5. **Future Planning**: Consider sowing seeds in a way that sets up future captures or extra moves.

1. First check for any moves that result in an immediate extra turn (landing in our store).

3. If multiple options exist at any step, prefer the rightmost house (higher index) to generally allow more flexibility in future moves.
4. As a fallback, choose the rightmost non-empty house.

def policy(you: list[int], opponent: list[int]) -> int:
    # First check for moves that give an extra turn

            continue
        if (you[i] + i) % 13 == 6:  # Lands in store
            return i

    best_capture = -1
