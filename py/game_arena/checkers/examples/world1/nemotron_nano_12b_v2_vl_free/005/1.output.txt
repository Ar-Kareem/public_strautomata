### Strategy Overview
The policy prioritizes capturing moves to remove opponent pieces. It selects moves that capture kings first, followed by moves promoting a piece to a king if possible. If no captures are available, it moves towards the promotion row (toward the opponent's back) and prefers central positions for increased mobility.

### Solution Code

    # Convert lists to sets for O(1) lookups
    my_men_set = set(my_men)
    my_kings_set = set(my_kings)
    opp_men_set = set(opp_men)

        if is_king:

            if color == 'b':
                dirs = [(-1, -1), (-1, 1)]

                dirs = [(1, -1), (1, 1)]

        for dr, dc in dirs:

            if not (0 <= inter_r < 8 and 0 <= inter_c < 8):
                continue
            # Check if there's an opponent's piece to capture
            if (inter_r, inter_c) in opp_men_set or (inter_r, inter_c) in opp_kings_set:

                land_c = c + 2 * dc
                # Check landing square validity
                if 0 <= land_r < 8 and 0 <= land_c < 8:

                    if (land_r, land_c) not in my_men_set and (land_r, land_c) not in my_kings_set:
                        capture_moves.append(((r, c), (land_r, land_c), dr, dc))

            dirs = [(-1, -1), (-1, 1), (1, -1), (1, 1)]
        else:
            if color == 'b':
                dirs = [(-1, -1), (-1, 1)]
            else:
                dirs = [(1, -1), (1, 1)]

### Explanation
1. **Capture Priority:**
   - Detects all possible captures by evaluating jumps over opponent's pieces.
   - Scores captures based on taking a king (higher value) and promotes pieces if they move to the promotion row.
   - Preferentially selects moves that also position the piece centrally.

2. **Non-Capture Moves:**
   - Moves pieces toward central positions to maximize mobility.
   - Prioritizes promotional moves for regular pieces to convert them into kings.
   - Selects the move with the best central positioning if no promotions are possible.

This strategy maximizes offensive potential through captures while optimizing the positioning of pieces to control the board effectively.
