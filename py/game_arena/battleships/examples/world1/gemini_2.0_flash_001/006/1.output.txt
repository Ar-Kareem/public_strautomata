The strategy for this policy is as follows:

1. **Hunting Phase:** Initially, target cells in a checkerboard pattern to efficiently explore the board. This maximizes the chance of finding ships early.  Specifically, alternate targeting (0,0), (0,2), (0,4), etc. Since we can't persist state, we'll assume that we just pick the next available square, prioritizing this pattern.

2. **Targeting Phase:** Once a hit is registered (board[row][col] == 1), switch to targeting mode.  Explore the adjacent cells (up, down, left, right) of the hit cell. If a direction yields another hit, continue in that direction until a miss or the board edge is reached. This increases the probability of sinking the ship quickly.

3. **Prioritization:** If multiple hit cells are known, prioritize exploring the hits that have fewer adjacent cells already explored.  This heuristic favors completing the sinking of partially exposed ships.

4. **Edge Cases:** When at the board edge, only consider legal moves within the grid.

5. **No Persistence:** Since we cannot persist state, the policy must be stateless and react appropriately at each turn. To emulate the targeting phase, we will prioritize cells adjacent to known hits. Since the checkerboard pattern might conflict, we will prioritize valid adjacent moves to all available hits and then implement a checkerboard search.

```python
import numpy as np

def policy(board: list[list[int]]) -> tuple[int, int]:
    """
    A Battleship policy that combines hunting and targeting strategies.
    """
    board_np = np.array(board)
    # 1. Prioritize targeting adjacent to known hits
    hit_cells = np.argwhere(board_np == 1)
    if len(hit_cells) > 0:
        for row, col in hit_cells:
            adjacent_moves = [(row - 1, col), (row + 1, col), (row, col - 1), (row, col + 1)]
            for r, c in adjacent_moves:
                if 0 <= r < 10 and 0 <= c < 10 and board[r][c] == 0:
                    return (r, c)

    # 2. Hunting Phase (Checkerboard pattern)
    for r in range(10):
        for c in range(10):
            if (r + c) % 2 == 0 and board[r][c] == 0:
                return (r, c)

    # 3. Last Resort: Find Any Remaining Unknown Cell
    for r in range(10):
        for c in range(10):
            if board[r][c] == 0:
                return (r, c)

    # If no valid move is found (which should never happen if the board is valid), return (0, 0) as a default.
    return (0, 0)
```
