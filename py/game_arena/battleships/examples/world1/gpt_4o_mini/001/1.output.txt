To develop a targeting policy for a Battleship-like game, we need a strategy to select cells to fire at on the opponent's grid. The board is represented as a 10x10 grid with three potential values in each cell: `0` for unknown, `-1` for a miss, and `1` for a hit.

The core attribute of a good targeting strategy is to prioritize areas where there is a high probability of hitting a ship, which involves using a mix of systematic scanning and targeting hotspots where ships are most likely to be found. 

Here's the approach for the targeting policy:
1. **Prioritize unshot cells:** Only target cells that are marked as `0`.
2. **Systematic search:** Fire in a checkerboard pattern to maximize coverage of the board initially, as this allows us to hit multiple ship placements effectively.
3. **Hotspot Strategy:** Once we have identified hits, we should then try to target adjacent cells to those hits, as ships occupy consecutive spaces.
4. **Random Targeting as fallback:** If there are no hits, select from the list of remaining unshot cells.

The targeting will focus on ensuring that we have a good spread across the board while ensuring that we utilize the high-density areas once we find a hit.

Here’s how we can implement this strategy in code:

1. Keep track of all the unshot cells.
2. Check for hits on the board, and if there are any, try to find their adjacent cells as targets.
3. If no hits are found and there are cells to shoot at, we will pick one randomly or based on our checkerboard approach.

import random

def policy(board: list[list[int]]) -> tuple[int, int]:
    width, height = 10, 10
    potential_targets = []
    
    # Collect all unshot cells
    for r in range(height):
        for c in range(width):
            if board[r][c] == 0:  # Only consider unshot cells
                potential_targets.append((r, c))
    
    if not potential_targets:
        raise RuntimeError("No more cells to target!")

    # Look for hits to target around
    targets_with_hits = []
    for r in range(height):
        for c in range(width):
            if board[r][c] == 1:  # Found a hit
                # Check the neighbors in a 3x3 area around the hit
                for dr in range(-1, 2):
                    for dc in range(-1, 2):
                        new_r, new_c = r + dr, c + dc
                        if 0 <= new_r < height and 0 <= new_c < width and board[new_r][new_c] == 0:
                            targets_with_hits.append((new_r, new_c))
    
    # We prefer targets around hits if any
    if targets_with_hits:
        return random.choice(targets_with_hits)
    
    # Fallback to random unshot target
    return random.choice(potential_targets)
