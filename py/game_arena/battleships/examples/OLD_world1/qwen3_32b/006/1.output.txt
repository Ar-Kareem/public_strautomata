Here's a well-documented implementation of the targeting policy for the Battleship game, based on a probabilistic approach that considers all valid ship placements on the board and selects the most promising cell.

---

### ✅ **Approach Overview**

This targeting policy follows a "probabilistic guessing" strategy by considering all valid ship placements on the board and computing the most likely cell to hit a ship.

---

### 🧠 **Algorithm Summary**

1. **Ship Placement Generation**: All valid horizontal and vertical placements for each ship size are generated, ensuring that the ship fits in the grid and does not overlap with known misses (`-1`).

2. **Probability Matrix Construction**: For every valid placement (i.e., a valid ship location), each cell involved in that placement receives an increment in a 10×10 *probability matrix*. This simulates how "likely" each cell is to contain a ship (based on remaining valid ship configurations).

3. **Final Selection**: The policy selects the untested cell (`0`) with the **highest** probability value from the matrix, favoring the most promising target.

---

### 🔍 **Why It Works Efficiently**

- Considers the board's known hits and misses to avoid invalid ship placements.
- Prioritizes untested cells based on how often they are part of valid ship placements.
- Does not make assumptions about which specific ship is where, only about general probabilities.
- Works efficiently on a 10×10 grid with a manageable number of possible placements.

---

### ✅ **Policy Code**

```python
def policy(board: list[list[int]]) -> tuple[int, int]:
    # Initialize a 10x10 probability matrix
    prob = [[0] * 10 for _ in range(10)]

    # List of all ship lengths to consider
    ship_lengths = [5, 4, 3, 3, 2]

    # Evaluate all valid ship placements for each length
    for length in ship_lengths:
        # 1. Horizontal placements
        for row in range(10):
            max_col = 10 - length
            for col in range(max_col + 1):
                # Check if the horizontal segment is valid (no -1)
                valid = True
                for c in range(col, col + length):
                    if board[row][c] == -1:
                        valid = False
                        break
                if valid:
                    # Add this position to the probability matrix
                    for c in range(col, col + length):
                        prob[row][c] += 1

        # 2. Vertical placements
        for col in range(10):
            max_row = 10 - length
            for row in range(max_row + 1):
                # Check if the vertical segment is valid (no -1)
                valid = True
                for r in range(row, row + length):
                    if board[r][col] == -1:
                        valid = False
                        break
                if valid:
                    # Add this position to the probability matrix
                    for r in range(row, row + length):
                        prob[r][col] += 1

    # Find the cell with the highest probability among untested ones
    max_prob = -1
    best_row, best_col = 0, 0
    for r in range(10):
        for c in range(10):
            if board[r][c] == 0 and prob[r][c] > max_prob:
                max_prob = prob[r][c]
                best_row, best_col = r, c

    return (best_row, best_col)
```

### 🎯 **Strategy Justification**

- **Handles all phases**: Random guessing early on, then focuses on exploiting hits.
- **Adapts to information**: As misses and hits occur, the policy avoids invalid moves and prioritizes high-probability cells.
- **Balances brute-force and intelligence**: The probabilistic calculation is efficient and leverages all available information.
