1. **Immediate winning moves**: First check if I can win immediately with a placement + rotation combination
2. **Blocking opponent's wins**: Check if opponent can win on their next turn and block those threats
3. **Positional advantages**: 
   - Control the center quadrants for better connectivity
   - Create multiple threats simultaneously
   - Use rotations to both advance my position and disrupt opponent's formations
4. **Evaluation function**: Score moves based on potential threats created, opponent threats blocked, and positional value
5. **Search depth**: Since we have 1 second, I'll use a shallow search (1-ply) with good heuristics rather than deep minimax

1. **Immediate win detection**: For each empty cell, try each rotation to see if it creates 5+ in a row
2. **Threat blocking**: Identify if opponent has 4-in-a-row opportunities and block them

   - +1000: Immediate win
   - +500: Blocking opponent's immediate win
   - +100: Creating multiple 3+ threats
   - +50: Creating a 3+ threat
   - +10: Center quadrant placement bonus
   - -1000: Giving opponent immediate win

        else:  # clockwise
            rotated = np.rot90(quad_view, 1)

        return result
