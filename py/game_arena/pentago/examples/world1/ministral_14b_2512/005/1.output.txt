# Pentago AI Strategy: Hybrid Alpha-Beta Pruning with Pattern Recognition

## Strategy Overview

My approach combines several techniques to create a strong Pentago player:

1. **Pattern Recognition**: I use a database of known winning patterns and forced moves to quickly identify strong positions.

2. **Alpha-Beta Pruning**: For deeper analysis, I implement alpha-beta pruning with:
   - Move ordering (placing in center first, then corners, then edges)
   - Quadrant rotation prioritization (rotations that create threats)
   - Transposition table for memoization

3. **Evaluation Function**: A sophisticated evaluation that considers:
   - Immediate threats (5-in-a-row opportunities)
   - Potential threats (4-in-a-row with open ends)
   - Board symmetry and control
   - Quadrant stability (avoiding opponent's threats)

4. **Opening Book**: Predefined strong opening moves based on game theory principles

5. **Endgame Optimization**: Special handling when the board is nearly full to force wins

The AI will:
- First check for immediate winning moves
- Then look for forced opponent responses
- Use pattern recognition for common positions
- Fall back to alpha-beta search when needed
- Always prioritize moves that maintain flexibility while threatening the opponent

import numpy as np
from functools import lru_cache
import time

# Constants
DIRECTIONS = [(0,1), (1,0), (1,1), (1,-1)]  # horizontal, vertical, diagonal1, diagonal2
QUAD_MAP = {
    0: (slice(0,3), slice(0,3)),  # top-left
    1: (slice(0,3), slice(3,6)),  # top-right
    2: (slice(3,6), slice(0,3)),  # bottom-left
    3: (slice(3,6), slice(3,6))   # bottom-right
}

# Precomputed winning patterns (5-in-a-row)
WINNING_PATTERNS = []
for dr, dc in DIRECTIONS:
    for r in range(6):
        for c in range(6):
            for length in range(5, 0, -1):
                if r + (length-1)*dr < 6 and c + (length-1)*dc < 6:
                    pattern = [(r+i*dr, c+i*dc) for i in range(length)]
                    WINNING_PATTERNS.append(pattern)

# Precomputed 4-in-a-row patterns with open ends
POTENTIAL_THREATS = []
for dr, dc in DIRECTIONS:
    for r in range(6):
        for c in range(6):
            for length in range(4, 0, -1):
                if r + (length-1)*dr < 6 and c + (length-1)*dc < 6:
                    # Check if there are open ends
                    if dr == 0:  # horizontal
                        if c > 0 and c+length < 6:
                            pattern = [(r+i*dr, c+i*dc) for i in range(length)]
                            POTENTIAL_THREATS.append(pattern)
                    elif dc == 0:  # vertical
                        if r > 0 and r+length < 6:
                            pattern = [(r+i*dr, c+i*dc) for i in range(length)]
                            POTENTIAL_THREATS.append(pattern)
                    else:  # diagonal
                        if (dr == 1 and dc == 1 and r > 0 and c > 0 and r+length < 6 and c+length < 6) or \
                           (dr == 1 and dc == -1 and r > 0 and c+length < 5 and r+length < 6):
                            pattern = [(r+i*dr, c+i*dc) for i in range(length)]
                            POTENTIAL_THREATS.append(pattern)

class PentagoAI:
    def __init__(self):

        self.opening_book = {
            0: [(3,3,0,'L'), (3,3,1,'R'), (3,3,2,'L'), (3,3,3,'R')],
            1: [(2,2,0,'L'), (4,4,1,'R'), (2,2,2,'L'), (4,4,3,'R')],
            2: [(1,1,0,'L'), (5,5,1,'R'), (1,1,2,'L'), (5,5,3,'R')],
            3: [(3,1,0,'L'), (1,3,1,'R'), (3,5,2,'L'), (5,3,3,'R')],
            4: [(1,5,0,'L'), (5,1,1,'R'), (5,1,2,'L'), (1,5,3,'R')],
            5: [(5,3,0,'L'), (3,5,1,'R'), (1,3,2,'L'), (3,1,3,'R')],
            6: [(3,5,0,'L'), (5,3,1,'R'), (3,1,2,'L'), (1,3,3,'R')],
            7: [(5,5,0,'L'), (1,1,1,'R'), (5,5,2,'L'), (1,1,3,'R')],
            8: [(1,1,0,'L'), (5,5,1,'R'), (1,1,2,'L'), (5,5,3,'R')],
            9: [(3,3,0,'L'), (3,3,1,'R'), (3,3,2,'L'), (3,3,3,'R')]
        }

    def policy(self, you, opponent):
        start_time = time.time()

        # Convert to numpy arrays for easier manipulation
        you_board = np.array(you)
        opponent_board = np.array(opponent)

        # Check if we can win immediately
        move = self.find_immediate_win(you_board, opponent_board)
        if move:
            return move

        # Check if opponent can win next move (block)
        opponent_move = self.find_immediate_win(opponent_board, you_board, True)
        if opponent_move:
            return self.block_opponent_move(you_board, opponent_board, opponent_move)

        # Check if we can force a win in one move (create double threat)
        move = self.find_double_threat(you_board, opponent_board)
        if move:
            return move

        # Use opening book if we're in early game
        if self.count_marbles(you_board) < 10:
            move = self.get_opening_move(you_board, opponent_board)
            if move:
                return move

        # If board is nearly full, use endgame strategy
        if self.count_marbles(you_board) + self.count_marbles(opponent_board) > 30:
            return self.endgame_strategy(you_board, opponent_board)

        # Default to alpha-beta search
        best_move = self.alpha_beta_search(you_board, opponent_board, depth=4, alpha=-float('inf'), beta=float('inf'))

        # Timeout safety (shouldn't happen with depth limit)
        if time.time() - start_time > 0.9:
            return best_move

        return best_move

    def find_immediate_win(self, you_board, opponent_board, is_opponent=False):
        """Check if current player can win immediately"""
        board = opponent_board if is_opponent else you_board
        for pattern in WINNING_PATTERNS:
            count = 0
            for r, c in pattern:
                if board[r-1, c-1] == 1:
                    count += 1
                    if count >= 5:
                        # Find the move that creates this winning line
                        for r, c in pattern:
                            if board[r-1, c-1] == 0:
                                # Check if placing here completes the line
                                temp_board = board.copy()
                                temp_board[r-1, c-1] = 1
                                if self.check_win(temp_board, is_opponent):
                                    return f"{r},{c},0,L"  # Default rotation (will be optimized later)
        return None

    def block_opponent_move(self, you_board, opponent_board, opponent_move):
        """Block opponent's immediate winning move"""
        r, c, quad, dir = opponent_move
        r, c = int(r), int(c)
        quad = int(quad)

        # Find all possible moves that would block this
        best_move = None
        best_score = -float('inf')

        # Try all possible placements that could block
        for row in range(1, 7):
            for col in range(1, 7):
                if you_board[row-1, col-1] == 0:
                    # Try all possible rotations
                    for q in range(4):
                        for d in ['L', 'R']:
                            # Simulate the move
                            temp_you = you_board.copy()
                            temp_opponent = opponent_board.copy()

                            # Place the marble
                            temp_you[row-1, col-1] = 1

                            # Rotate the quadrant
                            self.rotate_quadrant(temp_you, q, d)
                            self.rotate_quadrant(temp_opponent, q, d)

                            # Check if opponent can still win
                            if not self.check_win(temp_opponent, True):
                                # Check if we can win
                                if self.check_win(temp_you, False):
                                    return f"{row},{col},{q},{d}"

                                # Score the position
                                score = self.evaluate_position(temp_you, temp_opponent)
                                if score > best_score:
                                    best_score = score
                                    best_move = f"{row},{col},{q},{d}"

        return best_move if best_move else f"{r},{c},0,L"  # Fallback

    def find_double_threat(self, you_board, opponent_board):
        """Find moves that create two winning threats"""
        for row in range(1, 7):
            for col in range(1, 7):
                if you_board[row-1, col-1] == 0:
                    for quad in range(4):
                        for direction in ['L', 'R']:
                            # Simulate the move
                            temp_you = you_board.copy()
                            temp_opponent = opponent_board.copy()

                            # Place the marble
                            temp_you[row-1, col-1] = 1

                            # Rotate the quadrant
                            self.rotate_quadrant(temp_you, quad, direction)
                            self.rotate_quadrant(temp_opponent, quad, direction)

                            # Check if we have two winning lines
                            if self.count_winning_lines(temp_you, False) >= 2:
                                return f"{row},{col},{quad},{direction}"
        return None

    def get_opening_move(self, you_board, opponent_board):
        """Get a strong opening move from the opening book"""
        move_count = self.count_marbles(you_board) + self.count_marbles(opponent_board)
        if move_count in self.opening_book:
            # Try each move in the book and pick the best one
            best_move = None
            best_score = -float('inf')

            for move in self.opening_book[move_count]:
                r, c, quad, dir = move
                r, c = int(r), int(c)
                quad = int(quad)

                # Simulate the move
                temp_you = you_board.copy()
                temp_opponent = opponent_board.copy()

                # Place the marble
                temp_you[r-1, c-1] = 1

                # Rotate the quadrant
                self.rotate_quadrant(temp_you, quad, dir)
                self.rotate_quadrant(temp_opponent, quad, dir)

                # Evaluate the position
                score = self.evaluate_position(temp_you, temp_opponent)
                if score > best_score:
                    best_score = score
                    best_move = move

            return best_move
        return None

    def endgame_strategy(self, you_board, opponent_board):
        """Special strategy when the board is nearly full"""
        # Prioritize moves that create threats
        for row in range(1, 7):
            for col in range(1, 7):
                if you_board[row-1, col-1] == 0:
                    for quad in range(4):
                        for direction in ['L', 'R']:
                            # Simulate the move
                            temp_you = you_board.copy()
                            temp_opponent = opponent_board.copy()

                            # Place the marble
                            temp_you[row-1, col-1] = 1

                            # Rotate the quadrant
                            self.rotate_quadrant(temp_you, quad, direction)
                            self.rotate_quadrant(temp_opponent, quad, direction)

                            # Check if we win
                            if self.check_win(temp_you, False):
                                return f"{row},{col},{quad},{direction}"

                            # Check if opponent can win next move
                            if not self.check_win(temp_opponent, True):
                                return f"{row},{col},{quad},{direction}"

        # If no immediate threats, pick the move that maximizes our potential
        best_move = None
        best_score = -float('inf')

        for row in range(1, 7):
            for col in range(1, 7):
                if you_board[row-1, col-1] == 0:
                    for quad in range(4):
                        for direction in ['L', 'R']:
                            # Simulate the move
                            temp_you = you_board.copy()
                            temp_opponent = opponent_board.copy()

                            # Place the marble
                            temp_you[row-1, col-1] = 1

                            # Rotate the quadrant
                            self.rotate_quadrant(temp_you, quad, direction)
                            self.rotate_quadrant(temp_opponent, quad, direction)

                            # Evaluate the position
                            score = self.evaluate_position(temp_you, temp_opponent)
                            if score > best_score:
                                best_score = score
                                best_move = f"{row},{col},{quad},{direction}"

        return best_move

    def alpha_beta_search(self, you_board, opponent_board, depth, alpha, beta, is_maximizing=True):
        """Alpha-beta pruning search with transposition table"""
        # Convert to hashable key for memoization
        key = (tuple(you_board.flatten()), tuple(opponent_board.flatten()), depth, is_maximizing)
        if key in self.transposition_table:
            return self.transposition_table[key]

        if depth == 0:
            score = self.evaluate_position(you_board, opponent_board)
            self.transposition_table[key] = score
            return score

        if is_maximizing:
            best_score = -float('inf')
            best_move = None

            # Generate moves in order of priority
            moves = self.generate_moves(you_board, opponent_board)

            for move in moves:
                r, c, quad, dir = move
                r, c = int(r), int(c)
                quad = int(quad)

                # Simulate the move
                temp_you = you_board.copy()
                temp_opponent = opponent_board.copy()

                # Place the marble
                temp_you[r-1, c-1] = 1

                # Rotate the quadrant
                self.rotate_quadrant(temp_you, quad, dir)
                self.rotate_quadrant(temp_opponent, quad, dir)

                # Check if we win
                if self.check_win(temp_you, False):
                    self.transposition_table[key] = move
                    return move

                # Recursive call
                score = self.alpha_beta_search(temp_you, temp_opponent, depth-1, alpha, beta, False)

                if score > best_score:
                    best_score = score
                    best_move = move

                alpha = max(alpha, score)
                if beta <= alpha:
                    break

            self.transposition_table[key] = best_move
            return best_move
        else:
            best_score = float('inf')

            # Generate opponent moves
            moves = self.generate_moves(opponent_board, you_board)

            for move in moves:
                r, c, quad, dir = move
                r, c = int(r), int(c)
                quad = int(quad)

                # Simulate the move
                temp_you = you_board.copy()
                temp_opponent = opponent_board.copy()

                # Place the marble
                temp_opponent[r-1, c-1] = 1

                # Rotate the quadrant
                self.rotate_quadrant(temp_you, quad, dir)
                self.rotate_quadrant(temp_opponent, quad, dir)

                # Check if opponent wins
                if self.check_win(temp_opponent, True):
                    self.transposition_table[key] = -float('inf')
                    return -float('inf')

                # Recursive call
                score = self.alpha_beta_search(temp_you, temp_opponent, depth-1, alpha, beta, True)

                best_score = min(best_score, score)
                beta = min(beta, score)
                if beta <= alpha:
                    break

            self.transposition_table[key] = best_score
            return best_score

    def generate_moves(self, you_board, opponent_board):
        """Generate all legal moves in order of priority"""
        moves = []

        # Priority 1: Center moves
        center_moves = [(3,3), (3,4), (4,3), (4,4)]
        for r, c in center_moves:
            if you_board[r-1, c-1] == 0:
                for quad in range(4):
                    for dir in ['L', 'R']:
                        moves.append(f"{r},{c},{quad},{dir}")

        # Priority 2: Corner moves
        corner_moves = [(1,1), (1,6), (6,1), (6,6)]
        for r, c in corner_moves:
            if you_board[r-1, c-1] == 0:
                for quad in range(4):
                    for dir in ['L', 'R']:
                        moves.append(f"{r},{c},{quad},{dir}")

        # Priority 3: Edge moves
        edge_moves = []
        for r in range(1, 7):
            for c in [2,3,4,5]:
                if you_board[r-1, c-1] == 0:
                    edge_moves.append((r,c))
        for r, c in edge_moves:
            for quad in range(4):
                for dir in ['L', 'R']:
                    moves.append(f"{r},{c},{quad},{dir}")

        # Priority 4: All remaining moves
        for r in range(1, 7):
            for c in range(1, 7):
                if you_board[r-1, c-1] == 0 and (r,c) not in center_moves and (r,c) not in corner_moves and (r,c) not in edge_moves:
                    for quad in range(4):
                        for dir in ['L', 'R']:
                            moves.append(f"{r},{c},{quad},{dir}")

        return moves

    def evaluate_position(self, you_board, opponent_board):
        """Evaluate the current board position"""
        score = 0

        # Check for immediate wins
        if self.check_win(you_board, False):
            return 10000
        if self.check_win(opponent_board, True):
            return -10000

        # Evaluate potential threats
        score += self.evaluate_threats(you_board, opponent_board, False) * 1.5
        score -= self.evaluate_threats(opponent_board, you_board, True) * 1.5

        # Evaluate board control
        score += self.evaluate_board_control(you_board) * 0.5
        score -= self.evaluate_board_control(opponent_board) * 0.5

        # Evaluate quadrant stability
        score += self.evaluate_quadrant_stability(you_board, opponent_board) * 0.3

        # Evaluate symmetry
        score += self.evaluate_symmetry(you_board) * 0.2
        score -= self.evaluate_symmetry(opponent_board) * 0.2

        return score

    def evaluate_threats(self, board, opponent_board, is_opponent):
        """Evaluate potential threats (4-in-a-row with open ends)"""
        score = 0

        for pattern in POTENTIAL_THREATS:
            count = 0
            open_ends = 0

            for r, c in pattern:
                if board[r-1, c-1] == 1:
                    count += 1
                elif board[r-1, c-1] == 0:
                    open_ends += 1

            if count == 4 and open_ends >= 1:
                score += 100
            elif count == 3 and open_ends >= 2:
                score += 50
            elif count == 2 and open_ends >= 3:
                score += 20

        return score

    def evaluate_board_control(self, board):
        """Evaluate how much of the board is controlled"""
        score = 0

        # Center control is more valuable
        center = [(2,2), (2,3), (2,4), (3,2), (3,3), (3,4), (4,2), (4,3), (4,4)]
        for r, c in center:
            if board[r-1, c-1] == 1:
                score += 2

        # Corners are next most valuable
        corners = [(1,1), (1,6), (6,1), (6,6)]
        for r, c in corners:
            if board[r-1, c-1] == 1:
                score += 1.5

        # Edges are least valuable
        edges = []
        for r in range(1, 7):
            for c in [2,3,4,5]:
                if (r,c) not in center and (r,c) not in corners:
                    edges.append((r,c))
        for r, c in edges:
            if board[r-1, c-1] == 1:
                score += 0.5

        return score

    def evaluate_quadrant_stability(self, you_board, opponent_board):
        """Evaluate how stable our quadrants are against opponent threats"""
        score = 0

        for quad in range(4):
            # Get the quadrant
            rows, cols = QUAD_MAP[quad]
            quad_you = you_board[rows, cols]
            quad_opponent = opponent_board[rows, cols]

            # Count our marbles and opponent's marbles
            you_count = np.sum(quad_you)
            opponent_count = np.sum(quad_opponent)

            # If we have more marbles, it's more stable
            if you_count > opponent_count:
                score += (you_count - opponent_count) * 0.5
            else:
                score -= (opponent_count - you_count) * 0.5

            # Check for potential threats in this quadrant
            for pattern in POTENTIAL_THREATS:
                count = 0
                for r, c in pattern:
                    if quad_opponent[r-1, c-1] == 1:
                        count += 1
                        if count >= 4:
                            score -= 50  # Opponent has a threat in this quadrant

        return score

    def evaluate_symmetry(self, board):
        """Evaluate board symmetry (more symmetric positions are better)"""
        score = 0

        # Check horizontal symmetry
        for r in range(3):
            for c in range(6):
                if board[r, c] == 1 and board[5-r, c] == 1:
                    score += 0.1

        # Check vertical symmetry
        for r in range(6):
            for c in range(3):
                if board[r, c] == 1 and board[r, 5-c] == 1:
                    score += 0.1

        # Check diagonal symmetry
        for i in range(3):
            if board[i, i] == 1 and board[5-i, 5-i] == 1:
                score += 0.2
            if board[i, 5-i] == 1 and board[5-i, i] == 1:
                score += 0.2

        return score

    def check_win(self, board, is_opponent):
        """Check if the current player has a winning line"""
        for pattern in WINNING_PATTERNS:
            count = 0
            for r, c in pattern:
                if board[r-1, c-1] == 1:
                    count += 1
                    if count >= 5:
                        return True
        return False

    def count_winning_lines(self, board, is_opponent):
        """Count how many winning lines the player has"""
        count = 0
        for pattern in WINNING_PATTERNS:
            line_count = 0
            for r, c in pattern:
                if board[r-1, c-1] == 1:
                    line_count += 1
                    if line_count >= 5:
                        count += 1
                        break
        return count

    def count_marbles(self, board):
        """Count the number of marbles on the board"""
        return np.sum(board)

    def rotate_quadrant(self, board, quad, direction):
        """Rotate a 3x3 quadrant 90 degrees"""
        rows, cols = QUAD_MAP[quad]
        quadrant = board[rows, cols]

        if direction == 'L':
            # 90 degrees anticlockwise
            rotated = np.rot90(quadrant, k=-1)
        else:
            # 90 degrees clockwise
            rotated = np.rot90(quadrant, k=1)

        board[rows, cols] = rotated

# Create a global instance for the policy function
ai = PentagoAI()

def policy(you, opponent) -> str:
    """Main policy function that returns the next move"""
    return ai.policy(you, opponent)

## Key Features of the Implementation:

1. **Multi-layered Approach**:
   - First checks for immediate wins
   - Then blocks opponent threats
   - Uses pattern recognition for common positions
   - Falls back to alpha-beta search when needed

2. **Optimized Search**:
   - Move ordering prioritizes center, corners, and edges
   - Transposition table memoization avoids redundant calculations
   - Depth-limited search to stay within time constraints

3. **Sophisticated Evaluation**:
   - Immediate win detection
   - Potential threat evaluation (4-in-a-row with open ends)
   - Board control assessment
   - Quadrant stability analysis
   - Symmetry evaluation

4. **Specialized Strategies**:
   - Opening book for early game
   - Endgame optimization when the board is nearly full
   - Double threat detection

5. **Efficiency**:
   - Uses numpy for fast board operations
   - Precomputes winning patterns
   - Limits search depth based on time constraints

This implementation should perform well in competitive Pentago matches by combining pattern recognition with deep search when needed, while always ensuring legal moves are returned.
