# Strategy for Pentago Policy

My strategy for this Pentago policy is to implement a smart AI that combines:
1. **Immediate threat assessment** - prioritize blocking opponent 4-in-a-rows
2. **Winning move detection** - look for 5-in-a-row opportunities
3. **Positional evaluation** - favor center control and open lines
4. **Quadrant rotation evaluation** - choose rotations that help maximize my positions or block opponent moves
5. **Minimax with alpha-beta pruning** - for deeper strategic thinking for moves that don't immediately win/lose

The approach:
1. First check for immediate winning moves
2. Then check for immediate block moves
3. If none, use a minimax approach with a heuristic evaluation function that looks at:
   - Number of 3-in-a-rows / 4-in-a-rows available for each player
   - Positional value for center control
   - Potential for future 5-in-a-rows
4. For rotations, evaluate which rotation offers the most strategic value to my position
5. Use a depth-limited search to manage computational time

For simplicity and speed in the 1s time constraint, I'll primarily focus on:
- Detecting and playing immediate winning moves
- Playing to block opponent winning moves
- Using a simple evaluation to prefer center positions

The implementation uses a combination of heuristic evaluation and simple game Tree search with alpha-beta pruning for a limited depth.
