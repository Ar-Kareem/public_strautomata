## Strategy

My policy will use a combination of:

1. **Minimax with Alpha-Beta Pruning**: Search several moves ahead to find optimal plays
2. **Advanced Evaluation Function** considering:
   - **Progress towards goal**: Reward pieces closer to the opponent's home row
   - **Material count**: Value having more pieces
   - **Forward positioning**: Encourage aggressive advancement
   - **Center control**: Value pieces in central columns
   - **Piece safety**: Penalize pieces that can be captured
   - **Winning moves**: Heavily prioritize moves that reach the goal row
   - **Capture opportunities**: Value capturing opponent pieces

3. **Move Ordering**: Prioritize promising moves (captures, advances, winning moves) to improve alpha-beta pruning efficiency

4. **Depth-based Search**: Search as deep as time permits (typically 3-4 plies)

            new_row = row + direction
            
            if not (0 <= new_row <= 7):
                continue

            if (new_row, col) not in all_pieces:
                moves.append(((row, col), (new_row, col)))

            for dc in [-1, 1]:

                if 0 <= new_col <= 7:
                    target = (new_row, new_col)
                    # Can move diagonally if empty or opponent piece
                    if target not in all_pieces or target in opp_set:
                        moves.append(((row, col), target))

        score += (len(my_pieces) - len(opp_pieces)) * 100
