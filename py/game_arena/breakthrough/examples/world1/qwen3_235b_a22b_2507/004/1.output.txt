The strategy for this policy is to implement a greedy heuristic-based minimax decision rule with a depth of 1, augmented with prioritization of winning moves and immediate threats. The policy evaluates all legal moves and selects the one that maximizes progress toward victory, using a combination of the following heuristics:

1. **Winning Move**: If a move leads to reaching the opponent’s home row, it is immediately selected.
2. **Capture Priority**: Capturing an opponent piece, especially ones deeper in the opponent’s territory, is highly valued.
3. **Forward Progress**: Move pieces forward (toward the opponent’s home row) to increase board control and promote advancement.
4. **Avoid Losing**: Prefer moves that do not expose pieces unnecessarily, although full threat analysis is simplified due to depth-1 lookahead.
5. **Center Bias**: Slightly prefer advancing or capturing toward central columns (columns 3 and 4), as they offer greater mobility and strategic control.

The algorithm iterates through all of the player's pieces, generates all legal moves (forward, left-diagonal, right-diagonal), checks their legality, applies the heuristics, and returns the best-scoring move.

This approach ensures fast execution (well under 1 second) and a strong, aggressive playstyle that performs well in head-to-head matches.

from typing import List, Tuple

def policy(me: List[Tuple[int, int]], opp: List[Tuple[int, int]], color: str) -> Tuple[Tuple[int, int], Tuple[int, int]]:
    # Define movement direction based on color
    if color == 'w':
        direction = 1  # White moves up (increasing row)
        opponent_home_row = 7
    else:  # color == 'b'
        direction = -1  # Black moves down (decreasing row)
        opponent_home_row = 0

    # Convert opp list to set for fast lookup
    opp_set = set(opp)
    me_set = set(me)

    best_score = -float('inf')
    best_move = None

    # Precompute center preference to slightly favor central columns
    center_cols = {3, 4}

    for piece in me:
        r, c = piece

        # Try forward move (straight)
        new_r = r + direction
        if 0 <= new_r <= 7:
            # Straight forward
            if (new_r, c) not in me_set and (new_r, c) not in opp_set:
                if new_r == opponent_home_row:
                    return (r, c), (new_r, c)  # Immediate win
                score = 10  # Base move value
                score += new_r if color == 'w' else (7 - new_r)  # Promote progress
                if c in center_cols:
                    score += 1  # Center bonus
                if score > best_score:
                    best_score = score
                    best_move = (r, c), (new_r, c)

            for dc in [-1, 1]:
                nc = c + dc
                if not (0 <= nc <= 7):
                    continue
                if (new_r, nc) in opp_set:  # Can capture
                    if new_r == opponent_home_row:
                        return (r, c), (new_r, nc)  # Immediate win via capture
                    score = 20  # Capture bonus
                    # Bonus for capturing pieces that are deeper in our direction
                    capture_depth = new_r if color == 'w' else (7 - new_r)
                    score += capture_depth
                    if c in center_cols:
                        score += 1
                    if score > best_score:
                        best_score = score
                        best_move = (r, c), (new_r, nc)

    # If no move found above, something is wrong, but fallback to random legal move
    if best_move is None:
        # Should not happen, but for safety
        for piece in me:
