### Strategy for This Policy

The strategy for this policy is a **hybrid approach** combining **evaluation-based search** (similar to minimax with alpha-beta pruning) and **heuristic-driven move selection**. The key ideas are:

1. **Evaluation Function**:
   - **Piece Value**: Each piece is worth 1 point, but pieces closer to the opponent's home row are worth more (e.g., a piece in row 6 for white is worth 2 points).
   - **Mobility**: Prefer moves that increase the number of legal moves for your pieces.
   - **Control of Center**: Prefer moves that control the center (columns 3-4) or threaten the opponent's center.
   - **Captures**: Prioritize captures, especially those that reduce the opponent's mobility or threaten their pieces.
   - **King Potential**: If a piece can reach the opponent's home row in one move, prioritize it.

2. **Search Depth**:
   - Use a **fixed-depth search (depth 3-4)** for most positions, but increase depth if the position is critical (e.g., a piece is close to promotion).
   - Use **quiescence search** to avoid horizon effects near captures or promotions.

3. **Move Ordering**:
   - Order moves to favor captures, promotions, and center control early in the search to improve alpha-beta pruning efficiency.

4. **Opening and Endgame**:
   - **Opening**: Push pieces forward quickly to control the center and create threats.
   - **Endgame**: Focus on promoting pieces or capturing the last opponent piece.

5. **Legal Move Enforcement**:
   - Always return a legal move, even if the search fails (fallback to a simple heuristic like moving the first piece forward).

def generate_legal_moves(my_pieces, opp_pieces, direction):

def minimax(opp_pieces, my_pieces, color, alpha, beta, depth, direction, opponent_home_row):

def evaluate_position(my_pieces, opp_pieces, direction, opponent_home_row):
    score = 0

    # Piece value: closer to opponent's home row is better
    for (row, col) in my_pieces:
        distance = abs(row - opponent_home_row)
        score += 1 + (7 - distance) / 7  # Bonus for pieces closer to promotion

    for (row, col) in opp_pieces:
        distance = abs(row - opponent_home_row)
        score -= 1 + (7 - distance) / 7

    # Mobility: more legal moves is better
    my_mobility = len(generate_legal_moves(my_pieces, opp_pieces, direction))
    opp_mobility = len(generate_legal_moves(opp_pieces, my_pieces, -direction))
    score += 0.1 * (my_mobility - opp_mobility)

    # Center control: prefer pieces in columns 3-4
    for (row, col) in my_pieces:
        center_bonus = 1 - abs(col - 3.5) / 3.5
        score += 0.2 * center_bonus

    for (row, col) in opp_pieces:
        center_bonus = 1 - abs(col - 3.5) / 3.5
        score -= 0.2 * center_bonus

    return score

def is_terminal(my_pieces, opp_pieces, opponent_home_row):
    # Check if any of my pieces reached opponent's home row
    for (row, col) in my_pieces:
        if row == opponent_home_row:
            return True

    # Check if opponent has no pieces left
    if not opp_pieces:
        return True

    return False
