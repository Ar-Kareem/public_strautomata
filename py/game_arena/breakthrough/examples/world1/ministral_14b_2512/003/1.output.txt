### Strategy for This Policy

The strategy for this policy is a **hybrid approach** combining **evaluation-based search** with **heuristic-driven move selection**. The key ideas are:

1. **Depth-Limited Minimax with Alpha-Beta Pruning**:
   - Use a search depth of 3-4 plies (adjustable based on time constraints) to explore possible moves.
   - Alpha-beta pruning significantly reduces the search space, making deeper searches feasible.

2. **Position Evaluation**:
   - **Piece Value**: Promoted pieces (reaching the opponent's home row) are worth more (e.g., +1000).
   - **Mobility**: Prefer moves that increase the number of legal moves for your pieces.
   - **Control of Center**: Favor moves that control the central squares (rows 3-4, columns 3-4).
   - **Threat to Opponent**: Prioritize moves that threaten to capture opponent pieces or promote your own.
   - **Defense**: Avoid moves that leave your pieces vulnerable to capture.

3. **Move Ordering**:
   - Sort moves by:
     - Captures (highest priority).
     - Promotions (next highest).
     - Mobility gains (moves that open up new options for other pieces).
     - Central control.

4. **Quiescence Search**:
   - Stop searching if the position is "quiet" (no immediate captures or promotions) to avoid horizon effects.

5. **Time Management**:
   - If the search exceeds time limits, fall back to a simpler heuristic (e.g., prioritize captures and promotions).

6. **Edge Cases**:
   - If no captures or promotions are possible, default to a move that maximizes mobility or central control.

def minimax(my_pieces, opp_pieces, direction, opponent_home_row, alpha, beta, depth, is_maximizing):

def evaluate_position(my_pieces, opp_pieces, direction, opponent_home_row):
    """Evaluate the position using heuristics."""
    score = 0

    # Promoted pieces
    for (row, col) in my_pieces:
        if row == opponent_home_row:

    score += len(generate_legal_moves(my_pieces, opp_pieces, direction))

    for (row, col) in my_pieces:

            diag_col = col + dc

def is_game_over(my_pieces, opp_pieces, direction, opponent_home_row):
    """Check if the game is over (win or loss)."""
    # Check for promotions
    for (row, col) in my_pieces:
        if row == opponent_home_row:
            return True
    for (row, col) in opp_pieces:
        if row == opponent_home_row:
            return True

    # Check if opponent has no pieces left
    if not opp_pieces:
