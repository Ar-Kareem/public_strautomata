The strategy for this policy is to implement a minimax algorithm with alpha-beta pruning to evaluate board positions several moves deep, prioritizing moves that:
1. Advance pieces toward the opponent's home row (win condition)
2. Capture opponent pieces when possible
3. Maintain piece cohesion and avoid isolated pieces
4. Block opponent advancement
5. Prioritize moves that threaten immediate win

The evaluation function assigns bonuses for:
- Distance to opponent home row (higher reward for pieces closer)
- Number of opponent pieces captured or threatened
- Center control (central columns are more valuable)
- Pawn structure (pieces in a diagonal line are stronger)

The AI looks 3-4 moves deep depending on board state (shallow for early game, deeper when captures are possible). It prioritizes capturing moves and winning moves above all else. If no good moves are found, it defaults to safest forward move.

To handle edge cases, we always check for immediate wins before evaluating, and ensure we never return an illegal move.

def policy(me: List[Tuple[int, int]], opp: List[Tuple[int, int]], color: str) -> Tuple[Tuple[int, int], Tuple[int, int]]:

    BOARD_SIZE = 8

    # Black ('b') moves downward (row decreases)
    # White ('w') moves upward (row increases)
    direction = -1 if color == 'b' else 1

    opponent_home_row = 7 if color == 'b' else 0  # Opponent's home row

    for piece in me:

        new_r = r + direction

            return ((r, c), (new_r, c))
        if (new_r, c) not in me and (new_r, c) not in opp:
            # This is a legal move, but not necessarily winning
            pass  # We'll consider it in evaluation

        for dc in [-1, 1]:
            new_r = r + direction
            new_c = c + dc
            if 0 <= new_r < BOARD_SIZE and 0 <= new_c < BOARD_SIZE:
                if (new_r, new_c) in opp:
                    if new_r == home_row:
                        return ((r, c), (new_r, new_c))  # Immediate winning capture

        r, c = piece

            new_r = r + direction
            new_c = c + dc

        return ((me[0][0], me[0][1]), (me[0][0] + direction, me[0][1]))

        # Slight bonus for not being too far forward (avoids isolation)

        if depth == 0:

        # Generate moves for current player
        if maximizing_player:

                    target = (new_r, new_c)
                    if target not in current_player:

                    min_eval = eval_score
                    best_move = move
                beta = min(beta, eval_score)
                if beta <= alpha:
                    break

    def evaluate_board(me_positions, opp_positions, player_color, target_row):

        center_bonus = 0
        for r, c in me_positions:
            if 2 <= c <= 5:  # Center columns

        score += center_bonus * 2
        
        # Advancement bonus
        for r, c in me_positions:
            # Closer to target row = higher score
            dist = abs(r - target_row)
            score += (8 - dist)  # 8 - dist: 8 if at target, 1 if at home
            
            # Avoid being at home row (already won)
            if r == target_row:
                score += 50  # Bonus for winning position
        
        # Capture potential
        for r, c in me_positions:
            for dc in [-1, 1]:
                new_r = r + direction
                new_c = c + dc
                if 0 <= new_r < BOARD_SIZE and 0 <= new_c < BOARD_SIZE:
                    if (new_r, new_c) in opp_positions:
                        score += 25  # High value for capturing opponent
        
        # Opponent advancement penalty
        for r, c in opp_positions:
            dist_opponent = abs(r - opponent_home_row)
            score -= (8 - dist_opponent) * 1.5  # Opponent advancing is bad for us
        
        # Piece cohesion (bonus for having adjacent pieces)
        for i, (r1, c1) in enumerate(me_positions):
            for j, (r2, c2) in enumerate(me_positions):
                if i < j:
                    # Check if pieces are adjacent
                    if abs(r1 - r2) <= 1 and abs(c1 - c2) <= 1:
                        score += 2
            
        # Avoid isolated pieces (penalize pieces with no neighbors)
        for r, c in me_positions:
            isolated = True
            for dr in [-1, 0, 1]:
                for dc in [-1, 0, 1]:

                        continue

                    if 0 <= nr < BOARD_SIZE and 0 <= nc < BOARD_SIZE:

                            isolated = False
                            break
                if not isolated:
                    break

                score -= 10  # Large penalty for isolation

        for r, c in me_positions:
            for dc in [-1, 1]:
                new_r = r + direction
                new_c = c + dc
                if 0 <= new_r < BOARD_SIZE and 0 <= new_c < BOARD_SIZE:

                    if (new_r, new_c) in opp_positions:

                            back_c = opp_c + opp_dc

                                    break

        def heuristic_score(move):
