# Strategy

I'll design a Breakthrough policy that combines:
1. **Win condition checking** - prioritize moves that win immediately
2. **Capture prioritization** - favor captures over moves
3. **Piece advancement** - push pieces toward opponent's home row
4. **Positional awareness** - avoid moving pieces that are easily trapped
5. **Defensive moves** - block opponent's advancement

For the evaluation:
- Score captures highly (they're most likely to win)
- Score advancing pieces higher than staying put
- Consider if a piece is at risk of being trapped
- Prioritize moves that get pieces closer to the opponent's home row

I'll implement a minimax-like approach with a simple evaluation function that considers:
- Immediate win conditions
- Captures
- Advancement toward home row
- Block opponent advancement opportunities

import random
from typing import List, Tuple

def policy(me: List[Tuple[int, int]], opp: List[Tuple[int, int]], color: str) -> Tuple[Tuple[int, int], Tuple[int, int]]:
    """
    Policy for Breakthrough game.
    Returns a legal move as ((from_row, from_col), (to_row, to_col))
    """
    
    # Convert to sets for easier lookup
    me_set = set(me)
    opp_set = set(opp)
    
    # Helper function to check if a position is valid
    def is_valid(row, col):
        return 0 <= row < 8 and 0 <= col < 8
    
    # Helper function to get all legal moves for a piece
    def get_legal_moves(piece):
        row, col = piece
        moves = []
        
        # Determine direction based on color
        if color == 'b':  # black moves down
            forward_row = row - 1
            # Forward move (straight)
            if is_valid(forward_row, col) and (forward_row, col) not in me_set and (forward_row, col) not in opp_set:
                moves.append(((row, col), (forward_row, col)))
            # Diagonal captures
            for dc in [-1, 1]:
                new_row, new_col = forward_row, col + dc
                if is_valid(new_row, new_col) and (new_row, new_col) in opp_set:
                    moves.append(((row, col), (new_row, new_col)))
        else:  # white moves up
            forward_row = row + 1
            # Forward move (straight)
            if is_valid(forward_row, col) and (forward_row, col) not in me_set and (forward_row, col) not in opp_set:
                moves.append(((row, col), (forward_row, col)))
            # Diagonal captures
            for dc in [-1, 1]:
                new_row, new_col = forward_row, col + dc
                if is_valid(new_row, new_col) and (new_row, new_col) in opp_set:
                    moves.append(((row, col), (new_row, new_col)))
        
        return moves
    
    # Check for immediate win conditions
    # Win by reaching opponent's home row
    if color == 'b':
        # Black wins if any piece reaches row 0
        for piece in me:
            if piece[0] == 0:
                # This piece has already won, but let's make a safe move
                # Just move any piece that can advance legally
                pass
    else:
        # White wins if any piece reaches row 7
        for piece in me:
            if piece[0] == 7:
                # This piece has already won, but let's make a safe move
                # Just move any piece that can advance legally
                pass
    
    # Check for win by capturing all opponent pieces
    if len(opp) == 0:
        # Just make any legal move
        all_moves = []
        for piece in me:
            all_moves.extend(get_legal_moves(piece))
        if all_moves:
            return all_moves[0]
    
    # Try to find a winning move
    all_moves = []
    capturing_moves = []
    
    # Get all legal moves for all pieces
    for piece in me:
        moves = get_legal_moves(piece)
        all_moves.extend(moves)
        # Separate capturing moves
        for move in moves:
            if move[1] in opp_set:  # This is a capture move
                capturing_moves.append(move)
    
    # If there are captures, prioritize them
    if capturing_moves:
        # Try to capture pieces that bring us closer to winning
        # Prioritize capturing pieces that are on the edge or can advance further
        # Also consider if capturing gets us closer to opponent's home row
        # But first just pick one random capturing move
        return random.choice(capturing_moves)
    
    # If no captures, try to avoid giving opponent clear options
    # Find moves that don't expose our pieces too much
    if all_moves:
        # Prefer moves that move a piece closer to opponent's home row
        # For black, closer to row 0; for white, closer to row 7
        sorted_moves = []
        for move in all_moves:
            from_row, from_col = move[0]
            to_row, to_col = move[1]
            
            # Score based on advancement towards opponent's home row
            if color == 'b':
                advancement = from_row - to_row  # higher is better
            else:  # white
                advancement = to_row - from_row  # higher is better
                
            # Prefer moves that advance
            # Also prefer moves that stay within the middle rows to avoid being trapped
            middle_row_preference = 0 if 2 <= from_row <= 5 else -1
            
            score = advancement + middle_row_preference
            
            sorted_moves.append((score, move))
        
        # Sort by score in descending order
        sorted_moves.sort(key=lambda x: x[0], reverse=True)
        
        return sorted_moves[0][1]
    else:
        # No valid moves - shouldn't really happen but just return a random move if needed
        return random.choice(list(all_moves)) if all_moves else ((0, 0), (1, 0))
