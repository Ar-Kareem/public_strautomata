To provide an AI strategy for the Havannah game, we focuson a simplified but effective heuristic that prioritizes creating or blocking critical structures like **rings**. Additionally, the code selects moves that are strategically advantageous based on adjacency to already-placed stones.

---

### 🔍 Strategy Overview

The AI follows a three-phase strategy:

1. **Ring Formation**: Prioritize moves that immediately create a ring. A ring is formed when a stone is placed such that it connects to two already-placed stones of the same player, and those two stones are adjacent. This creates a closed triangle, which is a ring.

2. **Blocking Opponent's Rings**: If no immediate ring can be formed, the AI checks whether any move would allow the opponent to form a ring on their next turn. If such a threat is found, the AI blocks it.

3. **Greedy Heuristic**: If no immediate threats or opportunities exist, the AI selects the valid cell with the most adjacent stones of its own color to promote future structure formation.

---

### 🧠 Code Strategy

The AI checks for the three winning structures in order of importance:

1. **Ring Formation**
2. **Bridge Formation**
3. **Fork Formation**

If none of these are possible, it uses a greedy heuristic to maximize the number of adjacent player stones.

### ✅ Ring Formation Check

A ring is formed when a cell has **two adjacent stones of the same player** that are themselves adjacent.

To detect this, for each valid cell, we compute its six neighbors (based on the hexagonal grid) and check if any two are owned by the current player and are adjacent.

### 🛡️ Blocking Opponent's Winning Moves

We simulate the AI's move and check if the opponent can form a winning structure in the next turn. If so, the AI blocks that move.

---

### 🧮 Greedy Heuristic

If no immediate threats or opportunities are found, the AI selects the move that has the most adjacent stones from the current player, promoting future structure formation.

### 🧩 Implementation

```python
def is_adjacent(a, b):
    dr = abs(a[0] - b[0])
    dc = abs(a[1] - b[1])
    return (dr <= 1 and dc <= 1) and (dr != 0 or dc != 0)

def policy(me, opp, valid_mask):
    valid_cells = [(r, c) for r in range(15) for c in range(15) if valid_mask[r][c]]

    # Step 1: Check for immediate ring-forming moves
    for cell in valid_cells:
        r, c = cell
        neighbors = [
            (r - 1, c), (r + 1, c), (r, c - 1), (r, c + 1),
            (r - 1, c + 1), (r + 1, c - 1)
        ]

        my_neighbors = [n for n in neighbors if n in me]
        if len(my_neighbors) >= 2:
            for i in range(len(my_neighbors)):
                a = my_neighbors[i]
                for j in range(i + 1, len(my_neighbors)):
                    b = my_neighbors[j]
                    if is_adjacent(a, b):
                        return (r, c)

    # Step 2: Block opponent's possible ring moves
    for cell in valid_cells:
        r, c = cell
        if (r, c) in me:
            continue
        handled_me = me + [(r, c)]
        me_set = set(handled_me)
        opponent_can_win = False

        # Simulate placing the AI's stone and check if opponent can win
        for opp_cell in valid_cells:
            if opp_cell in me_set or opp_cell == cell:
                continue
            opp_neighbors = [
                (opp_cell[0] - 1, opp_cell[1]), (opp_cell[0] + 1, opp_cell[1]),
                (opp_cell[0], opp_cell[1] - 1), (opp_cell[0], opp_cell[1] + 1),
                (opp_cell[0] - 1, opp_cell[1] + 1), (opp_cell[0] + 1, opp_cell[1] - 1)
            ]
            opp_my_neighbors = [n for n in opp_neighbors if n in opp]
            for k in range(len(opp_my_neighbors)):
                for l in range(k + 1, len(opp_my_neighbors)):
                    a = opp_my_neighbors[k]
                    b = opp_my_neighbors[l]
                    if is_adjacent(a, b):
                        opponent_can_win = True
                        break
                if opponent_can_win:
                    break
            if opponent_can_win:
                break
        if opponent_can_win:
            return (r, c)

    # Step 3: Greedy heuristic: choose cell with the most adjacent me cells
    best_cell = None
    best_score = -1

    for r in range(15):
        for c in range(15):
            if not valid_mask[r][c]:
                continue

            num_me_neighbors = 0
            for dr, dc in [
                (-1, 0), (1, 0), (0, -1), (0, 1),
                (-1, 1), (1, -1)
            ]:
                nr, nc = r + dr, c + dc
                if (nr, nc) in me:
                    num_me_neighbors += 1

            if num_me_neighbors > best_score:
                best_score = num_me_neighbors
                best_cell = (r, c)

    if best_cell:
        return best_cell

    # Fallback: return first valid move
    return valid_cells[0] if valid_cells else None
```

This AI provides a **solid foundation** for Havannah by:

- Actively **blocking the opponent** from forming winning structures,
- Using a **greedy heuristic** to maximize adjacency and connectivity.
