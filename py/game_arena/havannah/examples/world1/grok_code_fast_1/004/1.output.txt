The strategy for this policy is to prioritize moves that are closest to the center of the 15x15 Havannah board, calculated as Euclidean distance to the center point ((7,7), since 15//2=7). This approach aims to occupy strategic central positions first, potentially allowing control over board dynamics, while ensuring legal moves are always selected. It may not prevent all opponent threats but provides a simple heuristic that could outperform random or edge-biased policies in some games, as central play helps in connection games like Havannah.

import numpy as np

def policy(me: list[tuple[int, int]], opp: list[tuple[int, int]], valid_mask) -> tuple[int, int]:
    """
    Policy function for Havannah.
    Always returns a legal move by selecting the unoccupied cell in valid_mask
    that is closest to the center (7,7) on a 15x15 board.
    """
    board_size = len(valid_mask)
    center_row, center_col = board_size // 2, board_size // 2  # (7, 7)
    
    occupied = set(me).union(set(opp))  # Quick set for occupied cells
    
    legal_moves = []
    for r in range(board_size):
        for c in range(board_size):
            if valid_mask[r][c] and (r, c) not in occupied:
                legal_moves.append((r, c))
    
    if not legal_moves:
        # Should never happen if valid_mask and board state are consistent
        raise ValueError("No legal moves available")
    
    # Sort by Euclidean distance to center and pick the closest
    legal_moves.sort(key=lambda pos: (pos[0] - center_row)**2 + (pos[1] - center_col)**2)
    return legal_moves[0]
