### Strategy for Havannah Policy

My strategy combines several key elements to create a strong Havannah policy:

1. **Immediate Threat Response**: First check if the opponent has any winning moves (rings, bridges, or forks) and block them immediately.

2. **Winning Move Detection**: If we can make a winning move (ring, bridge, or fork), we do so immediately.

3. **Center Control**: Prioritize controlling the center of the board, as it provides more options for creating winning structures.

4. **Edge and Corner Control**: Secure key edges and corners to limit opponent's options and create potential forks.

5. **Mobility**: Prefer moves that maximize our own mobility (number of valid moves) while minimizing the opponent's.

6. **Pattern Recognition**: Look for potential patterns that could lead to immediate wins or threats in the near future.

7. **Fallback to Best Move**: If no immediate threats or winning moves are found, choose the move that appears most promising based on positional evaluation.

The implementation uses a combination of pattern matching, graph theory (for connectivity), and positional evaluation to make decisions. The code is optimized to run within the time constraints by using efficient data structures and early termination when possible.

    opp_np = np.array(opp)
    valid = np.array(valid_mask, dtype=bool)

    # Get all possible moves
    possible_moves = np.argwhere(valid)

    # Check for immediate opponent threats and our winning moves
    opp_threats = find_threats(opp_np, me_np, valid)

    if opp_threats:

    if our_wins:
        return our_wins[0]

    best_score = -float('inf')

def path_exists(player, start, end):
    """Check if there's a path between two points using player's stones"""
    if start == end:
        return True

    # Build adjacency graph
    graph = {}
    for stone in player:
        graph[tuple(stone)] = get_neighbors(stone)

    # BFS to check connectivity
    visited = set()
    queue = deque([start])

        current = queue.popleft()
        if current == end:
            return True
        if current in visited:
            continue
        visited.add(current)

        for neighbor in graph.get(current, []):
            if neighbor not in visited:
                queue.append(neighbor)

    return False
