The strategy for this policy is based on evaluating potential connection paths on the Hex board, with a focus on analyzing both my own and the opponent's moves. This policy will employ the following techniques:

1. **Board Representation**: I'll represent the 11x11 board, tracking both my and the opponent's moves.
2. **Move Scoring**: Each potential move is evaluated based on its connection potential. The move score will depend on how many connections (to existing stones) each potential move can create.
3. **Winning Threats**: I will identify potential winning moves (for both my color and the opponent's color) to block or advance the connections toward my goal.
4. **Minimax Strategy**: A simple strategic decision-making approach where I evaluate possible future moves and their outcomes.

The `policy` function will iterate through possible moves, score them, and choose the one that maximizes my chance of connecting while minimizing the opponent's opportunities.

def policy(me: list[tuple[int, int]], opp: list[tuple[int, int]], color: str) -> tuple[int, int]:
    size = 11
    board = [['.' for _ in range(size)] for _ in range(size)]
    
    for r, c in me:
        board[r][c] = 'b' if color == 'b' else 'w'
    
    for r, c in opp:
        board[r][c] = 'w' if color == 'b' else 'b'

    def is_valid_move(r, c):
        return 0 <= r < size and 0 <= c < size and board[r][c] == '.'

    def get_neighbors(r, c):
        neighbors = []
        for dr in [-1, 0, 1]:
            for dc in [-1, 0, 1]:
                if abs(dr) + abs(dc) == 1:  # Ensures we only take the right neighbors in hex
                    nr, nc = r + dr, c + dc
                    if is_valid_move(nr, nc):
                        neighbors.append((nr, nc))
        return neighbors

    def score_move(r, c):
        score = 0
        neighbors = get_neighbors(r, c)
        for nr, nc in neighbors:
            if board[nr][nc] == ('b' if color == 'b' else 'w'):
                score += 1
            elif board[nr][nc] == ('w' if color == 'b' else 'b'):
                score -= 1
        return score

    best_move = None
    best_score = -float('inf')

    for r in range(size):
        for c in range(size):
            if is_valid_move(r, c):
                move_score = score_move(r, c)
                if move_score > best_score:
                    best_score = move_score
                    best_move = (r, c)

    return best_move if best_move else (0, 0)
