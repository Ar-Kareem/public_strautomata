### Strategy for This Policy

This policy is designed to be a **strong Hex player** that combines **pattern recognition, connectivity analysis, and strategic depth** to outperform other policies in the arena. The key components of the strategy are:

1. **Immediate Threat Detection**:
   - First, check if the opponent is about to win on their next move. If so, block their winning path immediately.
   - Similarly, check if you are about to win on your next move. If so, prioritize completing your connection.

2. **Connectivity-Based Moves**:
   - Use **BFS (Breadth-First Search)** to evaluate the strength of your and the opponent's potential connections.
   - Prefer moves that extend your existing chains or create multiple potential winning paths.
   - Avoid moves that leave your opponent with a forced win.

3. **Center Control**:
   - The center of the board is critical in Hex. Prioritize controlling the center early in the game.
   - Use a **center bias** to place stones near the center if no immediate threats exist.

4. **Pattern Recognition**:
   - Recognize common Hex patterns (e.g., "forks," "bridges," or "double threats") and respond accordingly.
   - Use **heuristics** to evaluate the board state, such as:
     - The number of potential winning paths for you and the opponent.
     - The length of your longest chain.
     - The symmetry of the board.

5. **Randomness for Uncertainty**:
   - If no clear move is found, fall back to a **random legal move** to avoid predictable behavior.

6. **Efficiency**:
   - The policy is optimized to run within the 1-second constraint by using **early termination** in searches and **precomputed neighbor relationships**.

def policy(me: list[tuple[int, int]], opp: list[tuple[int, int]], color: str) -> tuple[int, int]:

    my_stones = set(me)
    opp_stones = set(opp)

    # Determine the goal sides for the current player
    if color == 'b':
        goal_rows = {0, 10}  # Black connects top (0) and bottom (10)
    else:
        goal_cols = {0, 10}  # White connects left (0) and right (10)

    empty_cells = []
    for i in range(11):
        for j in range(11):

    if opp_move is not None:
        return opp_move

    # Check if we can win immediately (prioritize)
    my_move = find_my_win(my_stones, opp_stones, color)

            best_move = move

    # If no clear best move, pick a random one (fallback)

        best_move = random.choice(empty_cells)

def find_opponent_win(opp_stones, my_stones, color):

def find_my_win(my_stones, opp_stones, color):

        # Check if we have a winning path

def has_winning_path(stones, opponent_stones, color, is_opponent=False):

                    # BFS from this goal cell

                    queue = deque([(row, col)])
                    visited.add((row, col))

def evaluate_move(my_stones, opp_stones, color):
    """Evaluate the strength of a move using heuristics."""
    score = 0

    # Heuristic 1: Number of potential winning paths for us
    score += len(get_potential_winning_paths(my_stones, opp_stones, color))

    # Heuristic 2: Number of opponent's potential winning paths (avoid)
    score -= len(get_potential_winning_paths(opp_stones, my_stones, 'w' if color == 'b' else 'b'))

    # Heuristic 3: Center control (prefer moves near the center)

    score += center_bias * 10

    # Heuristic 4: Longest chain length
    longest_chain = max(get_chain_lengths(my_stones, opp_stones, color))
    score += longest_chain * 2

    return score

def get_potential_winning_paths(stones, opponent_stones, color):

def get_chain_lengths(my_stones, opp_stones, color):
    """Get the lengths of all chains for 'my_stones'."""
    visited = set()
    chains = []

    for stone in my_stones:
        if stone not in visited:
            # BFS to find the chain
            queue = deque([stone])
            visited.add(stone)
            chain = set([stone])

            while queue:
                i, j = queue.popleft()
                for neighbor in get_neighbors((i, j)):
                    if neighbor in my_stones and neighbor not in visited:
                        visited.add(neighbor)
                        chain.add(neighbor)
                        queue.append(neighbor)
            chains.append(len(chain))
    return chains

def get_neighbors(cell):
    """Get all 6 neighbors of a hexagonal cell (i, j)."""
    i, j = cell
    neighbors = [
        (i-1, j), (i-1, j+1),  # Top-left and top-right
        (i, j-1), (i, j+1),    # Left and right
        (i+1, j-1), (i+1, j)   # Bottom-left and bottom-right
    ]
    # Filter out invalid neighbors (out of bounds)
    valid_neighbors = []
    for ni, nj in neighbors:
        if 0 <= ni < 11 and 0 <= nj < 11:
            valid_neighbors.append((ni, nj))
    return valid_neighbors
