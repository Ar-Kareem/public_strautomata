**Strategy Overview**

The policy follows a classic “shortest‑path” heuristic that is very effective for Hex on an 11×11 board.

1. **Board model** – The board is represented as a graph where each cell has up to six neighbours:
   ```
   (i‑1, j)   (i‑1, j+1)
   (i,   j‑1) (i,   j+1)
   (i+1, j‑1) (i+1, j)
   ```
   Cells occupied by the opponent are treated as blocked, our own stones cost 0 to traverse and empty cells cost 1.

2. **Dijkstra distances** –  
   * For the colour we play we have two opposite borders (top‑bottom for black, left‑right for white).  
   * Using Dijkstra’s algorithm we compute the minimum “cost” from every cell to each of the two borders (`distA` and `distB`).  
   * The cost of a path is the number of empty cells it must still fill, i.e. the number of moves required to complete a connection if the current stones stay where they are.

3. **Choosing the move** –  
   For every empty cell we evaluate the simple score  

   `score(cell) = distA[cell] + distB[cell]`

   (the cost of a path that goes through that cell, without counting the cell twice).  
   The empty cell with the smallest score lies on the current shortest connection route between our two borders, so placing a stone there reduces the required number of future moves the most.  
   Ties are broken randomly to avoid deterministic loops.

4. **Legal move guarantee** –  
   The algorithm only ever selects from the set of empty cells, so the returned move is always legal.

This heuristic is fast (the whole board is only 121 cells) and provides strong, often optimal, play against generic opponents.

#!/usr/bin/env python3
import heapq
import random
from typing import List, Tuple

N = 11  # board size

def _neighbors(i: int, j: int) -> List[Tuple[int, int]]:
    """Return the six axial neighbours of cell (i,j) on an 11×11 Hex board."""
    dirs = [(-1, 0), (-1, 1), (0, -1), (0, 1), (1, -1), (1, 0)]
    res = []
    for di, dj in dirs:
        ni, nj = i + di, j + dj
        if 0 <= ni < N and 0 <= nj < N:
            res.append((ni, nj))
    return res

def _dijkstra(starts: List[Tuple[int, int]],
              own: set,
              opp: set) -> List[List[int]]:
    """
    Multi‑source Dijkstra on the Hex graph.
    Moving onto a cell costs 0 if it is our own stone,
    1 if the cell is empty, and INF if it belongs to the opponent.
    Returns a 2‑D list of distances from the nearest start cell.
    """
    INF = 10 ** 9
    dist = [[INF] * N for _ in range(N)]
    heap = []

    for (i, j) in starts:
        if (i, j) in opp:
            continue                # start cells blocked by opponent are ignored
        cost = 0 if (i, j) in own else 1
        dist[i][j] = cost
        heapq.heappush(heap, (cost, i, j))

    while heap:
        cur, i, j = heapq.heappop(heap)
        if cur != dist[i][j]:
            continue
        for ni, nj in _neighbors(i, j):
            if (ni, nj) in opp:
                continue          # cannot step on opponent
            step_cost = 0 if (ni, nj) in own else 1
            nd = cur + step_cost
            if nd < dist[ni][nj]:
                dist[ni][nj] = nd
                heapq.heappush(heap, (nd, ni, nj))
    return dist

def policy(me: List[Tuple[int, int]],
           opp: List[Tuple[int, int]],
           color: str) -> Tuple[int, int]:
    """
    Returns the next move for Hex on an 11×11 board.
    `me`  – list of our stones,
    `opp` – list of opponent stones,
    `color` – 'b' (black, top‑bottom) or 'w' (white, left‑right).
    """
    own_set = set(me)
    opp_set = set(opp)

    # Determine the two opposite borders for our colour
    if color == 'b':  # black connects top (row 0) ↔ bottom (row N‑1)
        side_a = [(0, j) for j in range(N)]
        side_b = [(N - 1, j) for j in range(N)]
    else:            # white connects left (col 0) ↔ right (col N‑1)
        side_a = [(i, 0) for i in range(N)]
        side_b = [(i, N - 1) for i in range(N)]

    # Distance from each border
    dist_a = _dijkstra(side_a, own_set, opp_set)
    dist_b = _dijkstra(side_b, own_set, opp_set)

    best_score = 10 ** 9
    best_moves = []

    # Evaluate every empty cell
    for i in range(N):
        for j in range(N):
            if (i, j) in own_set or (i, j) in opp_set:
                continue  # not empty
            score = dist_a[i][j] + dist_b[i][j]
            # smaller score → closer to completing the connection
            if score < best_score:
                best_score = score
                best_moves = [(i, j)]
            elif score == best_score:
                best_moves.append((i, j))

    # As a safety net (should never happen) fall back to a random empty cell
    if not best_moves:
        empties = [(i, j) for i in range(N) for j in range(N)
                   if (i, j) not in own_set and (i, j) not in opp_set]
        return random.choice(empties)

    return random.choice(best_moves)
