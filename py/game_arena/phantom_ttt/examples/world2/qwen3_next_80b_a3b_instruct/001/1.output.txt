The strategy for this policy is to use a combination of tactical reasoning and probabilistic inference based on the imperfect information of Phantom Tic Tac Toe. Since we only know our own confirmed marks (1) and nothing about the opponent's marks (0 means uncertain), we must assume that any unconfirmed cell (0) could be either empty or occupied by the opponent. 

Our approach:
1. First, check if we can win immediately by completing a line (row, column, or diagonal) with two of our own marks and one empty/unconfirmed cell — we assume the unconfirmed cell is empty and attempt to claim it.
2. If no immediate win is possible, block potential opponent wins: look for lines where the opponent likely has two marks and one empty cell. Since we cannot see opponent marks, we infer: if a line has two 0s and one 1, it's unlikely the opponent has two marks (since we'd have seen two 1s). But if a line has two 0s and no 1s, those could be opponent marks. The safest inference is that a line with two 0s and one 1 might be a threat if the 1 is the only mark — so we treat any line with two 0s and one 1 as potentially being a winning line for the opponent if the remaining 0s are opponent's. Hence, we prioritize picking 0s in lines that already have one of our marks (to prevent them from becoming opponent wins).
3. If neither, prioritize center (1,1) — highest connectivity. Then corners — they offer the most potential winning lines. Then edges.
4. Avoid our own confirmed marks at all costs (already taken).
5. Use a small randomization on equal-value moves to avoid predictable patterns.

This strategy is robust under imperfect information, focuses on maximizing our winning chances while minimizing opponent opportunities, and is grounded in the structure of tic-tac-toe geometry.

def policy(board: list[list[int]]) -> tuple[int, int]:
    board = np.array(board)
    
    # Get coordinates of all empty/uncertain cells (0) and exclude our own confirmed cells (1)
    possible_moves = [(i, j) for i in range(3) for j in range(3) if board[i][j] == 0]

    if not possible_moves:
        return (0, 0)

    for i in range(3):

        if np.sum(board[i, :] == 1) == 2:
            for j in range(3):
                if board[i][j] == 0:
                    return (i, j)

        if np.sum(board[:, i] == 1) == 2:

                if board[j][i] == 0:
                    return (j, i)

    diag1 = [board[i][i] for i in range(3)]
    if np.sum(np.array(diag1) == 1) == 2:
        for i in range(3):
            if diag1[i] == 0:
                return (i, i)
    
    # Check anti-diagonal
    diag2 = [board[i][2-i] for i in range(3)]
    if np.sum(np.array(diag2) == 1) == 2:
        for i in range(3):
            if diag2[i] == 0:
                return (i, 2-i)

    # We prioritize blocking threats in lines with one of our marks and two 0s.
    threats = []
    
    for i in range(3):
        # Rows
        row = board[i, :]
        if np.sum(row == 1) == 1 and np.sum(row == 0) == 2:
            for j in range(3):
                if row[j] == 0:
                    threats.append((i, j))
        # Columns
        col = board[:, i]
        if np.sum(col == 1) == 1 and np.sum(col == 0) == 2:
            for j in range(3):
                if col[j] == 0:
                    threats.append((j, i))
    
    # Diagonals
    diag1 = [board[i][i] for i in range(3)]
    if np.sum(np.array(diag1) == 1) == 1 and np.sum(np.array(diag1) == 0) == 2:
        for i in range(3):
            if diag1[i] == 0:

            if diag2[i] == 0:
                threats.append((i, 2-i))
    
    # If there are threat moves, pick one from them
    if threats:
        # Pick one threat move (prefer center or corner if possible)
        for move in threats:
            if move == (1, 1):  # center
                return move
        for move in threats:
            if move in [(0,0), (0,2), (2,0), (2,2)]:  # corner
                return move
        return threats[0]  # fallback to first threat
    
    # If no immediate win or threat, prioritize center, then corners, then edges
    priority_moves = [(1,1)]  # center
    priority_moves.extend([(0,0), (0,2), (2,0), (2,2)])  # corners
    priority_moves.extend([(0,1), (1,0), (1,2), (2,1)])  # edges
    
    # Find first priority move that is still available
    for move in priority_moves:
        if move in possible_moves:
            return move
    
    # Fallback: pick any legal move (should never reach here if game state is valid)
    return possible_moves[0]
