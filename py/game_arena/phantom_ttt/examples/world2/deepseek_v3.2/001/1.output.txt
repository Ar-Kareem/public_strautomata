**Strategy Overview**  

1. **Information Handling**  
   - We cannot directly see opponent moves, but we know which cells are confirmed ours (`board[row][col] == 1`).  
   - All other cells (`board[row][col] == 0`) could be empty, ours-but-unconfirmed, or opponent’s.  

2. **Move Priorities**  
   - **Winning move:** If a row/col/diag has two confirmed ours (`1`s) and one `0` cell, try that cell to win immediately.  
   - **Blocking move:** Since we can’t see opponent’s pieces perfectly, we estimate likely opponent threats by assuming any cell not confirmed ours might be opponent’s if blocking would stop a potential line of two opponent pieces in a row.  
   - **Center control:** Try center (`1,1`) early if uncertain.  
   - **Corner preference:** After center, corners are strong.  
   - **Avoid known ours:** Never pick `board[r][c] == 1`.  

3. **Heuristic Fallback**  
   When no clear win/block, use a simple scoring:  
   - Each `0` cell gets a score based on how many confirmed ours are in its row/col/diag.  
   - Prefer cells with more “potential” (more `1`s in same line).  
   - Tie-breaking by board order.  

4. **Randomization**  
   If multiple equally good moves, pick randomly to avoid being predictable.  

**Implementation Plan**  
- Scan board for immediate win moves.  
- Scan for urgent blocks using assumption that opponent might have two in a row in any line with zero confirmed ours cells.  
- Otherwise, pick highest heuristic score cell.

import random

def policy(board: list[list[int]]) -> tuple[int, int]:
    # Helper: check line for two confirmed ours and one empty spot
    def find_winning_move(player_value=1):
        lines = []
        # rows
        for r in range(3):
            lines.append([(r, c) for c in range(3)])
        # cols
        for c in range(3):
            lines.append([(r, c) for r in range(3)])
        # diags
        lines.append([(0, 0), (1, 1), (2, 2)])
        lines.append([(0, 2), (1, 1), (2, 0)])
        
        for line in lines:
            vals = [board[r][c] for (r, c) in line]
            if vals.count(player_value) == 2:
                for (r, c) in line:
                    if board[r][c] == 0:  # unconfirmed cell
                        return (r, c)
        return None
    
    # Winning move for us
    win_move = find_winning_move(1)
    if win_move is not None:
        return win_move
    
    # Blocking move: assume opponent has value 2 (not visible to us)
    # but we guess blocking if a line has no 1s (ours) and two 0s where opponent might have two marks.
    # Actually better: if a line has exactly two cells that are 0 and one cell that is 1? That's not a block.
    # Let's treat a "block" as: a line where if opponent placed there, they'd have two in a row
    # But we can't see opponent's marks. So we approximate: if a line has 0 confirmed ours (all 0s), 
    # and exactly two of those 0 cells have been "tried by us earlier and failed"? We don't track that.
    # Alternative simpler: assume opponent is trying to win same as us, so their winning move would be
    # a line with two opponent marks. Since we don't see theirs, this is guesswork.
    # Safer: just go for heuristic after checking immediate win.
    
    # Heuristic: score each 0 cell by number of 1s in its row+col+diag
    candidates = []
    for r in range(3):
        for c in range(3):
            if board[r][c] == 0:
                score = 0
                # row
                score += sum(1 for cc in range(3) if board[r][cc] == 1)
                # col
                score += sum(1 for rr in range(3) if board[rr][c] == 1)
                # diag1
                if r == c:
                    score += sum(1 for i in range(3) if board[i][i] == 1)
                # diag2
                if r + c == 2:
                    score += sum(1 for i in range(3) if board[i][2-i] == 1)
                candidates.append((score, r, c))
    
    if not candidates:
        # fallback: pick first 0 cell
        for r in range(3):
            for c in range(3):
                if board[r][c] == 0:
                    return (r, c)
    
    # pick highest score, break ties randomly
    max_score = max(c[0] for c in candidates)
    best_candidates = [(r, c) for (sc, r, c) in candidates if sc == max_score]
    return random.choice(best_candidates)
