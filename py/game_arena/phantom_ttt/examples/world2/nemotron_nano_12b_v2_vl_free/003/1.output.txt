To address the challenge of playing Phantom Tic Tac Toe with imperfect information, we aim to create a strategy that prioritizes both winning opportunities and safe moves while accounting for the uncertainty around the opponent's moves. The key is to leverage the visible information (our own moves) and assume the opponent plays optimally, allowing us to focus on the most strategic moves available to us.

---

### **Strategy Overview**

1. **Immediate Win Detection**  
   - The AI first checks all available cells (those not yet marked by itself) to see if placing a mark in any of them would result in a win.
   - If such a move exists, the AI selects it immediately, prioritizing the most strategically valuable cell first (center, corners, then edges).

2. **Strategic Placement**  
   - If no winning move is available, the AI selects the move with the highest strategic value:
     - **Center** is the highest priority.
     - **Corners** follow the center.
     - **Edges** are the last priority.
   - This mimics the standard optimal strategy for Tic Tac Toe, ensuring the AI plays effectively even when it doesn't know the opponent's moves.

3. **Avoiding Failed Moves**  
   - The AI only considers cells that are currently marked as `0` (not confirmed to be our own). These are assumed to be either empty or occupied by the opponent.
   - By avoiding cells already marked `1` (our own), we ensure all returned moves are valid.

### **Code Implementation**

```python
def policy(board):
    # Collect all possible moves (cells with 0)
    possible_moves = []
    for i in range(3):
        for j in range(3):
            if board[i][j] == 0:
                possible_moves.append((i, j))

    # Check if any of the possible moves would result in a win
    for move in [(1, 1), (0, 0), (0, 2), (2, 0), (2, 2), (0, 1), (1, 0), (1, 2), (2, 1)]:
        if move not in possible:
            continue

        # Simulate placing move
        temp = [row.copy() for row in board]
        temp[move[0]][move[1]] = 1

        # Check row
        if all(temp[move[0]][x] == 1 for x in range(3)):
            return move
        # Check column
        col = [temp[x][move[1]] for x in range(3)]
        if all(cell == 1 for cell in col):
            return move
        # Check diagonals
        if move[0] == move[1]:  # Main diagonal
            diag = [temp[x][x] for x in range(3)]
            if all(cell == 1 for cell in diag):
                return move
        if move[0] + move[1] == 2:  # Anti-diagonal
            diag = [temp[x][2 - x] for x in range(3)]
            if all(cell == 1 for cell in diag):
                return move

    # If no winning move, choose the highest priority available cell
    for move in [(1, 1), (0, 0), (0, 2), (2, 0), (2, 2), (0, 1), (1, 0), (1, 2), (2, 1)]:
        if move in possible:
            return move

    # Fallback (shouldn't happen if there are possible moves)
    return possible[0]
```

---

### **Explanation of Strategy**

- **Winning Moves First**: The AI prioritizes any move that would immediately result in a win. This is the most critical decision point in Tic Tac Toe.
- **Strategic Ordering**: If no immediate win is available, the AI selects the most valuable cell in the standard Tic Tac Toe strategy:
  - **Center** `(1, 1)` is the most valuable.
  - **Corners** `(0,0), (0,2), (2,0), (2,2)` follow.
  - **Edges** `(0,1), (1,0), (1,2), (2,1)` are the least strategic.
- **Simulated Moves**: For each candidate move, the AI simulates placing a mark and checks if it completes a row, column, or diagonal of three marks.
- **Risk Mitigation**: By prioritizing cells that are more likely to be empty (center, corners), the AI reduces the chance of failed moves while maintaining optimal play.

This strategy balances aggressive winning attempts with a conservative playstyle that minimizes the risk of invalid moves, making it a robust choice in the Phantom Tic Tac Toe environment.
