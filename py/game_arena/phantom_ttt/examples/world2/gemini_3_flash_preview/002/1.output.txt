### Strategy for Phantom Tic Tac Toe

In Phantom Tic Tac Toe, the challenge is minimizing wasted turns on cells already occupied by the opponent while pursuing a winning line. Because the board only shows your successful marks (`1`) and everything else as unconfirmed (`0`), the policy must prioritize:
1.  **Immediate Wins:** If you have two marks in a row/column/diagonal and the third cell is unconfirmed, attempt to take it.
2.  **Strategic Blocking:** Since we cannot see the opponent's pieces, we assume they are playing optimally. However, without knowing their positions, the best heuristic is to control the **Center**, then the **Corners**, then the **Edges**.
3.  **Information Gathering / Persistence:** If a move fails (handled by the game engine), we will be called again. In this specific API, the `board` reflects our successful placements.
4.  **Probability of Availability:** Highly contested spots (like the center) are likely to be taken first by an opponent. If our own marks are not yet in the center, we try it immediately.

The following code implements a prioritized search order: Win-threats > Center > Corners > Random Edges.

<code>
import random

def policy(board: list[list[int]]) -> tuple[int, int]:
    """
    Phantom Tic Tac Toe Policy.
    board[r][c] == 1: Confirmed mine.
    board[r][c] == 0: Unknown (Empty or Opponent's).
    """

    # 1. Check for immediate winning moves
    # We look for lines where we have 2 marks and 1 unknown.
    lines = []
    # Rows
    for r in range(3):
        lines.append([(r, 0), (r, 1), (r, 2)])
    # Cols
    for c in range(3):
        lines.append([(0, c), (1, c), (2, c)])
    # Diagonals
    lines.append([(0, 0), (1, 1), (2, 2)])
    lines.append([(0, 2), (1, 1), (2, 0)])

    for line in lines:
        marks = [board[r][c] for r, c in line]
        if sum(marks) == 2:
            # Find the index of the 0
            for r, c in line:
                if board[r][c] == 0:
                    return (r, c)

    # 2. Heuristic positioning
    # Priority: Center -> Corners -> Edges
    
    # Center
    if board[1][1] == 0:
        return (1, 1)

    # Corners
    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    random.shuffle(corners)
    for r, c in corners:
        if board[r][c] == 0:
            return (r, c)

    # Edges
    edges = [(0, 1), (1, 0), (1, 2), (2, 1)]
    random.shuffle(edges)
    for r, c in edges:
        if board[r][c] == 0:
            return (r, c)

    # Fallback (should not be reached if there are any 0s left)
    for r in range(3):
        for c in range(3):
            if board[r][c] == 0:
                return (r, c)

    return (0, 0) # Ultimate fallback
