To solve this problem, we need to create a strategy for a game of Phantom Tic Tac Toe where both players' moves are not directly visible. The goal is to choose the best move from the available legal moves to maximize the chances of winning.

### Approach
The solution involves two main steps:
1. **Check for Immediate Winning Move**: Evaluate each legal move to see if it would result in a win by forming a line of three marks in a row, column, or diagonal.
2. **Select Optimal Move Based on Priority**: If no winning move is available, prioritize moves based on their strategic importance, considering the number of lines (row, column, diagonal) each move is part of. The center cell is the most valuable, followed by corners and edges.

### Solution Code
```python
import random

def policy(board, legal_moves):
    # Check if any legal move can result in a win
    winning_moves = []
    for cell in legal_moves:
        row, col = cell // 3, cell % 3
        if check_win(board, row, col):
            winning_moves.append(cell)
    
    if winning_moves:
        return (random.choice(winning_moves) // 3, random.choice(winning_moves) % 3)
    
    # Define cell priorities based on their strategic importance
    cell_weights = [3, 2, 3, 2, 4, 2, 3, 2, 3]
    candidates = sorted(legal_moves, key=lambda x: (-cell_weights[x], x))
    
    # Select the highest priority cell
    chosen_cell = candidates[0]
    return (chosen_cell // 3, chosen_cell % 3)

def check_win(board, row, col):
    # Check if placing a mark at (row, col) would complete a three-in-a-row
    # Check row
    win = True
    for j in range(3):
        if j != col and board[row][j] != 1:
            win = False
    if win:
        return True
    
    # Check column
    win = True
    for i in range(3):
        if i != row and board[i][col] != 1:
            win = False
    if win:
        return True
    
    # Check main diagonal (if applicable)
    if row == col:
        win = True
        for i in range(3):
            if i != row and board[i][i] != 1:
                win = False
        if win:
            return True
    
    # Check anti-diagonal (if applicable)
    if row + col == 2:
        win = True
        for i in range(3):
            if i != row and board[i][2 - i] != 1:
                win = False
        if win:
            return True
    
    return False
```

### Explanation
1. **Winning Move Check**: The function `check_win` determines if placing a mark at a given cell would result in a winning line (three in a row, column, or diagonal). It checks each relevant line (row, column, and diagonals) to see if all cells in the line are already occupied by the player's marks (except the current cell being evaluated).

2. **Prioritizing Moves**: If no immediate winning move is found, the code prioritizes moves based on a predefined weight system. The center cell has the highest weight (4), followed by corners (3), and edges (2). This strategic priority ensures that the code places marks in positions that are most likely to contribute to future wins.

3. **Selecting the Best Move**: The legal moves are sorted by their weights in descending order. The highest priority move (center if available, then corners, then edges) is selected. This approach maximizes the chances of forming a winning line in subsequent moves while considering potential threats.

By following this approach, the code efficiently leverages the game's structure to make optimal moves, balancing immediate wins and strategic positioning to outmaneuver the opponent.
