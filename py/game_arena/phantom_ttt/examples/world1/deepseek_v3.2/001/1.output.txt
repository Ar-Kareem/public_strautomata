**Strategy Overview**

For Phantom Tic Tac Toe, the key challenges are:

1. **Imperfect Information**: The board only shows your confirmed marks (`1`) and "unconfirmed" cells (`0`). An unconfirmed cell could be empty, opponent-occupied, or your own unconfirmed mark.
2. **Move Rejection**: If you try to place a mark where the opponent already placed theirs, your move fails (no mark is placed, but turn passes).
3. **Goal**: Still get 3 in a row, but with hidden opponent positions.

**Key Tactics:**

- **Prioritize winning**: First check if any move gives you an immediate confirmed win (completes a row/col/diag of your `1`s).
- **Block known threats**: Since opponent positions are hidden, we can't directly block. But we can try to place marks in positions that would block *if* the opponent has certain positions. Because information is hidden, we instead prioritize high-value strategic cells (center, corners) that maximize possible winning lines.
- **Adapt to confirmed board**: If we have two in a row (with `1`s) and the third cell is unconfirmed (`0`), attempt that cell—it might complete the line or fail if opponent occupies it.
- **Center then corners**: Classic Tic Tac Toe opening priority works here too: center (1,1) is most valuable, then corners, then edges.
- **Random tie-breaking**: Among equally good strategic cells, pick randomly to avoid being predictable.

**Implementation Plan:**

1. Convert `board` and `legal_moves` to usable forms.
2. Check for any immediate winning moves (two `1`s and an unconfirmed in a line).
3. Otherwise, prioritize:  
   a) Center (4 in 0–8 indexing)  
   b) Corners (0,2,6,8)  
   c) Edges (1,3,5,7)  
   but only if unconfirmed (`board` shows `0` there) and legal.
4. Fallback: choose randomly from legal moves (must always return legal).

import random
from typing import List, Tuple

def policy(board: List[List[int]], legal_moves: List[int]) -> Tuple[int, int]:
    # Convert legal_moves from 0-8 to (r,c) form
    legal_cells = [(move // 3, move % 3) for move in legal_moves]
    
    # Helper: check line for two 1s and one unconfirmed (0) cell
    def winning_move_available():
        lines = [
            # rows
            [(0,0),(0,1),(0,2)],
            [(1,0),(1,1),(1,2)],
            [(2,0),(2,1),(2,2)],
            # cols
            [(0,0),(1,0),(2,0)],
            [(0,1),(1,1),(2,1)],
            [(0,2),(1,2),(2,2)],
            # diags
            [(0,0),(1,1),(2,2)],
            [(0,2),(1,1),(2,0)]
        ]
        for line in lines:
            vals = [board[r][c] for r,c in line]
            if vals.count(1) == 2:
                # Find the unconfirmed cell in this line
                for r,c in line:
                    if board[r][c] == 0 and (r,c) in legal_cells:
                        return (r,c)
        return None
    
    win_move = winning_move_available()
    if win_move:
        return win_move
    
    # Strategy priority: center, corners, edges
    priority_order = [(1,1), (0,0), (0,2), (2,0), (2,2), (0,1), (1,0), (1,2), (2,1)]
    
    for cell in priority_order:
        if cell in legal_cells and board[cell[0]][cell[1]] == 0:
            return cell
    
    # Fallback: random legal move (should always be reachable)
    return random.choice(legal_cells)
