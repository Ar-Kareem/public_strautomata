In Phantom Tic Tac Toe, the challenge lies in the uncertainty of the opponent's pieces. Since we only know our own confirmed positions and are given a list of legal moves (cells we haven't successfully claimed yet), the strategy focuses on maximizing our chances of completing a line while blocking potential opponent lines.

### Strategy
1.  **Immediate Win**: Check if any move in the `legal_moves` list can complete a line of three for us based on our confirmed marks (`board`).
2.  **Strategic Blocking**: Since we cannot see the opponent's marks, we assume they might have pieces in cells that are not ours. We prioritize cells that are common in many winning lines (like the center) and cells that would obstruct the most likely lines for an opponent.
3.  **Heuristic Scoring**: Every legal move is assigned a score based on:
    *   **Center Control**: The center (index 4) is highly valued.
    *   **Corner Control**: Corners (0, 2, 6, 8) are valued over edges.
    *   **Line Potential**: We prioritize moves that contribute to lines where we already have one mark and the rest are "open" (not occupied by us).
4.  **Fallback**: If no strategic move stands out, the agent picks the first available legal move to ensure a valid action is always returned.

<code>
import random

def policy(board: list[list[int]], legal_moves: list[int]) -> tuple[int, int]:
    # Flatten the board for easier indexing
    flat_board = []
    for row in board:
        flat_board.extend(row)
    
    # Define winning combinations
    winning_combos = [
        [0, 1, 2], [3, 4, 5], [6, 7, 8], # Rows
        [0, 3, 6], [1, 4, 7], [2, 5, 8], # Cols
        [0, 4, 8], [2, 4, 6]             # Diagonals
    ]
    
    # 1. Check for immediate win
    for move in legal_moves:
        for combo in winning_combos:
            if move in combo:
                others = [flat_board[i] for i in combo if i != move]
                if all(val == 1 for val in others):
                    return (move // 3, move % 3)
                    
    # 2. Positional Heuristics
    # Prefer center, then corners, then edges
    weights = {4: 10, 0: 5, 2: 5, 6: 5, 8: 5, 1: 2, 3: 2, 5: 2, 7: 2}
    
    # 3. Line potential: prefer moves that are part of lines we are building
    move_scores = {move: weights.get(move, 0) for move in legal_moves}
    
    for move in legal_moves:
        for combo in winning_combos:
            if move in combo:
                # Count how many pieces we have in this line
                count_ours = sum(1 for i in combo if flat_board[i] == 1)
                # In Phantom Tic Tac Toe, we don't know the opponent's positions,
                # but we prioritize lines where we already have progress.
                move_scores[move] += (count_ours * 3)

    # Find the move with the highest score
    best_move = legal_moves[0]
    max_score = -1
    
    for move in legal_moves:
        if move_scores[move] > max_score:
            max_score = move_scores[move]
            best_move = move
            
    return (best_move // 3, best_move % 3)
