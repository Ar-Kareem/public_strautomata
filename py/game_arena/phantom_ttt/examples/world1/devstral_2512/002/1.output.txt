# Strategy for Phantom Tic Tac Toe Policy

This policy implements a smart strategy for Phantom Tic Tac Toe with imperfect information. The approach combines several key elements:

1. **Immediate Win Check**: First checks if any move would complete a winning line (3 in a row) for the player.

2. **Opponent Blocking**: Attempts to block potential opponent wins by identifying cells that would complete a line for the opponent based on the current board state.

3. **Center Control**: Prioritizes the center position (4) as it's part of the most potential winning lines.

4. **Corner Preference**: Prefers corner positions (0, 2, 6, 8) over edges when no immediate threats exist.

5. **Random Selection**: Falls back to random selection from legal moves when no strategic advantage is apparent.

The policy handles the imperfect information by:
- Only considering confirmed player marks (1s) when evaluating potential wins
- Avoiding cells already confirmed as player marks
- Making reasonable assumptions about opponent positions based on the current board state

The implementation is efficient, using simple list operations and avoiding expensive computations to stay within the 1-second constraint.

import random

def policy(board: list[list[int]], legal_moves: list[int]) -> tuple[int, int]:
    # Flatten the board for easier processing
    flat_board = [cell for row in board for cell in row]

    # Convert legal moves to (row, col) format
    legal_positions = [(move // 3, move % 3) for move in legal_moves]

    # Check for immediate winning moves
    for move in legal_positions:
        row, col = move
        # Create a temporary board with this move
        temp_board = [row[:] for row in board]
        temp_board[row][col] = 1

        # Check if this move wins
        if (check_win(temp_board, 1)):
            return move

    # Check for opponent blocking opportunities
    # We'll assume opponent has marks where we don't have confirmed marks (0s)
    # and look for positions that would complete their lines
    for move in legal_positions:
        row, col = move
        # Create a temporary board assuming opponent has this position
        temp_board = [row[:] for row in board]
        temp_board[row][col] = -1  # Using -1 to represent opponent

        # Check if this would complete opponent's win
        if check_win(temp_board, -1):
            return move

    # Prefer center if available
    if 4 in legal_moves:
        return (1, 1)

    # Prefer corners if available
    corners = [0, 2, 6, 8]
    available_corners = [pos for pos in corners if pos in legal_moves]
    if available_corners:
        return (available_corners[0] // 3, available_corners[0] % 3)

    # If no strategic move found, return a random legal move
    return random.choice(legal_positions)

def check_win(board, player):
    # Check rows
    for row in board:
        if all(cell == player for cell in row):
            return True

    # Check columns
    for col in range(3):
        if all(board[row][col] == player for row in range(3)):
            return True

    # Check diagonals
    if board[0][0] == board[1][1] == board[2][2] == player:
        return True
    if board[0][2] == board[1][1] == board[2][0] == player:
        return True

    return False
