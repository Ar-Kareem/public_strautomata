The strategy for this policy is to prioritize winning moves first, then block potential opponent wins, and finally select the best available move based on game theory principles for imperfect information. Since we cannot see opponent marks directly, we assume that any cell that is not confirmed as ours (i.e., board[row][col] == 0) may be occupied by the opponent or empty. We use the following approach:

1. **Win Detection**: Look for any row, column, or diagonal where we have 2 marks and the third cell is legal — prioritize playing there to win immediately.
2. **Block Detection**: Since opponent can also be close to winning, we treat any cell with 2 zeros (unknowns) in a line with one of our marks as potentially dangerous. However, without seeing opponent placements, we assume that any pair of zeros in a line (with our mark) might mean the opponent has two in a row. We block those lines if possible.
3. **Center Priority**: The center (1,1) is the most valuable square in Tic Tac Toe — prioritize it if available.
4. **Corner Preference**: After center, prefer corners (0,0), (0,2), (2,0), (2,2) as they are part of more potential lines.
5. **Edge Avoidance**: Avoid edges (0,1), (1,0), (1,2), (2,1) unless no other moves are available.
6. **Fallback**: If multiple legal moves remain, choose randomly from those that are not already ours but weighted by strategic value.

We use a score-based system to rank all legal moves (excluding our own confirmed cells). We assign higher scores to center and corners, and we prioritize moves that complete our lines or block opponent lines.

def check_win(board, row, col):
    """Check if placing a mark at (row, col) results in a win."""
    # Check row
    if all(board[row, :] == 1):
        return True
    # Check column
    if all(board[:, col] == 1):
        return True
    # Check main diagonal
    if row == col and all(np.diag(board) == 1):
        return True
    # Check anti-diagonal
    if row + col == 2 and all(np.diag(np.fliplr(board)) == 1):
        return True
    return False
