### Strategy for Phantom Tic Tac Toe

Phantom Tic Tac Toe introduces imperfect information, meaning we don't know the opponent's moves unless they conflict with ours. Here's the strategy:

1. **Prioritize Winning Moves**: First, check if any move can complete a line (row, column, or diagonal) with our confirmed marks (`1`s). If such a move exists, take it.

2. **Block Opponent's Potential Wins**: Since we don't see the opponent's marks, we infer their possible positions based on our `0`s (unconfirmed cells). If a line has two `0`s and no `1`s, assume the opponent might be close to winning and block by placing in the remaining cell.

3. **Take the Center**: The center is the most strategic position in Tic Tac Toe. If it's legal and not already confirmed ours, take it.

4. **Take Corners**: Corners are the next best positions. Prioritize empty corners to maximize winning opportunities.

5. **Take Edges**: As a last resort, take any remaining legal edge positions.

6. **Random Choice**: If no strategic moves are available (unlikely), choose randomly from legal moves.

This strategy balances offense (winning) and defense (blocking) while leveraging the strongest positions on the board.

import numpy as np

def policy(board: list[list[int]], legal_moves: list[int]) -> int:
    # Convert board to numpy array for easier manipulation
    board_np = np.array(board)
    legal_moves_np = np.array([(move // 3, move % 3) for move in legal_moves])
    
    # Check for winning moves (complete a line with two 1s and one empty)
    lines = [
        # Rows
        [(0, 0), (0, 1), (0, 2)],
        [(1, 0), (1, 1), (1, 2)],
        [(2, 0), (2, 1), (2, 2)],
        # Columns
        [(0, 0), (1, 0), (2, 0)],
        [(0, 1), (1, 1), (2, 1)],
        [(0, 2), (1, 2), (2, 2)],
        # Diagonals
        [(0, 0), (1, 1), (2, 2)],
        [(0, 2), (1, 1), (2, 0)]
    ]
    
    for line in lines:
        values = [board_np[row, col] for row, col in line]
        if values.count(1) == 2 and 0 in values:
            empty_pos = line[values.index(0)]
            if empty_pos in legal_moves_np:
                return empty_pos
    
    # Check for blocking moves (two 0s and no 1s in a line)
    for line in lines:
        values = [board_np[row, col] for row, col in line]
        if values.count(0) == 2 and 1 not in values:
            empty_pos = line[values.index(0)]
            if empty_pos in legal_moves_np:
                return empty_pos
    
    # Prefer center if available
    center = (1, 1)
    if center in legal_moves_np and board_np[1, 1] == 0:
        return center
    
    # Prefer corners
    corners = [(0, 0), (0, 2), (2, 0), (2, 2)]
    for corner in corners:
        if corner in legal_moves_np and board_np[corner] == 0:
            return corner
    
    # Prefer edges
    edges = [(0, 1), (1, 0), (1, 2), (2, 1)]
    for edge in edges:
        if edge in legal_moves_np and board_np[edge] == 0:
            return edge
    
    # Fallback: choose first legal move (shouldn't happen if legal_moves is correct)
    return legal_moves_np[0]
