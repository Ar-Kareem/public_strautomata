The game Clobber requires us to make a strategic decision about moving our pieces by capturing adjacent opponent pieces. A key feature to note is that we must select a legal move that always involves moving to a square occupied by an opponent's piece, ensuring the opponent's piece is removed from the board.

To devise a suitable policy function, we can break down the steps as follows:

1. **Identify Your Pieces**: Iterate over the board to find your pieces (cells where `you[row, col] == 1`).
   
2. **Check Adjacent Opponent Pieces**: For each of your pieces, check its orthogonal neighbors (up, down, left, right) to find any opponent pieces (cells where `opponent[row, col] == 1`).

3. **Store Legal Moves**: For each adjacent opponent piece found, we will construct the move string in the required format and store it.

4. **Select a Move**: From the list of legal moves, we can return one. If there are multiple options, any selection method can be used (like returning the first legal move).

5. **Output**: The move should be formatted as string `row,col,dir` where `dir` indicates the direction of the move.

The following Python code implements this logic:

```python
def policy(you: list[int], opponent: list[int]) -> str:
    rows, cols = 5, 6
    directions = [(0, -1, 'L'), (0, 1, 'R'), (-1, 0, 'U'), (1, 0, 'D')]  # (row_change, col_change, dir)
    
    for row in range(rows):
        for col in range(cols):
            if you[row][col] == 1:  # You have a piece here
                for dr, dc, dir in directions:
                    new_row, new_col = row + dr, col + dc
                    if 0 <= new_row < rows and 0 <= new_col < cols:  # Check bounds
                        if opponent[new_row][new_col] == 1:  # There's an opponent's piece
                            return f"{row},{col},{dir}"  # Return the legal move

    # Since it's guaranteed a move exists, we shouldn't reach here
    raise RuntimeError("No legal moves available when the function was called.")
```

Here, the key parts of the implementation consist of:
- A nested loop to iterate over each cell in your representation of the board.
- Checking orthogonal directions around each of your pieces.
- Constructing a legal move as soon as one is found and returning it immediately.

Since the problem states that `policy` will never be called with no legal moves available, this function will always return a valid move string.
