To determine a move selection policy for the game Clobber, we want to strategically select a move that captures an opponent's piece. The goal is to make a legal move by identifying which of our pieces can move into a neighboring square occupied by an opponent piece, thus removing that piece from the board.

### Approach and Strategy:
1. **Identify Your Pieces**: Scan the board to find all of your pieces.
2. **Evaluate Possible Moves**: For each of your pieces, check the orthogonal neighbors (up, down, left, right) to see if they are occupied by an opponent's piece.
3. **Construct the Move**: For every valid move (where your piece can capture an opponent's piece), create a corresponding move string in the format `'row,col,dir'`.
4. **Return a Valid Move**: Since the prompt guarantees that there will always be at least one legal move available, we can choose the first valid capture found.

The strategy favors simplicity by just choosing the first available capture, as there is no need to analyze deeper strategies like position evaluation or future consequences in this case.

### Python Implementation:
Here's the Python code for the move selection policy:

```python
def policy(you: list[int], opponent: list[int]) -> str:
    moves = []

    # Directions for moving: (row_offset, col_offset, direction)
    directions = [(-1, 0, 'U'), (1, 0, 'D'), (0, -1, 'L'), (0, 1, 'R')]

    for row in range(5):
        for col in range(6):
            if you[row][col] == 1:  # If this cell has your piece
                for dr, dc, direction in directions:
                    new_row = row + dr
                    new_col = col + dc
                    # Check if the new position is in bounds and has an opponent's piece
                    if 0 <= new_row < 5 and 0 <= new_col < 6 and opponent[new_row][new_col] == 1:
                        # Found a valid move
                        return f"{row},{col},{direction}"

    # Should never reach here based on problem statement
    return ""

```

### Explanation:
- **Loop through the board**: The nested loops iterate over each cell in the 5x6 board.
- **Check for your piece**: If a cell contains your piece, it checks each of the four possible directions for a valid move.
- **Return a valid capture**: The first valid capture found is returned immediately in the required format. 

This simple and straightforward approach ensures we always return a valid move while adhering to the game's rules and structure.
