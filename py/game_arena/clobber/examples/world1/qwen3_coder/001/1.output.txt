1. Find all legal moves - moves where I can capture an adjacent opponent piece
2. Return one of these moves in the format 'row,col,dir'

A legal move requires:
- Starting position has my piece (you[row][col] == 1)
- Moving orthogonally (up, right, down, left) to adjacent square
- Destination square has opponent piece (opponent[new_row][new_col] == 1)
- Destination square is within bounds

My strategy should be:
1. Generate all possible legal moves
2. Select the best move based on some evaluation criteria

For a simple but effective strategy, I'll:
- Look for moves that maximize my future mobility
- Prefer moves that don't isolate my pieces
- Possibly look for moves that limit opponent's options

                    if 0 <= new_row < 5 and 0 <= new_col < 6:
