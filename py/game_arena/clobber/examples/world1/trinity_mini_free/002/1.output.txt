

The strategy involves checking eachof my pieces in row-major order (top to bottom, left to right) and for each piece, checking the four possible orthogonal directions (up, right, down, left) for adjacent opponent pieces. The first valid move encountered is returned immediately. This ensures a legal move is always returned as required, leveraging the guarantee that at least one legal move exists when the function is called.

def policy(you, opponent):
    for row in range(5):
        for col in range(6):
            if you[row][col] == 1:
                # Check up
                if row > 0 and opponent[row-1][col] == 1:
                    return f"{row},{col},U"
                # Check right
                if col < 5 and opponent[row][col+1] == 1:
                    return f"{row},{col},R"
                # Check down
                if row < 4 and opponent[row+1][col] == 1:
                    return f"{row},{col},D"
                # Check left
                if col > 0 and opponent[row][col-1] == 1:
                    return f"{row},{col},L"
