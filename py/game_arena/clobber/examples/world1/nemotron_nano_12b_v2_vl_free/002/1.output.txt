To solve this problem, we need to write a Python function that selects a valid move for the game Clobber based on the current state of the board. The function must return a move string in the format 'row,col,dir' where the move is always legal.

### Approach
The approach involves systematically checking each cell on the board to find a valid move. The steps are as follows:

1. **Iterate through each cell**: Check each cell in row-major order (from top-left to bottom-right) to see if it is occupied by our piece (value 1).
2. **Check all directions**: For each cell occupied by our piece, check all four possible orthogonal directions (up, right, down, left) to determine if moving in that direction would land on an opponent's piece.
3. **Validate the move**: Ensure the destination cell is within the bounds of the board and contains an opponent's piece. If a valid move is found, return it immediately in the specified format.

This approach ensures that the first valid move found in the specified order is returned, which is efficient and meets the problem's requirements.

### Solution Code

<code>
def policy(you: list[int], opponent: list[int]) -> str:
    # Define possible directions (Up, Right, Down, Left)
    directions = [(-1, 0, 'U'), (0, 1, 'R'), (1, 0, 'D'), (0, -1, 'L')]
    # Iterate through each row and column
    for row in range(5):
        for col in range(6):
            if you[row][col] == 1:
                # Check each direction in order
                for dr, dc, dir_char in directions:
                    new_row = row + dr
                    new_col = col + dc
                    # Check if the new position is within bounds
                    if 0 <= new_row < 5 and 0 <= new_col < 6:
                        if opponent[new_row][new_col] == 1:
                            # Return the first valid move found
                            return f"{row},{col},{dir_char}"
    # The problem states there will always be a legal move
    return ""
</code>

### Explanation
1. **Direction Handling**: The directions are checked in the order up, right, down, left. Each direction is represented by a tuple indicating the delta row and column, along with the corresponding character.
2. **Row and Column Iteration**: The nested loops iterate through each cell in row-major order, ensuring we check all rows from top to bottom and columns from left to right.
3. **Validation Checks**: For each direction, the destination cell is checked to ensure it is within bounds and contains an opponent's piece. The first valid move found in the specified order is returned immediately, ensuring efficiency.

This method efficiently finds a valid move by leveraging systematic iteration and early termination upon finding the first valid move, adhering to the constraints and ensuring correctness.
