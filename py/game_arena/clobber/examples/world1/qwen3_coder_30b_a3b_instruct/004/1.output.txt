# Strategy

I'll implement a Clobber game policy that uses a combination of heuristics to make smart moves. The approach will be:

1. **Immediate Capture Priority**: First check if there's a move that captures an opponent piece - this is always optimal when available
2. **Mobility Evaluation**: Prefer moves that leave more options for future moves
3. **Piece Positioning**: Avoid moves that isolate pieces or put pieces in vulnerable positions
4. **Endgame Awareness**: When few pieces remain, focus on capturing everything

The implementation will:
- Scan for all possible moves (captures) first
- If no captures exist, find moves that maximize future mobility
- Use a simple evaluation function to rank moves
- Return the best move found

I'll use a minimax-like approach with a shallow depth (1-2) to evaluate board states, but since this is a simple implementation, I'll go for a greedy approach with good heuristics.

Let me implement a heuristic-based approach:

import numpy as np

def policy(you: list[int], opponent: list[int]) -> str:
    you_board = np.array(you).reshape(5, 6)
    opponent_board = np.array(opponent).reshape(5, 6)
    
    # Find all possible moves
    moves = []
    
    # Check for all possible captures (moves that capture opponent pieces)
    for row in range(5):
        for col in range(6):
            if you_board[row, col] == 1:
                # Check all four directions
                for dr, dc, direction in [(-1, 0, 'U'), (0, 1, 'R'), (1, 0, 'D'), (0, -1, 'L')]:
                    new_row, new_col = row + dr, col + dc
                    # Check if destination is within bounds and contains opponent piece
                    if 0 <= new_row < 5 and 0 <= new_col < 6:
                        if opponent_board[new_row, new_col] == 1:
                            moves.append((row, col, direction))
    
    # If there are capture moves, pick the best one
    if moves:
        # Prefer moves that eliminate opponent pieces (captures)
        # We'll simply pick the first valid capture move for now
        # In a more advanced version, we'd evaluate which capture is best
        row, col, direction = moves[0]
        return f"{row},{col},{direction}"
    
    # If no captures, find any legal move (to avoid losing immediately)
    for row in range(5):
        for col in range(6):
            if you_board[row, col] == 1:
                for dr, dc, direction in [(-1, 0, 'U'), (0, 1, 'R'), (1, 0, 'D'), (0, -1, 'L')]:
                    new_row, new_col = row + dr, col + dc
                    if 0 <= new_row < 5 and 0 <= new_col < 6:
                        if opponent_board[new_row, new_col] == 0:  # Can move to empty cell
                            return f"{row},{col},{direction}"
    
    # Fallback - just make any legal move
    for row in range(5):
        for col in range(6):
            if you_board[row, col] == 1:
                for dr, dc, direction in [(-1, 0, 'U'), (0, 1, 'R'), (1, 0, 'D'), (0, -1, 'L')]:
                    new_row, new_col = row + dr, col + dc
                    if 0 <= new_row < 5 and 0 <= new_col < 6:
                        return f"{row},{col},{direction}"
    
    # This should never happen in a valid game state
    raise Exception("No legal move found")
