The strategy for this policy is to implement a smart backgammon player that prioritizes bearing off when possible, entering checkers from the bar when necessary, and making strategic moves based on board position and pip count. The policy will:

1. Handle special cases first (no dice, both dice, one die)
2. Prioritize bearing off when all checkers are home
3. Handle bar entries when checkers are on the bar
4. Make strategic moves by evaluating possible positions
5. Always ensure legal moves are returned in proper format

The implementation will use pip count evaluation, prioritize safe moves, and follow backgammon best practices like not leaving blots when possible.

def evaluate_position(my_pts, opp_pts):
    # Simple evaluation function - prefer:
    # 1. Moving checkers closer to home
    # 2. Safe points (not blots)
    # 3. Occupying higher points when possible
    score = 0
    
    # Pip count (lower is better for us)
    my_pip = sum((i+1) * my_pts[i] for i in range(24))
    opp_pip = sum((24-i) * opp_pts[i] for i in range(24))
    score += (opp_pip - my_pip) * 0.1
    
    # Prefer safe points
    for i in range(24):
        if my_pts[i] >= 2:
            score += 2  # Safe point
        elif my_pts[i] == 1:
            score -= 1  # Blot
    
    # Prefer advancing checkers
    home_count = sum(my_pts[i] for i in range(6))
    score += home_count * 0.5
    
    return score
