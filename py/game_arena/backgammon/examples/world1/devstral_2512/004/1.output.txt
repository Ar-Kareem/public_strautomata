### Strategy for the Backgammon Policy

The strategy for this policy is to prioritize the following actions in order of importance:

1. **Bearing Off**: If all checkers are in the home board (points 0-5), prioritize bearing off checkers using the highest possible die values to maximize efficiency.
2. **Hitting Opponent's Blots**: If there's an opportunity to hit an opponent's blot (a single checker on a point), prioritize this to disrupt their position.
3. **Moving Checkers Forward**: Move checkers forward to safer positions, especially to points where they can form primes or block the opponent.
4. **Re-entering from the Bar**: If checkers are on the bar, prioritize re-entering them into the opponent's home board.
5. **Avoiding Vulnerable Positions**: Avoid leaving blots in positions where they can be easily hit by the opponent.
6. **Using Both Dice**: If possible, use both dice to maximize the movement of checkers.

The policy will evaluate all possible legal moves based on the current state and dice roll, then select the move that best aligns with the above priorities. The move selection will also consider the order of dice usage (higher or lower die first) to ensure the move is legal and optimal.

    my_pts = state['my_pts']
    opp_pts = state['opp_pts']
    my_bar = state['my_bar']
    opp_bar = state['opp_bar']
    my_off = state['my_off']
    opp_off = state['opp_off']
    dice = state['dice']

    # Check if we have checkers on the bar
    if my_bar > 0:
        # Must re-enter from the bar
        # Try to re-enter using the higher die first
        higher_die = max(dice)
        lower_die = min(dice)
        # Check if we can re-enter with the higher die
        re_entry_point = 23 - higher_die
        if opp_pts[re_entry_point] < 2:
            # Can re-enter with higher die
            # Check if we can also re-enter with the lower die
            re_entry_point_lower = 23 - lower_die
            if opp_pts[re_entry_point_lower] < 2 and len(dice) == 2:
                return f"H:B,B"
            else:
                return f"H:B,P"
        else:
            # Try with the lower die
            re_entry_point_lower = 23 - lower_die
            if opp_pts[re_entry_point_lower] < 2:
                return f"L:B,P"
            else:
                return "H:P,P"

    # Check if all checkers are in the home board (points 0-5)
    all_in_home = True
    for i in range(6, 24):
        if my_pts[i] > 0:
            all_in_home = False
            break

    if all_in_home:
        # Bearing off phase

        higher_die = max(dice)
        lower_die = min(dice)
        # Find the highest point with a checker
        for i in range(5, -1, -1):
            if my_pts[i] > 0:
                if i + 1 <= higher_die:
                    # Can bear off with higher die
                    # Check if we can also bear off with the lower die
                    remaining_checkers = my_pts.copy()
                    remaining_checkers[i] -= 1
                    for j in range(5, -1, -1):
                        if remaining_checkers[j] > 0:
                            if j + 1 <= lower_die:
                                return f"H:A{i},A{j}"
                            else:
                                return f"H:A{i},P"

    # Not in bearing off phase, try to hit opponent's blots or move forward

    possible_moves = []

            # Check higher die move
            dest_higher = i - higher_die

                for j in range(24):
                    if j != i and my_pts[j] > 0:
                        dest_lower = j - lower_die
                        if dest_lower >= 0 and opp_pts[dest_lower] < 2:

                possible_moves.append((f"H:A{i},P", 2))

            dest_lower = i - lower_die
            if dest_lower >= 0 and opp_pts[dest_lower] == 1:

    if not possible_moves:

                # Check higher die move

                    for j in range(24):

                            dest_lower = j - lower_die

                                possible_moves.append((f"H:A{i},A{j}", 1))  # Priority 1 for moving

                    # Check if we can also move with higher die

    if possible_moves:

        possible_moves.sort(key=lambda x: (-x[1], x[0]))

        return "H:P,P"
