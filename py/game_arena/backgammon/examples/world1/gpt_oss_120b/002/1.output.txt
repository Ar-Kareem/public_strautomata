**Strategy Overview**

The policy builds the complete set of legal moves for the current roll and simply picks the first legal sequence it finds.  
The move generation follows the official Backgammon rules:

* **Bar priority** – if any of our checkers are on the bar they must be entered first.  
* **Destination legality** – we may land on an empty point, on a point with a single opponent checker (hit), or on a point with two or more of our own checkers. Landing on a point with two or more opponent checkers is illegal.  
* **Bearing‑off** – allowed only when every own checker is inside the home board (points 0‑5). A checker may bear off exactly (`die == distance`) or, if the die is larger, only from the highest occupied point.  
* **Dice usage** – when both dice can be played we must play both; when only one die can be played we must use the higher die if possible.  
* **Pass** – a pass (`P`) is used only when no legal move exists for that die.

The algorithm works as follows:

1. **Enumerate sources** (`B` for the bar, `A{i}` for point *i*) that are legal for a given die in the current board state.  
2. For a roll of one die we either make the first legal move or pass.  
3. For a roll of two dice we try the two possible orders (`H` = high‑first, `L` = low‑first).  
   * For the chosen order we pick a legal source for the first die, apply the move to obtain a temporary state, then pick a legal source for the second die.  
   * The first complete pair that is legal is returned.  
4. The result is formatted as required: `<ORDER>:<FROM1>,<FROM2>`.

Because the move generator strictly respects all Backgammon constraints, the returned move is always legal, and the simple “first‑fit” heuristic works reasonably well against a variety of opponents.

import copy

def policy(state: dict) -> str:
    """
    Return a legal move string for the given Backgammon state.
    The move is generated by exhaustively trying legal moves for the
    current dice roll and picking the first successful sequence.
    """

    my_pts = state['my_pts']
    opp_pts = state['opp_pts']
    my_bar = state['my_bar']
    opp_bar = state['opp_bar']
    dice = state['dice'][:]   # list of 0, 1 or 2 ints (1..6)

    # ------------------------------------------------------------------
    # Helper utilities
    # ------------------------------------------------------------------

    def all_in_home(my_pts):
        """True when all my checkers are on points 0..5 (the home board)."""
        return sum(my_pts[6:]) == 0

    def distance_from_off(i):
        """Number of points a checker on index i must travel to bear off."""
        return i + 1  # point 0 needs distance 1, point 5 needs distance 6

    def can_enter_from_bar(die, my_bar, opp_pts):
        """Check whether a bar entry with the given die is legal."""
        if my_bar == 0:
            return False
        dest = 24 - die          # entry point for us
        return opp_pts[dest] <= 1

    def can_move_from(i, die, my_pts, opp_pts, my_bar, opp_bar):
        """Return True if a checker on point i may move with the given die."""
        if my_pts[i] == 0:
            return False
        dest = i - die
        # Bearing off case
        if dest < 0:
            if not all_in_home(my_pts):
                return False
            dist = distance_from_off(i)
            if die == dist:
                return True
            # die larger than needed: allowed only if no higher checker
            if die > dist:
                for j in range(i + 1, 6):
                    if my_pts[j] > 0:
                        return False
                return True
            return False
        # Normal move inside the board
        if opp_pts[dest] >= 2:
            return False
        return True

    def legal_sources(current_state, die):
        """Return a list of legal source identifiers for `die`."""
        srcs = []
        my_pts = current_state['my_pts']
        opp_pts = current_state['opp_pts']
        my_bar = current_state['my_bar']

        # Bar must be entered first if any checker is on the bar
        if my_bar > 0:
            if can_enter_from_bar(die, my_bar, opp_pts):
                srcs.append('B')
            else:
                # No legal entry – we cannot move this die at all
                srcs.append('P')
            return srcs

        # Normal points
        for i in range(24):
            if can_move_from(i, die, my_pts, opp_pts,
                             current_state['my_bar'],
                             current_state['opp_bar']):
                srcs.append(f'A{i}')
        if not srcs:
            srcs.append('P')
        return srcs

    def apply_move(current_state, src, die):
        """Apply a move and return the new state (or None if illegal)."""
        # Work on copies to avoid mutating the original
        new_state = {
            'my_pts': current_state['my_pts'][:],
            'opp_pts': current_state['opp_pts'][:],
            'my_bar': current_state['my_bar'],
            'opp_bar': current_state['opp_bar'],
            'my_off': current_state['my_off'],
            'opp_off': current_state['opp_off'],
            'dice': current_state['dice'][:]
        }

        if src == 'P':
            # Pass – nothing changes
            return new_state

        if src == 'B':
            # Enter from the bar
            if new_state['my_bar'] == 0:
                return None
            dest = 24 - die
            if new_state['opp_pts'][dest] >= 2:
                return None
            # Remove from bar
            new_state['my_bar'] -= 1
            # Hit?
            if new_state['opp_pts'][dest] == 1:
                new_state['opp_pts'][dest] = 0
                new_state['opp_bar'] += 1
            else:
                new_state['opp_pts'][dest] = 0
            # Place our checker
            new_state['my_pts'][dest] += 1
            return new_state

        # Normal point move
        if not src.startswith('A'):
            return None
        i = int(src[1:])
        if new_state['my_pts'][i] == 0:
            return None
        dest = i - die
        # Bearing off
        if dest < 0:
            if not all_in_home(new_state['my_pts']):
                return None
            dist = distance_from_off(i)
            if die == dist:
                # Exact bear off
                new_state['my_pts'][i] -= 1
                new_state['my_off'] += 1
                return new_state
            elif die > dist:
                # Can bear off only if no higher checkers
                for j in range(i + 1, 6):
                    if new_state['my_pts'][j] > 0:
                        return None
                new_state['my_pts'][i] -= 1
                new_state['my_off'] += 1
                return new_state
            else:
                return None
        # Normal board move
        if new_state['opp_pts'][dest] >= 2:
            return None
        # Move the checker
        new_state['my_pts'][i] -= 1
        # Hit?
        if new_state['opp_pts'][dest] == 1:
            new_state['opp_pts'][dest] = 0
            new_state['opp_bar'] += 1
        new_state['my_pts'][dest] += 1
        return new_state

    # ------------------------------------------------------------------
    # Main move selection
    # ------------------------------------------------------------------

    # No dice – just pass
    if not dice:
        return "H:P,P"

    # Single die case
    if len(dice) == 1:
        die = dice[0]
        srcs = legal_sources(state, die)
        src = srcs[0]  # first legal source (or 'P')
        return f"H:{src},P"

    # Two dice case
    die1, die2 = dice[0], dice[1]
    high, low = max(die1, die2), min(die1, die2)

    # Try high‑first (order H) then low‑first (order L)
    for order_char, first_die, second_die in (
        ('H', high, low),
        ('L', low, high)
    ):
        first_sources = legal_sources(state, first_die)
        for src1 in first_sources:
            # Apply first move
            after_first = apply_move(state, src1, first_die)
            if after_first is None:
                continue
            second_sources = legal_sources(after_first, second_die)
            for src2 in second_sources:
                after_second = apply_move(after_first, src2, second_die)
                if after_second is None:
                    continue
                # Both moves succeeded – we have a legal complete move
                return f"{order_char}:{src1},{src2}"

    # If we reach this point, no legal two‑move sequence was found.
    # Fall back to playing a single die (the higher one) if possible.
    # This situation can happen when only one die is usable.
    srcs = legal_sources(state, high)
    src = srcs[0]
    return f"H:{src},P"
