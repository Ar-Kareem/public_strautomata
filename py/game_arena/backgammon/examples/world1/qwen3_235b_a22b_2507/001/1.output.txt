The strategy for this policy is to prioritize re-entering checkers from the bar as quickly as possible, then focus on building and advancing prime structures while minimizing exposure to hits. The policy uses a hierarchical decision-making approach:

1. If checkers are on the bar, attempt to re-enter using either die, prioritizing the lower die if both can be used (to leave the higher die for deeper entry).
2. If all checkers are in home board and bearing off is possible, make bearing off moves with the highest feasible points first, while using the "pip efficiency" rule (i.e., use larger die to bear off from farther points).
3. Otherwise, prioritize:
   - Hitting blots (opponent singletons) if safe.
   - Advancing rear checkers (stragglers) to reduce lag.
   - Building points in the home board and forming primes.
   - Avoiding leaving blots unless absolutely necessary.
4. The policy respects die order constraints (higher die must be played first if only one can be played) by evaluating both H and L sequences and selecting a valid, highest-priority move.

It computes a set of candidate legal moves (simulated via helper logic), scores them based on strategic value, and returns the best legal move string.

def policy(state: dict) -> str:

    opp_pts = state['opp_pts']
    my_bar = state['my_bar']
    opp_bar = state['opp_bar']
    my_off = state['my_off']
    opp_off = state['opp_off']
    dice = sorted(state['dice'], reverse=True)
    if len(dice) == 0:
        return "H:P,P"
    if len(dice) == 1:
        high_die, low_die = dice[0], dice[0]
    else:
        high_die, low_die = dice[0], dice[1]

    all_in_home = sum(my_pts[6:]) == 0

    def is_enter_valid(die: int) -> bool:
        idx = 24 - die  # entering from bar to point 24-die (0-based)
        if idx < 0 or idx >= 24:
            return False
        return opp_pts[idx] <= 1

    def is_move_valid(src: int, die: int) -> bool:
        dst = src - die
        if dst < 0:
            # Borne off
            return all_in_home and src < 6
        if dst >= 24:
            return False
        return opp_pts[dst] <= 1

            return False

        for i in range(5, -1, -1):
            if my_pts[i] > 0:
                if i < die:
                    return True  # Can bear off from point i
                elif i == die - 1:
                    return True
                else:
                    # Check if no checkers on points higher than die-1
                    if all(my_pts[j] == 0 for j in range(die - 1, 6)):
                        return True
                    return False
        return False

                    actions.append(("H", "B", "P"))

                    # But after first entry, still might be on bar
