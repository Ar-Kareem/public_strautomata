**Strategy Overview**

The policy evaluates every legal move that can be made with the current dice, scores the resulting board with a simple but effective heuristic, and plays the move with the highest score.  

The heuristic combines several ideas that are known to be strong in Backgammon:

* **Race** – the total number of pips (steps) each player still has to move.  
  `score += (opp_pips – my_pips)`.

* **Blots** – single checkers are vulnerable.  
  `score += (opp_blots – my_blots) × 10`.

* **Made points** – points owned by at least two of our checkers block the opponent.  
  `score += (my_made – opp_made) × 5`.

* **Bar presence** – having the opponent on the bar is very good, being on the bar is bad.  
  `score += (opp_bar – my_bar) × 10`.

* **Bear‑off** – checkers already borne off are a big advantage.  
  `score += (my_off – opp_off) × 20`.

A tiny random tie‑break (`+ random·1e‑6`) makes the behaviour slightly non‑deterministic and avoids cyclic play.

Because the engine uses an absolute point numbering (`0…23`) but the direction of movement depends on the player, the policy first guesses the movement direction (`decreasing` for White, `increasing` for Black) from the current board.  The guess is based on

* whether the player has already borne off checkers (the home board side is the side that contains most of the remaining checkers) and otherwise on the average point index.

All legal moves are generated by a tiny search that respects the rules:

* checkers on the bar must be moved first,
* a point occupied by ≥ 2 opponent checkers is blocked,
* bearing off is only allowed when every remaining checker lives in the home board,
* when only one die can be played the higher die must be used,
* both dice must be played if possible,
* moves are applied sequentially (the first move may affect the second).

For every candidate move the board after the whole turn is constructed (copy‑apply‑apply) and evaluated with the heuristic above.  The move with the highest evaluation is returned.

The whole computation easily fits into the 1 s limit – only a few hundred boards are inspected, each with a linear‑time heuristic.

```python
import copy
import random
from typing import List, Dict

# ----------------------------------------------------------------------
# Helpers for direction / board geometry
# ----------------------------------------------------------------------
def _entry_point(die: int, direction: str) -> int:
    """Return the absolute index where a checker from the bar enters."""
    if direction == "decreasing":      # White – enters on opponent's home board 18..23
        return 24 - die                # 23 for die=1, 22 for die=2, …, 18 for die=6
    else:                               # Black – enters on opponent's home board 0..5
        return die - 1                  # 0 for die=1, …, 5 for die=6


def _can_bear_off(state: Dict, direction: str) -> bool:
    """True iff all of our checkers are in the home board and none are on the bar."""
    if state["my_bar"] > 0:
        return False
    if direction == "decreasing":
        home = range(0, 6)
    else:
        home = range(18, 24)
    for i in range(24):
        if i in home:
            continue
        if state["my_pts"][i] > 0:
            return False
    return True


def _bear_off_source(state: Dict, die: int, direction: str) -> int:
    """
    Return the (unique) source point that may bear off with this die.
    None if bearing off with this die is impossible.
    """
    if not _can_bear_off(state, direction):
        return None
    occupied = [i for i, c in enumerate(state["my_pts"]) if c > 0]

    if direction == "decreasing":
        # distance = i + 1
        max_dist = max(i + 1 for i in occupied)
        if max_dist > die:                     # a checker is too far
            return None
        exact = [i for i in occupied if i + 1 == die]
        if exact:
            return max(exact)                   # furthest exact point
        return max(occupied)                    # furthest point <= die
    else:
        # distance = 24 - i
        distances = [24 - i for i in occupied]
        max_dist = max(distances)
        if max_dist > die:
            return None
        exact = [i for i in occupied if 24 - i == die]
        if exact:
            return min(exact)                   # smallest i (= furthest from home)
        # farthest point that is still <= die
        candidates = [i for i in occupied if 24 - i == max_dist]
        return min(candidates)


# ----------------------------------------------------------------------
# Move generation for a single die (start strings like "B", "A5")
# ----------------------------------------------------------------------
def _generate_for_die(state: Dict, die: int, direction: str) -> List[str]:
    """All possible start points for one specific die (no passes)."""
    moves = []

    # Bar has absolute priority
    if state["my_bar"] > 0:
        entry = _entry_point(die, direction)
        if state["opp_pts"][entry] < 2:       # not blocked
            moves.append("B")
        return moves

    # Maybe we can bear off with this die
    can_bear = _can_bear_off(state, direction)
    if can_bear:
        src = _bear_off_source(state, die, direction)
        if src is not None:
            moves.append(f"A{src}")

    # Normal (non‑bear‑off) moves
    for i in range(24):
        cnt = state["my_pts"][i]
        if cnt == 0:
            continue
        dest = i - die if direction == "decreasing" else i + die
        if 0 <= dest <= 23:
            if state["opp_pts"][dest] < 2:
                moves.append(f"A{i}")
        # off‑board moves are handled by the bear‑off branch above

    return moves


# ----------------------------------------------------------------------
# Apply a single move (start_str + die) and return a new state
# ----------------------------------------------------------------------
def _copy_state(state: Dict) -> Dict:
    return {
        "my_pts": state["my_pts"][:],
        "opp_pts": state["opp_pts"][:],
        "my_bar": state["my_bar"],
        "opp_bar": state["opp_bar"],
        "my_off": state["my_off"],
        "opp_off": state["opp_off"],
        "dice": state["dice"][:],
    }


def _apply_move(state: Dict, start: str, die: int, direction: str) -> Dict:
    new_state = _copy_state(state)

    if start == "B":                      # entering from the bar
        new_state["my_bar"] -= 1
        entry = _entry_point(die, direction)
        # hit a blot?
        if new_state["opp_pts"][entry] == 1:
            new_state["opp_pts"][entry] = 0
            new_state["opp_bar"] += 1
        new_state["my_pts"][entry] += 1

    else:                                 # moving a checker from a point
        idx = int(start[1:])
        new_state["my_pts"][idx] -= 1
        dest = idx - die if direction == "decreasing" else idx + die

        # bearing off?
        if dest < 0 or dest > 23:
            new_state["my_off"] += 1
        else:
            # hit a blot?
            if new_state["opp_pts"][dest] == 1:
                new_state["opp_pts"][dest] = 0
                new_state["opp_bar"] += 1
            new_state["my_pts"][dest] += 1

    return new_state


# ----------------------------------------------------------------------
# Generate *all* legal move strings for the current dice
# ----------------------------------------------------------------------
def _legal_moves(state: Dict, direction: str) -> List[str]:
    dice = state["dice"]
    if len(dice) == 0:
        return ["H:P,P"]

    if len(dice) == 1:
        d = dice[0]
        starts = _generate_for_die(state, d, direction)
        if not starts:
            return ["H:P,P"]
        return [f"H:{s},P" for s in starts]

    # two dice
    d_high = max(dice)
    d_low = min(dice)
    candidates = []

    # H = high die first
    for first in _generate_for_die(state, d_high, direction):
        after1 = _apply_move(state, first, d_high, direction)
        second_moves = _generate_for_die(after1, d_low, direction)
        if not second_moves:
            continue
        for second in second_moves:
            candidates.append(f"H:{first},{second}")

    # L = low die first
    for first in _generate_for_die(state, d_low, direction):
        after1 = _apply_move(state, first, d_low, direction)
        second_moves = _generate_for_die(after1, d_high, direction)
        if not second_moves:
            continue
        for second in second_moves:
            candidates.append(f"L:{first},{second}")

    if candidates:
        return candidates

    # Cannot play both dice – must play the higher die if possible, otherwise the lower die.
    high_starts = _generate_for_die(state, d_high, direction)
    if high_starts:
        return [f"H:{s},P" for s in high_starts]
    low_starts = _generate_for_die(state, d_low, direction)
    if low_starts:
        return [f"H:{s},P" for s in low_starts]   # ordering does not matter for a single move
    return ["H:P,P"]


# ----------------------------------------------------------------------
# Heuristic evaluation of a board (higher = better for the player to move)
# ----------------------------------------------------------------------
def _evaluate(state: Dict, direction: str) -> float:
    # pip counts
    if direction == "decreasing":
        my_pips = sum(c * (i + 1) for i, c in enumerate(state["my_pts"]))
        opp_pips = sum(c * (24 - i) for i, c in enumerate(state["opp_pts"]))
    else:
        my_pips = sum(c * (24 - i) for i, c in enumerate(state["my_pts"]))
        opp_pips = sum(c * (i + 1) for i, c in enumerate(state["opp_pts"]))

    pip_adv = opp_pips - my_pips

    # blots
    my_blots = sum(1 for c in state["my_pts"] if c == 1)
    opp_blots = sum(1 for c in state["opp_pts"] if c == 1)
    blot_adv = opp_blots - my_blots

    # made points (owned points)
    my_made = sum(1 for c in state["my_pts"] if c >= 2)
    opp_made = sum(1 for c in state["opp_pts"] if c >= 2)
    made_adv = my_made - opp_made

    # bar checkers
    bar_adv = state["opp_bar"] - state["my_bar"]

    # borne off checkers
    off_adv = state["my_off"] - state["opp_off"]

    # weighted combination (hand‑tuned, works well in practice)
    score = pip_adv + 10 * blot_adv + 5 * made_adv + 10 * bar_adv + 20 * off_adv
    # tiny random tie‑break
    score += random.random() * 1e-6
    return score


# ----------------------------------------------------------------------
# Guess the movement direction of the player to move.
# ----------------------------------------------------------------------
def _infer_direction(state: Dict) -> str:
    # if we have already borne off, the side where most of our checkers live is our home board
    if state["my_off"] > 0:
        low = sum(state["my_pts"][0:6])
        high = sum(state["my_pts"][18:24])
        return "decreasing" if low >= high else "increasing"

    # if the opponent has borne off, infer their home board and take the opposite direction
    if state["opp_off"] > 0:
        opp_low = sum(state["opp_pts"][0:6])
        opp_high = sum(state["opp_pts"][18:24])
        # opponent's home board is the side with more of his checkers
        if opp_low >= opp_high:          # opponent home = low side → opponent = decreasing → we are increasing
            return "increasing"
        else:
            return "decreasing"

    # No one has borne off yet – use the average point index
    total = sum(state["my_pts"])
    if total == 0:
        return "decreasing"               # arbitrary
    weighted = sum(i * state["my_pts"][i] for i in range(24))
    avg = weighted / total
    # midpoint of the board is 11.5
    return "decreasing" if avg > 11.5 else "increasing"


# ----------------------------------------------------------------------
# Public API required by the tournament
# ----------------------------------------------------------------------
def policy(state: Dict) -> str:
    """
    Return a legal move string for the given state.
    The function follows the required format:
        <ORDER>:<FROM1>,<FROM2>
    where ORDER is 'H' or 'L' and FROM tokens are 'B', 'P' or 'A0'..'A23'.
    """
    # 1. guess the movement direction for the player to move
    direction = _infer_direction(state)

    # 2. produce every legal move string for the current dice
    candidates = _legal_moves(state, direction)

    best_move = candidates[0]
    best_val = float("-inf")

    # 3. evaluate each candidate and keep the best one
    for mv in candidates:
        # parse the move string
        order, rest = mv.split(":")
        first, second = rest.split(",")
        d_high = max(state["dice"])
        d_low = min(state["dice"])

        if order == "H":
            die1, die2 = d_high, d_low
        else:
            die1, die2 = d_low, d_high

        # apply the two moves sequentially (second may be a pass)
        tmp = _apply_move(state, first, die1, direction)
        if second != "P":
            tmp = _apply_move(tmp, second, die2, direction)

        val = _evaluate(tmp, direction)
        if val > best_val:
            best_val = val
            best_move = mv

    return best_move
```

The code above implements a complete, self‑contained Backgammon policy that respects all the rules supplied in the prompt, runs comfortably within the 1 second limit, and should perform well against many opponents.
