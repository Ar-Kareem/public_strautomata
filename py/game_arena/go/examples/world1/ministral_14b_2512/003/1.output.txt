The strategy for this policy is a **hybrid approach** combining **territory control**, **influence-based expansion**, and **basic pattern recognition** to create a competitive Go AI. Here's the breakdown:

1. **Territory Control**:
   - The policy prioritizes securing territory by filling large empty spaces near existing stones.
   - It evaluates potential moves based on the size of the surrounding empty area (liberties) and how well they connect existing groups.

2. **Influence-Based Expansion**:
   - If no clear territory is available, it expands influence by placing stones in high-value areas (e.g., near opponent stones or at board centers).
   - Uses a **simplified influence map** (weighted by distance to opponent stones) to guide expansion.

3. **Pattern Recognition**:
   - Detects simple **ko threats** and **basic shapes** (e.g., open 3-4 points, enclosures).
   - Avoids placing stones in **atari** (capture threats) unless necessary.

4. **Defensive Play**:
   - Responds to opponent threats by filling liberties or capturing if safe.
   - Uses **basic life-and-death heuristics** to avoid suicide.

5. **Opening Strategy**:
   - Follows a **predefined opening book** for the first few moves (e.g., 3-4, 4-4, 12-12).
   - Falls back to influence-based play if the opening book is exhausted.

6. **Endgame Tactics**:
   - Attempts to **reduce opponent's liberties** or **fill own liberties** when the board is mostly filled.
   - Uses **passing** strategically when no good move is found.

7. **Legal Move Enforcement**:
   - Always returns a legal move (no ko violations, no placing on opponent stones, etc.).
   - Falls back to **passing** if no valid move is found.

import numpy as np
from typing import List, Tuple

def policy(me: List[Tuple[int, int]], opponent: List[Tuple[int, int]]) -> Tuple[int, int]:
    # Convert to numpy arrays for easier manipulation (1-based to 0-based)
    my_stones = np.array(me)
    opp_stones = np.array(opponent)

    # Initialize board (19x19)
    board = np.zeros((19, 19), dtype=int)
    board[my_stones[:, 0] - 1, my_stones[:, 1] - 1] = 1
    board[opp_stones[:, 0] - 1, opp_stones[:, 1] - 1] = 2

def opening_book(total_moves: int) -> Tuple[int, int]:

def check_defensive(board: np.ndarray, my_stones: np.ndarray, opp_stones: np.ndarray) -> Tuple[int, int]:

def count_liberties(board: np.ndarray, row: int, col: int) -> int:
    # Count liberties for a stone at (row, col)
    liberties = 0
    for dr in [-1, 0, 1]:
        for dc in [-1, 0, 1]:

            nr, nc = row + dr, col + dc
            if 0 <= nr < 19 and 0 <= nc < 19 and board[nr, nc] == 0:
                liberties += 1
    return liberties

def is_ko(board: np.ndarray, move: Tuple[int, int]) -> bool:
    # Simplified ko check: avoid placing near opponent's single stone if it's a ko
    row, col = move
    row, col = row - 1, col - 1  # Convert to 0-based
    if board[row, col] != 0:
        return True  # Illegal move (not empty)

    for dr in [-1, 0, 1]:
        for dc in [-1, 0, 1]:

def evaluate_territory(board: np.ndarray, row: int, col: int) -> float:
    # Simplified territory evaluation: count empty adjacent points
    count = 0
    for dr in [-1, 0, 1]:
        for dc in [-1, 0, 1]:
            if dr == 0 and dc == 0:
                continue
            nr, nc = row + dr, col + dc
            if 0 <= nr < 19 and 0 <= nc < 19 and board[nr, nc] == 0:
                count += 1
    return count * 0.5  # Weighted score

def evaluate_influence(board: np.ndarray, row: int, col: int, opp_stones: np.ndarray) -> float:
    # Influence score: distance to opponent stones (higher if far)
