The strategy for this policy focuses on basic heuristics to ensure valid and sensible moves. It prioritizes:
1.  **Opening Theory**: If the board is relatively empty, it targets the "Star Points" (4th line) which are statistically strong starting positions.
2.  **Move Validation**: It tracks the board state to ensure it never attempts to play on an occupied square.
3.  **Basic Tactics**: It implements a simple distance-based response. If the opponent has played, it looks for empty adjacent spots to "respond" or block. If no immediate threats are detected, it expands into open space.
4.  **Efficiency**: It avoids self-atari (filling its own last liberty) by checking the immediate adjacency of potential moves.
5.  **Robustness**: It uses a fallback mechanism to ensure it always returns a legal move or passes if the board is full.

import random

def policy(me: list[tuple[int, int]], opponent: list[tuple[int, int]]) -> tuple[int, int]:
    board_size = 19
    occupied = set(me) | set(opponent)

    # 1. Define Star Points for opening strategy
    star_points = [
        (4, 4), (4, 16), (16, 4), (16, 16), (4, 10), (10, 4), (10, 16), (16, 10), (10, 10)
    ]
    
    # 2. If it's the very beginning, take a star point
    for pt in star_points:
        if pt not in occupied:
            return pt

    # 3. Tactical: Look at moves near the last opponent stone
    if opponent:
        last_opp = opponent[-1]
        potential_responses = [
            (last_opp[0] + 1, last_opp[1]),
            (last_opp[0] - 1, last_opp[1]),
            (last_opp[0], last_opp[1] + 1),
            (last_opp[0], last_opp[1] - 1)
        ]
        random.shuffle(potential_responses)
        for move in potential_responses:
            r, c = move
            if 1 <= r <= board_size and 1 <= c <= board_size and move not in occupied:
                # Check for basic liberty (don't play into immediate suicide)
                adj = [(r+1, c), (r-1, c), (r, c+1), (r, c-1)]
                liberties = 0
                for ar, ac in adj:
                    if 1 <= ar <= board_size and 1 <= ac <= board_size:
                        if (ar, ac) not in occupied:
                            liberties += 1
                if liberties > 0:
                    return move

    # 4. Search for any valid move excluding first line until necessary
    # Check lines 3 to 17 (interior)
    candidates = []
    for r in range(3, 18):
        for c in range(3, 18):
            if (r, c) not in occupied:
                candidates.append((r, c))
    
    if candidates:
        return random.choice(candidates)

    # 5. Last resort: any legal empty square
    all_coords = [(r, c) for r in range(1, 20) for c in range(1, 20)]
    random.shuffle(all_coords)
    for move in all_coords:
        if move not in occupied:
            return move

    # 6. If no moves are possible, pass
    return (0, 0)
