The strategy for this policy is to leverage the standard Nim game theory where the optimal move is determined by the XOR (nim-sum) of all pile sizes. When the nim-sum is non-zero, the player can force a win by making the nim-sum zero after their move. If the nim-sum is zero, we are in a losing position assuming optimal play from the opponent, so the player chooses moves that minimize the opponent's chances. The exception to this is when all piles are size 1 and there is an even number of piles, as this allows the player to force a win by taking one stone. Otherwise, the code selects a conservative move to take one stone from the first non-zero pile.

def policy(piles: list[int]) -> str:
    xor_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    
    if xor_sum != 0:
        # Standard Nim strategy to make nim-sum zero
        for i in range(4):
            pile_size = piles[i]
            take = pile_size ^ xor_sum
            if take < pile_size:
                return f"{i},{pile_size - take}"
    
    else:
        # Handle all-ones case with even number of piles
        if all(p == 1 for p in piles):
            return "0,1"
        # Otherwise, take 1 from the first non-zero pile
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    
    # This line is unreachable for valid inputs
    return "0,1"
