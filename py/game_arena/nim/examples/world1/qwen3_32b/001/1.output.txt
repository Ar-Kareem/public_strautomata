The logic for this policy is based on the standard Nim game strategy using XOR (nim-sum) with special handling for endgame scenarios in misère play. The policy follows these rules:

1. **Single Non-Zero Pile Handling**: If only one pile has stones, take all but one to force the opponent to take the last object.
2. **Nim-Sum Strategy**: For most positions, compute the XOR of all piles. If non-zero, find a move to zero the nim-sum. If zero (losing position), make any valid move.
3. **Fallback for Zero Nim-Sum**: If the nim-sum is zero and not in the single-pile case, choose the first non-zero pile and take one stone.

This approach ensures optimal play for most positions while handling edge cases like single-pile scenarios where standard nim-sum fails due to the misère rule.

def policy(piles: list[int]) -> str:
    non_zero = [i for i, p in enumerate(piles) if p > 0]
    if len(non_zero) == 1:
        pile = non_zero[0]
        return f"{pile},{piles[pile] - 1}"
    
    nim_sum = 0
    for p in piles:
        nim_sum ^= p
    
    if nim_sum == 0:
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    else:
        for i in range(4):
            if piles[i] > 0:
                desired = piles[i] ^ nim_sum
                take = piles[i] - desired
                if take > 0 and take <= piles[i]:
                    return f"{i},{take}"
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    
    # Fallback (should never be reached with valid input)
    for i in range(4):
        if piles[i] > 0:
            return f"{i},1"
