The strategy for this policy is to leverage standard Nim theory with an adaptation for the unique losing condition where taking the last object causes a loss. The core idea is to play optimal normal Nim strategy whenever possible and handle all-1s scenarios explicitly. Here's the logic:

1. **Compute XOR (Nim-Sum)** of all piles.
2. **If Nim-Sum is non-zero**: Make a move to zero it, following the standard optimal Nim strategy.
3. **If Nim-Sum is zero**:
   - **When all piles are 1s** (e.g., `[1, 1, 1, 1]`): Take one object, reducing one pile to 0. The opponent is forced to play and will eventually lose if this is the only remaining move.
   - **Otherwise**: Find the first pile with more than 1 object and reduce it to 1. This forces the game into a state where all piles will eventually be forced to 1 via the opponent's optimal moves. This increases the likelihood of the opponent being forced to take the last object.

This policy is designed to ensure that the AI can systematically outmaneuver opponents in most scenarios by minimizing their options and maximizing the pressure for the opponent to take the final object.

def policy(piles: list[int]) -> str:
    xor_sum = 0
    for pile in piles:
        xor_sum ^= pile

    if xor_sum != 0:
        for i in range(4):
            p = piles[i]
            desired = p ^ xor_sum
            if desired < p:
                take = p - desired
                return f"{i},{take}"

    all_ones = all(p == 1 for p in piles)
    if all_ones:
        return "0,1"  # All piles are 1s; even count allows you to win
    else:
        for i in range(4):
            if piles[i] > 1:
                take = piles[i] - 1
                return f"{i},{take}"
        for i in range(4):
            if piles[i] == 1:
                return f"{i},1"

    # Fallback for cases not handled (should never reach here)
    for i in range(4):
        if piles[i] > 0:
            return f"{i},1"
    return "0,1"
