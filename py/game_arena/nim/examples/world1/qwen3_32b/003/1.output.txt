The logic for this policy is based on the standard Nim game strategy with adjustments for the misère variant (where taking the last object loses). The policy follows these steps:

1. **Handle All-Ones Case**: If all piles contain 1 object, the outcome depends on the parity of the total count. An even count allows the current player to force a win by leaving an odd number of piles for the opponent.

2. **XOR Calculation for General Cases**: For non-all-ones configurations, compute the XOR (nim-sum) of all piles. 
   - If the XOR is non-zero, find a move that results in a zero XOR (optimal in standard Nim and misère when at least one pile >1).
   - If the XOR is zero, any valid move is made (since the position is losing under optimal play).

3. **Fallback Moves**: Always return a legal move, defaulting to removing 1 object from the first non-empty pile when necessary.

This approach ensures the policy is optimal except for forced-losing positions in misère Nim, where it follows the best possible move under the rules.

def policy(piles: list[int]) -> str:
    # Check if all piles have exactly 1 object
    all_ones = all(p == 1 for p in piles)
    if all_ones:
        # If even number of 1s, take one from first pile (winning move)
        return "0,1"
    
    # Compute nim-sum (XOR) of all piles
    xor_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    
    if xor_sum != 0:
        # Find a move to make the new XOR zero
        for pile in range(4):
            if piles[pile] > 0:
                desired = piles[pile] ^ xor_sum
                if desired < piles[pile]:
                    take = piles[pile] - desired
                    return f"{pile},{take}"
    else:
        # XOR is zero and not all ones: make any valid move
        for pile in range(4):
            if piles[pile] > 0:
                return f"{pile},1"
    
    # Fallback: always return a valid move
    for pile in range(4):
        if piles[pile] > 0:
            return f"{pile},1"
