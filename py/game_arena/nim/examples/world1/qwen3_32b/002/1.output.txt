The logic for this policy is to apply the standard Nim game strategy with adjustments for the misère rule where the player who takes the last object loses. The key steps are:

1. **Check for All Ones**: If all piles are size 1, return a move to reduce one pile to 0 (since the opponent will then be forced into a losing position).
2. **Nim Sum Calculation**: Compute the XOR (nim-sum) of all pile sizes.
3. **Non-Zero Nim Sum**: If the nim-sum is not zero, find a move that reduces the nim-sum to zero by adjusting a pile.
4. **Zero Nim Sum**: If the nim-sum is zero, make a move to disrupt the opponent's strategy by taking 1 object from the first pile that has more than 1 object.

This approach ensures that we follow optimal play for standard Nim when possible, while handling edge cases in the misère variant.

def policy(piles: list[int]) -> str:
    if all(p == 1 for p in piles):
        return "0,1"  # Winning move when all piles are 1
    
    xor_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    
    if xor_sum != 0:
        # Find a pile to make the new XOR sum zero
        for i in range(4):
            current = piles[i]
            target = current ^ xor_sum
            if target < current:
                take = current - target
                return f"{i},{take}"
        # Fallback (should not reach here if logic is correct)
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    else:
        # Nim sum is zero, force a non-optimal configuration for opponent
        for i in range(4):
            if piles[i] > 1:
                return f"{i},1"
        # Fallback (should not reach here)
        return "0,1"
