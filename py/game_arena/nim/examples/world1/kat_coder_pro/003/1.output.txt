The logic for this policy is to implement the optimal strategy for the game of Nim, which is based on the XOR (nim-sum) of the pile sizes. In Nim, a position is losing if the XOR of all pile sizes is zero. The optimal strategy is to always move to a position where the XOR is zero, if possible. If the current XOR is already zero, any move will put the opponent in a winning position, so we make a safe move (remove one object from a non-empty pile).

The policy works as follows:
1. Calculate the XOR of all pile sizes.
2. If the XOR is zero, we are in a losing position, so we make a safe move (remove one object from the first non-empty pile).
3. If the XOR is non-zero, we try to find a pile where we can reduce its size to make the overall XOR zero. We do this by calculating `new_size = pile_size ^ xor_sum` for each pile and checking if `new_size < pile_size`. If found, we reduce that pile to `new_size`.
4. If no such pile is found (which shouldn't happen in a winning position), we fall back to a safe move.

from typing import List

def policy(piles: List[int]) -> str:
    # Calculate XOR of all piles
    xor_sum = 0
    for pile in piles:
        xor_sum ^= pile
    
    # If XOR is 0, we're in a losing position, make a safe move
    if xor_sum == 0:
        # Find the first non-empty pile and remove 1 object
        for i, pile in enumerate(piles):
            if pile > 0:
                return f"{i},1"
    
    # Try to find a pile where we can make the XOR zero
    for i, pile in enumerate(piles):
        if pile > 0:
            # Calculate what the new pile size should be
            new_size = pile ^ xor_sum
            # If the new size is smaller than the current size, we can make this move
            if new_size < pile:
                return f"{i},{pile - new_size}"
    
    # Fallback (should not happen in a winning position)
    # Find the first non-empty pile and remove 1 object
    for i, pile in enumerate(piles):
        if pile > 0:
            return f"{i},1"
