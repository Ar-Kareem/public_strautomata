 The strategy is to play perfect misère Nim. The key insight is that this is the misère version where the player taking the last object loses. The algorithm distinguishes between positions where all non-zero piles are size 1 (all-ones positions) and positions with at least one pile larger than 1.

For all-ones positions: leave an odd number of piles of size 1 for the opponent (even for us means opponent gets odd and loses).

For other positions: use the XOR (nim-sum) strategy. If the XOR is 0, it's a losing position—make any move. If XOR is non-zero, find a pile to reduce to make XOR 0, but with a special exception: if this move would leave only piles of size 1, ensure the remaining count of size-1 piles is odd (not even). This involves adjusting the target from 0 to 1 or vice versa when necessary.

def policy(piles: list[int]) -> str:
    n = len(piles)
    
    # Check if all non-zero piles are size 1
    non_zero = [p for p in piles if p > 0]
    all_ones = all(p == 1 for p in non_zero) if non_zero else True
    
    if all_ones:
        # Only 1s and 0s remain
        count_ones = len(non_zero)
        # Find a pile with 1 to take
        for i, p in enumerate(piles):
            if p == 1:
                # If count is even, we win by taking 1 (leaving odd)
                # If count is odd, we lose anyway, take 1
                return f"{i},1"
        # Should not reach here if non_zero exists
        return "0,0"
    
    # Calculate XOR (nim-sum)
    x = 0
    for p in piles:
        x ^= p
    
    if x == 0:
        # Losing position (P-position): make any legal move
        # Taking 1 from the first available non-zero pile
        for i, p in enumerate(piles):
            if p > 0:
                return f"{i},1"
        return "0,0"
    
    # Winning position (N-position): find the correct pile to reduce
    for i, s in enumerate(piles):
        target = s ^ x
        if target < s:
            # Check if we're transitioning to an all-ones position
            # (all other non-zero piles are size 1, and target is 0 or 1)
            others_all_small = all(piles[j] <= 1 for j in range(n) if j != i)
            
            if others_all_small and target <= 1:
                # Count how many 1s exist in other piles
                c = sum(1 for j, p in enumerate(piles) if j != i and p == 1)
                final_ones = c + (1 if target == 1 else 0)
                
                # We want odd number of 1s for opponent (losing for them)
                # If even, we need to flip target
                if final_ones % 2 == 0:
                    if target == 0 and s > 1:
                        # Instead of taking all, leave 1 behind
                        target = 1
                    elif target == 1:
                        # Take all instead of leaving 1
                        target = 0
            
            take = s - target
            return f"{i},{take}"
    
    # Fallback (should not reach here if x != 0)
    for i, p in enumerate(piles):
        if p > 0:
            return f"{i},1"
    return "0,0"
