**Strategy**

The game is misère Nim (the player who makes the last move loses).  
The optimal strategy is:

1. **All piles are size 1** (or empty).  
   Let `c` be the number of non‑empty piles.  
   * If `c` is odd the position is losing (P‑position); play any legal move.  
   * If `c` is even the position is winning (N‑position); remove a whole pile (take 1).

2. **At least one pile has size > 1.**  
   Compute the nim‑sum `x = p0 ⊕ p1 ⊕ p2 ⊕ p3`.  
   * If `x = 0` the position is a P‑position; play any legal move.  
   * If `x ≠ 0` the position is an N‑position.  
        * If there is **exactly one** pile larger than 1, say of size `B` with `k` other piles of size 1:
            * If `k` is even, reduce the big pile to size 1 (take `B‑1`).  
            * If `k` is odd, remove the big pile entirely (take `B`).  
        * Otherwise (two or more piles > 1), play the standard Nim move:  
          find a pile `i` with `target = x ⊕ piles[i] < piles[i]` and reduce pile `i` to `target` (take `piles[i]‑target`).  
          This makes the new nim‑sum zero and, because at least one pile > 1 remains, the resulting position is indeed a P‑position.

The algorithm always returns a legal move and runs in O(4) time.

**Python implementation**

<code>
def policy(piles: list[int]) -> str:
    """
    Return a legal move "pile,take" for misère Nim with 4 piles.
    """
    # Terminal (should not be called, but guard anyway)
    if max(piles) == 0:
        return "0,0"

    # Case: all non‑zero piles are of size 1
    if max(piles) == 1:
        cnt = sum(1 for p in piles if p > 0)   # number of 1's
        if cnt % 2 == 0:
            # Winning: remove any 1‑pile
            i = next(i for i, p in enumerate(piles) if p == 1)
            return f"{i},1"
        else:
            # Losing: any legal move
            i = next(i for i, p in enumerate(piles) if p > 0)
            return f"{i},1"

    # General case: at least one pile > 1
    nim_sum = 0
    for p in piles:
        nim_sum ^= p

    # P‑position (nim‑sum zero) – any move is legal
    if nim_sum == 0:
        i = next(i for i, p in enumerate(piles) if p > 0)
        return f"{i},1"

    # N‑position: find a winning move
    bigs = [i for i, p in enumerate(piles) if p > 1]

    # Exactly one big pile: special misère handling
    if len(bigs) == 1:
        big_idx = bigs[0]
        big_val = piles[big_idx]
        # count 1‑piles among the others
        ones_cnt = sum(1 for j, p in enumerate(piles) if j != big_idx and p == 1)
        if ones_cnt % 2 == 0:
            # Reduce the big pile to 1, leaving an odd number of 1's
            return f"{big_idx},{big_val - 1}"
        else:
            # Remove the big pile entirely, leaving the odd count of 1's
            return f"{big_idx},{big_val}"

    # Normal Nim move: make nim‑sum zero
    for i, p in enumerate(piles):
        if p == 0:
            continue
        target = nim_sum ^ p
        if target < p:
            return f"{i},{p - target}"

    # Fallback (should never be reached)
    i = next(i for i, p in enumerate(piles) if p > 0)
    return f"{i},1"
