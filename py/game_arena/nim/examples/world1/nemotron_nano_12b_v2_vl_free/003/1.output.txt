To solve the 4-pile Nim game where the player who takes the last object loses, we implement an optimal strategy based on XOR (Nim-sum) analysis. This approach follows the standard Nim strategy for most positions, with special handling for the endgame where all piles are size 1.

### Approach
1. **Compute Nim-Sum**: Calculate the XOR of all pile sizes. If the Nim-sum is non-zero, the current player can force a win by making a move that results in a zero Nim-sum.
2. **Handle Endgame Cases**: If all piles have a size of 1, the player should take 1 stone from the first pile, leaving an odd number of piles of size 1, forcing the opponent to take the last stone.
3. **No Winning Move**: If the Nim-sum is zero and not all piles are size 1, any move is a losing move, so the player takes 1 stone from the first non-zero pile to delay defeat.

### Solution Code
```python
def policy(piles):
    xor_total = 0
    for p in piles:
        xor_total ^= p
    
    all_ones = all(p == 1 for p in piles)
    
    if xor_total != 0:
        for i in range(4):
            target = piles[i] ^ xor_total
            if target < piles[i] and target != 0:
                return f"{i},{piles[i] - target}"
            elif target < piles[i]:
                if all([p == 0 for p in piles] and p != piles[i]):
                    continue
                return f"{i},{piles[i] - target}"
            if target < piles[i]:
                return f"{i},{piles[i] - target}"
    else:
        if all_ones:
            return "0,1"
        else:
            for i in range(4):
                if piles[i] > 0:
                    return f"{i},1"
    return "0,1"
```

### Explanation
1. **Nim-Sum Calculation**: The Nim-sum is calculated using XOR of all pile sizes. This helps in determining the optimal move in normal Nim play.
2. **Non-Zero Nim-Sum**: If the Nim-sum is non-zero, the player can make a move to set the Nim-sum to zero, typically leading to a win. The move is chosen to reduce the pile such that the new XOR of all piles is zero.
3. **Endgame Handling**: If the current position has all piles of size 1, the player takes 1 stone to leave an odd number of piles, ensuring the opponent must take the last stone.
4. **Fallback for Loss**: If the Nim-sum is zero and not all piles are size 1, the player makes a move to take 1 stone from the first non-zero pile, delaying the inevitable loss.

This approach ensures optimal play by leveraging Nim-sum analysis and handling special cases where the standard strategy would lead to an immediate loss.
