The strategy for this policy is to play **optimal misère Nim** for 4 piles.

Key idea:
- In normal Nim, the winning strategy is based on making the xor-sum of the piles equal to 0.
- But this game is **misère Nim**: the player who takes the last object **loses**.
- Misère Nim differs from normal Nim only in the endgame where all nonzero piles have size 1.

Optimal policy:
1. Compute the nonzero piles.
2. If all nonzero piles are size 1:
   - If the count of such piles is odd, remove 1 from any nonzero pile, leaving an even number for the opponent.
   - If the count is even, any move loses against perfect play, so still return a legal move.
3. Otherwise, play the standard Nim move:
   - Let `x` be the xor of all pile sizes.
   - If `x != 0`, reduce some pile `p` to `p ^ x`, which makes the xor 0.
4. Special misère adjustment when there is exactly one pile larger than 1:
   - If the number of size-1 piles is even, reduce the large pile to 1.
   - If the number of size-1 piles is odd, remove the large pile entirely to 0.
5. If no winning move exists, return the first legal move.

This is the standard perfect-play solution for misère Nim and is optimal.

<code>
def policy(piles: list[int]) -> str:
    # Ensure legality fallback
    def first_legal():
        for i, p in enumerate(piles):
            if p > 0:
                return f"{i},1"
        return "0,0"  # unreachable in valid nonterminal calls

    nonzero = [p for p in piles if p > 0]
    if not nonzero:
        return "0,0"  # terminal fallback, though arena should not call here

    ones = sum(1 for p in piles if p == 1)
    big_indices = [i for i, p in enumerate(piles) if p > 1]
    big_count = len(big_indices)

    # Misère endgame: all nonzero piles are 1
    if ones == len(nonzero):
        for i, p in enumerate(piles):
            if p > 0:
                return f"{i},1"

    # Special misère case: exactly one pile > 1
    if big_count == 1:
        i = big_indices[0]
        big = piles[i]
        # Leave odd number of 1-heaps to opponent
        target = 1 if ones % 2 == 0 else 0
        take = big - target
        if 1 <= take <= big:
            return f"{i},{take}"
        return first_legal()

    # Normal Nim move: make xor zero
    x = 0
    for p in piles:
        x ^= p

    if x != 0:
        for i, p in enumerate(piles):
            target = p ^ x
            if target < p:
                take = p - target
                if 1 <= take <= p:
                    return f"{i},{take}"

    # Losing position fallback: make any legal move
    return first_legal()
