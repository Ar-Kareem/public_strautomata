

The strategy for this policy is to use the standard Nim-sum approach for determining winning moves, but with special handling for endgame scenarios where only one pile has objects. In standard Nim (misère version where taking the last object loses), the optimal move is to leave the opponent with a Nim-sum of zero. However, when only one pile is non-empty, the optimal play is to leave exactly one object, forcing the opponent to take it and lose.

<code>
def policy(piles: list[int]) -> str:
    non_zero = [p for p in piles if p > 0]
    if len(non_zero) == 1:
        pile_size = non_zero[0]
        pile_index = piles.index(pile_size)
        if pile_size == 1:
            return f"{pile_index},1"
        else:
            return f"{pile_index},{pile_size - 1}"
    else:
        nim_sum = 0
        for p in piles:
            nim_sum ^= p
        if nim_sum != 0:
            for i in range(len(piles)):
                pile_size = piles[i]
                if pile_size == 0:
                    continue
                target = pile_size ^ nim_sum
                if target < pile_size:
                    take = pile_size - target
                    return f"{i},{take}"
        # If no winning move (nim_sum is zero), take 1 from the largest pile
        max_size = max(piles)
        for i in range(len(piles)):
            if piles[i] == max_size:
                return f"{i},1"
