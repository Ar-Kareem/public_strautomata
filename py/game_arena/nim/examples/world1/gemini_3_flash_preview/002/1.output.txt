The logic for this policy is based on the mathematical theory of **Nim under Misere Play**. In a standard Nim game, the goal is to take the last object. In a Misere Play game (which this is), the goal is to *avoid* taking the last object.

1.  **Calculate the Nim-Sum**: We compute the XOR sum of all pile sizes. 
2.  **Optimal Strategy (Normal Phase)**: If the Nim-Sum is non-zero, there exists a move that makes the Nim-Sum zero. This is usually the optimal move.
3.  **Misere Play Adjustment**: The strategy for Misere Nim differs from standard Nim only when a move would leave a state where **all piles are of size 1**. 
    *   If a move can result in an **odd** number of piles of size 1 and no piles of size > 1, that is the winning move.
    *   If we follow the "Nim-Sum = 0" rule blindly when only one pile has size > 1, we might accidentally leave an even number of size-1 piles, which is a losing state in Misere play.
4.  **Fallback**: If the Nim-Sum is already zero (a losing position), we make the smallest legal move possible (remove 1 from the first non-empty pile) to prolong the game and hope for an opponent's mistake.

<code>
def policy(piles: list[int]) -> str:
    # Calculate the XOR sum (Nim-sum) of the piles
    nim_sum = 0
    for p in piles:
        nim_sum ^= p

    # Count how many piles have more than 1 object
    piles_greater_than_1 = sum(1 for p in piles if p > 1)

    # Misere Play adjustment: 
    # If only one pile has more than 1 object, we want to leave an ODD number of 1s.
    if piles_greater_than_1 == 1:
        # Find the index of the pile with more than 1
        idx = next(i for i, p in enumerate(piles) if p > 1)
        # Count how many other piles have exactly 1
        count_ones = sum(1 for i, p in enumerate(piles) if i != idx and p == 1)
        
        # If count_ones is even, we leave 1 in this pile (total ones will be odd)
        # If count_ones is odd, we leave 0 in this pile (total ones will be odd)
        take = piles[idx] - 1 if count_ones % 2 == 0 else piles[idx]
        return f"{idx},{take}"

    # Standard Nim strategy: make the Nim-sum zero
    if nim_sum != 0:
        for i, p in enumerate(piles):
            target = p ^ nim_sum
            if target < p:
                take = p - target
                return f"{i},{take}"

    # If already in a losing position (nim_sum == 0), take 1 from the first available pile
    for i, p in enumerate(piles):
        if p > 0:
            return f"{i},1"

    # Default fallback (should not reach here if state is legal)
    return "0,1"
