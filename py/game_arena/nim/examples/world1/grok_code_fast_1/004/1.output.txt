The strategy for this policy is as follows: Nim is an impartial combinatorial game, and for the misère variant (where the player who takes the last object loses), the standard normal-play (who-takes-last-wins) XOR-based strategy remains optimal when the number of piles is greater than 3 (as here with 4 piles). If the XOR of all pile sizes is non-zero, compute and execute the move that reduces it to zero. If the XOR is zero (which happens only in endgame positions where no winning moves exist), take 1 object from the pile with the most objects remaining to force play. This policy guarantees victory against optimal opponents and defends against suboptimal ones by always making legal, calculated moves.

def policy(piles: list[int]) -> str:
    total_xor = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    if total_xor == 0:
        # Losing position in normal Nim: take 1 from the largest pile
        max_index = piles.index(max(piles))
        take = 1
        return f"{max_index},{take}"
    else:
        # Winning move: find a pile to adjust to make total_xor zero
        for i in range(4):
            s_i = total_xor ^ piles[i]
            take = piles[i] - s_i
            if 1 <= take <= piles[i]:
                return f"{i},{take}"
        # This should not happen in standard Nim theory, but as a safeguard
        return "0,1"  # Arbitrary legal move if needed
