### Strategy Explanation
This policy implements optimal play for misère Nim with 4 piles, where the player who takes the last object loses. The strategy uses a recursive approach with memoization (via `@functools.lru_cache`) to determine if the current position is a losing position for the player to move. 

- A position is considered losing if no legal move (that doesn't immediately leave all piles empty) leads to a position where the opponent is also in a losing position. 
- The policy prioritizes selecting a move that leaves the opponent in a losing position, ensuring optimal play.
- If no such winning move exists (i.e., all possible moves lead to non-losing positions), it defaults to a legal move (e.g., taking 1 from the first non-empty pile).
- Memoization caches results for states (represented as tuples of 4 integers), making it efficient for small pile sizes (assumed ≤20 based on typical Nim constraints and 1-second time limit).

This approach ensures perfect play and is sufficient to beat suboptimal policies like random or basic Nim-sum strategies.

import functools

@functools.lru_cache(maxsize=None)
def is_losing(piles_tuple: tuple[int, ...]) -> bool:
    piles = list(piles_tuple)
    if all(p == 0 for p in piles):
        return True
    for pile in range(4):
        for take in range(1, piles[pile] + 1):
            new_piles = piles[:]
            new_piles[pile] -= take
            new_tuple = tuple(new_piles)
            if is_losing(new_tuple):
                return False
    return True

def policy(piles: list[int]) -> str:
    for pile in range(4):
        for take in range(1, piles[pile] + 1):
            new_piles = piles[:]
            new_piles[pile] -= take
            if not all(p == 0 for p in new_piles) and is_losing(tuple(new_piles)):
                return f"{pile},{take}"
    # No winning move, pick a default legal move
    for pile in range(4):
        if piles[pile] > 0:
            return f"{pile},1"
    # Should not reach here if game is ongoing, but safe fallback
    raise ValueError("No legal moves available")
