The strategy for this policy is perfect-play misère Nim.

Key facts:
- This is **misère Nim**: taking the last object loses.
- The losing positions are:
  1. If every nonzero pile has size 1, then positions with an **odd** number of nonzero piles are losing.
  2. Otherwise, positions with **nim-sum = 0** are losing.

So the policy is:
- If all remaining piles are 1s, remove one from any nonempty pile. This is winning exactly when it leaves an odd number of 1-piles for the opponent.
- If there is exactly one pile larger than 1, use the special misère rule:
  - if the number of 1-piles is even, reduce the large pile to 1;
  - if the number of 1-piles is odd, remove the large pile entirely.
  This leaves the opponent an odd number of 1-piles.
- If there are at least two piles larger than 1, play normal Nim: move to make the xor of pile sizes equal to 0.
- If the current position is already losing, return a legal fallback move.

def policy(piles: list[int]) -> str:
    """
    Perfect-play policy for 4-pile misère Nim.
    Returns a move string: 'pile,take'
    """

    def first_legal_move() -> str:
        for i, p in enumerate(piles):
            if p > 0:
                return f"{i},1"
        # Terminal state should not be queried, but keep a syntactically valid fallback.
        return "0,1"

    # Count piles of size 1 and >1
    ones = 0
    big_indices = []
    xor_sum = 0

    for i, p in enumerate(piles):
        xor_sum ^= p
        if p == 1:
            ones += 1
        elif p > 1:
            big_indices.append(i)

    big = len(big_indices)

    # No legal move if terminal; assumed unreachable.
    if ones == 0 and big == 0:
        return "0,1"

    # Case 1: all nonzero piles are 1
    if big == 0:
        # Winning iff number of ones is even; removing 1 leaves odd.
        return first_legal_move()

    # Case 2: exactly one pile > 1
    # Special misère Nim endgame rule.
    if big == 1:
        i = big_indices[0]
        p = piles[i]
        if ones % 2 == 0:
            # Leave an odd number of 1-piles by turning the big pile into 1.
            take = p - 1
        else:
            # Leave an odd number of 1-piles by removing the big pile entirely.
            take = p
        return f"{i},{take}"

    # Case 3: at least two piles > 1
    # If xor != 0, play normal Nim: move to xor 0.
    if xor_sum != 0:
        for i, p in enumerate(piles):
            target = p ^ xor_sum
            if target < p:
                take = p - target
                return f"{i},{take}"

    # Already in a losing position: return a legal fallback move.
    # Slightly prefer reducing a big pile to 1, which can be awkward for imperfect opponents.
    for i, p in enumerate(piles):
        if p > 1:
            return f"{i},{p - 1}"
    return first_legal_move()
