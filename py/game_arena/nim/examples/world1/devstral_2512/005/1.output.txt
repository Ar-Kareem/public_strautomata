# Strategy for Nim Policy

The strategy for this policy is based on the mathematical theory of Nim, which is a well-known impartial game. The key insight is to use the concept of "Nim-sum" (bitwise XOR of all pile sizes) to determine winning and losing positions.

1. **Nim-sum Calculation**: Compute the XOR of all pile sizes. If the Nim-sum is zero, the current position is losing (assuming optimal play from the opponent). If non-zero, it's a winning position.

2. **Winning Move Selection**: When the Nim-sum is non-zero, we can always make a move that brings the Nim-sum to zero. This involves:
   - Finding a pile where the XOR of its size with the total Nim-sum is less than its current size
   - Reducing that pile to this new value (which is equivalent to taking the difference)

3. **Losing Position Handling**: If the Nim-sum is zero (losing position), we must make a move that leaves the opponent in a winning position. In this case, we'll make a random legal move (though any move will do since the position is already losing).

4. **Special Case Handling**: For the endgame (when only one pile remains), we must leave at least one object to avoid taking the last one (which would make us lose).

This strategy ensures perfect play in Nim, as it's based on the fundamental mathematical solution to the game. The policy will always make optimal moves when possible and handle losing positions gracefully.

# Python Implementation

```python
def policy(piles: list[int]) -> str:
    # Calculate the Nim-sum (bitwise XOR of all pile sizes)
    nim_sum = 0
    for pile in piles:
        nim_sum ^= pile

    # If Nim-sum is zero, we're in a losing position - make any legal move
    if nim_sum == 0:
        # Find the first non-empty pile
        for i in range(4):
            if piles[i] > 0:
                # Take 1 object (or more if it's the last pile)
                take = 1
                if sum(piles) == piles[i]:  # Only one pile left
                    take = max(1, piles[i] - 1)  # Leave at least 1 to avoid losing
                return f"{i},{take}"

    # Otherwise, find a winning move
    for i in range(4):
        if piles[i] > 0:
            # Calculate what the pile should be to make Nim-sum zero
            target = piles[i] ^ nim_sum
            if target < piles[i]:
                take = piles[i] - target
                return f"{i},{take}"

    # Fallback (should never reach here for valid Nim positions)
    return "0,1"
```
