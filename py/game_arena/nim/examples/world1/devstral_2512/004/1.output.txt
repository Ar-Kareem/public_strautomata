# Strategy for Nim Policy

The strategy for this policy is based on the mathematical theory of Nim, which is a well-known impartial game. The key insight is to use the concept of "Nim-sum," which is the bitwise XOR of all pile sizes. The optimal strategy is to always make a move that leaves the Nim-sum at zero. This ensures that the opponent is forced into a losing position if they don't also play optimally.

Here's how the strategy works:
1. Calculate the Nim-sum (bitwise XOR) of all pile sizes.
2. If the Nim-sum is zero, the position is losing, so we make a random legal move (since any move will leave the opponent in a winning position).
3. If the Nim-sum is non-zero, we find a move that will make the Nim-sum zero after our move. This involves:
   - For each pile, calculate what the new size should be to make the Nim-sum zero.
   - Find a pile where this new size is less than the current size (meaning we can take objects from it).
   - Take the difference between the current size and the target size from that pile.

This strategy guarantees perfect play in Nim, meaning it will never lose against any opponent that doesn't also play perfectly.

import random

def policy(piles: list[int]) -> str:
    # Calculate the Nim-sum (bitwise XOR of all pile sizes)
    nim_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]

    # If Nim-sum is zero, we're in a losing position - make a random legal move
    if nim_sum == 0:
        # Find all possible moves
        possible_moves = []
        for i in range(4):
            for take in range(1, piles[i] + 1):
                possible_moves.append((i, take))

        # Choose a random move
        if possible_moves:
            pile, take = random.choice(possible_moves)
            return f"{pile},{take}"
        else:
            # This should never happen as the game would be over
            return "0,1"

    # If Nim-sum is non-zero, find a move that makes it zero
    for i in range(4):
        # Calculate what the pile should be to make Nim-sum zero
        target = piles[i] ^ nim_sum

        # If this is smaller than current pile, we can take from it
        if target < piles[i]:
            take = piles[i] - target
            return f"{i},{take}"

    # Fallback (should never reach here if input is valid)
    return "0,1"
