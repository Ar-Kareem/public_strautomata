**Explanation**

The game is *misère Nim* (the player who takes the last object loses).  
For this variant the optimal strategy is well‑known:

1. **All piles have size 0 or 1**  
   - If every non‑empty pile contains exactly one object, the winning move is to leave an **even** number of non‑empty piles after your turn.  
   - The simplest way is to remove a whole pile (take the single object) from any pile that still has one object.  
   - When the number of piles with one object is already even you are in a losing position; any legal move (removing one object from any pile) is fine.

2. **At least one pile has size ≥ 2**  
   - In this case misère Nim behaves exactly like normal Nim.  
   - Compute the Nim‑sum (`xor`) of the pile sizes.  
   - If `xor == 0` the position is losing; we just make a trivial legal move (take one from the first non‑empty pile).  
   - If `xor != 0` we can force a win by reducing a pile `i` from `piles[i]` to `piles[i] ^ xor` (which is strictly smaller).  
     The number to take is `piles[i] - (piles[i] ^ xor)`.

The policy below implements this logic and always returns a legal move string `"pile,take"` (0‑based pile index, number of objects taken). It runs in O(1) time and uses only the standard library.

```python
def policy(piles):
    """
    Return a legal move for misère Nim (4 piles) in the form "pile,take".
    piles: list[int] of length 4, current pile sizes.
    The player who takes the last object loses.
    """
    # indices of non‑empty piles
    non_empty = [i for i, v in enumerate(piles) if v > 0]
    if not non_empty:                     # terminal position – no legal move
        return "0,0"

    # ---- Case 1: all piles are 0 or 1 (misère special case) ----
    if all(v <= 1 for v in piles):
        # Count how many piles have a single object
        ones = sum(v for v in piles)      # same as number of non‑empty piles here
        # The winning move is to make the number of remaining 1‑piles even.
        # The simplest: remove a whole pile (take the single object) from any pile.
        # If the count is already even we are losing; any move works.
        for i, v in enumerate(piles):
            if v == 1:
                return f"{i},1"

    # ---- Case 2: at least one pile > 1 (normal Nim rule) ----
    xor = 0
    for v in piles:
        xor ^= v

    if xor == 0:
        # Losing position – just take 1 from the first non‑empty pile
        pile = non_empty[0]
        return f"{pile},1"
    else:
        # Find a pile we can reduce to make xor zero
        for i, v in enumerate(piles):
            target = v ^ xor
            if target < v:
                take = v - target
                return f"{i},{take}"

        # Fallback (should never be reached)
        pile = non_empty[0]
        return f"{pile},1"
```
