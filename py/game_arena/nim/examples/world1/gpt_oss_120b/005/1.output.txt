**Strategy Overview**

This policy plays **optimal misère Nim** on four piles.

* Misère Nim rule: the player who takes the last object **loses**.  
* The optimal strategy is:
  1. **All piles are size 0 or 1**  
     * If the number of non‑empty piles is **even**, we can win by removing an entire pile (take 1 from any pile that has a stone).  
     * If the number is **odd**, the position is losing; we make any legal move (remove one stone from the first non‑empty pile).
  2. **At least one pile has size > 1**  
     * Play exactly as in normal Nim: compute the XOR (nim‑sum) of the pile sizes.  
     * If the nim‑sum is zero the position is losing, so we make a minimal legal move (take 1 from the first non‑empty pile).  
     * Otherwise, choose a pile where reducing it makes the nim‑sum zero:  
       `new_size = piles[i] XOR nim_sum` which is smaller than the current size.  
       Remove `take = piles[i] - new_size` stones from that pile.

The move is returned as the string `"pile,take"` where `pile` is a 0‑based index.

```python
def policy(piles):
    """
    Return a legal move for misère Nim with 4 piles.
    Move format: "pile,take" (both integers, comma‑separated).
    The player who takes the last object loses.
    """
    # Helper: find first non‑empty pile index
    def first_nonzero():
        for i, v in enumerate(piles):
            if v > 0:
                return i
        return -1  # should not happen in a legal call

    # Case 1: all piles are 0 or 1
    if max(piles) <= 1:
        ones = sum(1 for v in piles if v == 1)
        if ones % 2 == 0:
            # Winning: remove an entire pile (take 1)
            for i, v in enumerate(piles):
                if v == 1:
                    return f"{i},1"
        else:
            # Losing: any legal move (remove 1 from first non‑empty pile)
            idx = first_nonzero()
            return f"{idx},1"

    # Case 2: at least one pile > 1  (normal Nim strategy)
    nim_sum = 0
    for v in piles:
        nim_sum ^= v

    if nim_sum == 0:
        # Losing position: make a minimal move
        idx = first_nonzero()
        return f"{idx},1"

    # Find a pile to make nim_sum zero after the move
    for i, v in enumerate(piles):
        target = v ^ nim_sum
        if target < v:
            take = v - target
            return f"{i},{take}"

    # Fallback (should never be reached)
    idx = first_nonzero()
    return f"{idx},1"
```
