**Strategy Overview**

The game is **misère Nim** (the player who takes the last object loses).  
For misère Nim the optimal play is:

* If *all* non‑empty piles contain exactly one object, the position is decided only by the parity of the number of such piles.  
  *Even* count → winning: remove one whole pile (take 1 from any pile).  
  *Odd* count → losing: any legal move will hand the opponent a winning even count, but we still must return a legal move.

* Otherwise (there is at least one pile larger than 1) the game can be played exactly like normal Nim: compute the XOR (nim‑sum) of the pile sizes and make a move that leaves a nim‑sum of 0.  
  If the current nim‑sum is already 0 the position is losing; we simply make a legal move (take 1 from the first non‑empty pile).

The implementation follows these rules, guaranteeing a legal move in every situation and playing perfectly against any opponent.

```python
def policy(piles):
    """
    Return a legal move for misère Nim on four piles.

    Parameters
    ----------
    piles : list[int]
        List of length 4, each entry the number of objects in a pile.

    Returns
    -------
    str
        Move encoded as "pile,take" where `pile` is 0‑based index and
        `take` is the number of objects to remove (>=1 and <= piles[pile]).
    """
    # Helper: first non‑empty pile index
    def first_nonempty():
        for i, v in enumerate(piles):
            if v > 0:
                return i
        return 0  # should never happen in a valid call

    # Count piles larger than 1
    big_piles = [v for v in piles if v > 1]

    # --- Case 1: all non‑empty piles are size 1 (or all zero) ---
    if not big_piles:
        ones = sum(1 for v in piles if v == 1)   # total non‑empty piles
        # Winning when the count of ones is even: remove one whole pile
        if ones % 2 == 0:
            idx = first_nonempty()
            return f"{idx},1"
        # Losing position (odd count): any legal move is fine
        idx = first_nonempty()
        return f"{idx},1"

    # --- Case 2: at least one pile > 1 ⇒ play normal Nim strategy ---
    # Compute nim‑sum (xor of all pile sizes)
    xor_sum = 0
    for v in piles:
        xor_sum ^= v

    # If xor is zero the position is losing; just take 1 from any pile
    if xor_sum == 0:
        idx = first_nonempty()
        return f"{idx},1"

    # Otherwise, find a pile to reduce so that the resulting xor becomes 0
    for i, v in enumerate(piles):
        target = v ^ xor_sum          # size we want to leave in this pile
        if target < v:
            take = v - target
            # Safety check: take must be at least 1 and not exceed pile size
            if take >= 1 and take <= v:
                return f"{i},{take}"

    # Fallback (should never be reached): take 1 from the first non‑empty pile
    idx = first_nonempty()
    return f"{idx},1"
```
