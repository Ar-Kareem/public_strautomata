### Strategy for this Policy

The strategy for this policy is based on the **misère Nim** variant (where the player to take the last object loses). For standard Nim (where the player to take the last object wins), the optimal strategy is to make the XOR (nim-sum) of all pile sizes zero. However, in misère Nim, the strategy is more complex, especially for the case of 4 piles.

For misère Nim with 4 piles, the optimal strategy involves:
1. If the nim-sum (XOR of all piles) is non-zero, play as in standard Nim (reduce the nim-sum to zero).
2. If the nim-sum is zero, we must force the opponent into a losing position. The key insight is to leave the opponent with a position where all piles are equal (if possible) or to reduce the largest pile to match the second-largest pile (if all piles are not equal). This ensures that the opponent cannot force a win.

The policy will:
- Compute the nim-sum (XOR of all piles).
- If the nim-sum is non-zero, make a move that reduces the nim-sum to zero (standard Nim strategy).
- If the nim-sum is zero, make a move that leaves all piles equal (if possible) or reduces the largest pile to match the second-largest pile (if not all piles are equal). This is a known winning strategy for misère Nim with 4 piles.

```python
def policy(piles: list[int]) -> str:
    # Compute the nim-sum (XOR of all piles)
    nim_sum = 0
    for pile in piles:
        nim_sum ^= pile

    # If nim-sum is non-zero, play standard Nim (reduce nim-sum to zero)
    if nim_sum != 0:
        for i in range(4):
            if (piles[i] ^ nim_sum) < piles[i]:
                take = piles[i] - (piles[i] ^ nim_sum)
                return f"{i},{take}"

    # If nim-sum is zero, we are in a losing position if opponent plays optimally.
    # For 4 piles, the strategy is to leave all piles equal or reduce the largest pile to match the second-largest.
    # First, check if all piles are equal (losing position)
    if all(p == piles[0] for p in piles):
        # If all piles are equal and non-zero, take 1 from any pile (forcing opponent to break symmetry)
        return f"0,1"

    # Find the largest pile
    max_pile = max(piles)
    max_index = piles.index(max_pile)

    # If there are multiple piles with max_pile, choose the first one
    if piles.count(max_pile) > 1:
        max_index = piles.index(max_pile)

    # If the largest pile is not unique, reduce it to match the second-largest
    second_largest = sorted(piles, reverse=True)[1]
    if max_pile != second_largest:
        take = max_pile - second_largest
        return f"{max_index},{take}"

    # If all piles are equal except one, reduce the largest to match the others
    # This is a fallback (should not happen if nim-sum is zero and not all equal)
    for i in range(4):
        if piles[i] != piles[0]:
            take = piles[i] - piles[0]
            return f"{i},{take}"

    # Fallback: take 1 from any pile (should not reach here for valid inputs)
    return "0,1"
```

### Explanation:
1. **Nim-sum check**: If the nim-sum is non-zero, the policy reduces the nim-sum to zero by making a move in the standard Nim way.
2. **All piles equal**: If all piles are equal (and non-zero), the policy takes 1 object from any pile to break the symmetry.
3. **Largest pile handling**: If the nim-sum is zero and not all piles are equal, the policy reduces the largest pile to match the second-largest pile. This is a known winning strategy for misère Nim with 4 piles.
4. **Fallback**: A fallback move is included to ensure a legal move is always returned (though this should theoretically never be reached for valid inputs).

This strategy ensures optimal play against any opponent in a 4-pile misère Nim game.
