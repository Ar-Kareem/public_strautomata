The strategy implements optimal play for misère Nim (the player who takes the last object loses).  
- If all piles are 0 or 1, the only sensible moves are to remove a whole pile of size 1.  
  The position is winning when the number of ones is even, losing when odd.  
  In either case we take a 1 (it is optimal when winning and any move is fine when losing).  
- If there is at least one pile of size ≥2 and the XOR of all pile sizes is 0, the position is losing;  
  we make an arbitrary legal move (taking 1 from a pile ≥2).  
- If there is at least one pile of size ≥2 and the XOR is non‑zero, the position is winning.  
  We first try a normal Nim move: reduce a pile so that the new XOR becomes 0,  
  provided that after the move at least one pile still has size ≥2.  
  If such a move exists, it is optimal.  
  The only situation where it does not exist is when exactly one pile is larger than 1  
  and all others are 0 or 1. In that special case we instead make the number of piles  
  of size 1 odd: if the current number of ones is even, we take all but one from the  
  large pile (leaving it as 1); if it is odd, we take everything from the large pile  
  (leaving it as 0). This always leads to a losing position for the opponent.

The code follows this strategy and always returns a legal move string.

<code>
def policy(piles):
    # Count ones and check whether all piles are 0 or 1
    ones = piles.count(1)
    all_small = all(p <= 1 for p in piles)

    if all_small:
        # Only moves are taking a whole pile of size 1.
        for i, p in enumerate(piles):
            if p == 1:
                return f"{i},1"
        # Should never be called on an all‑zero position.
        return "0,1"   # fallback, but input is guaranteed non‑terminal

    # There is at least one pile with size >= 2
    x = 0
    for p in piles:
        x ^= p

    if x == 0:
        # Losing position – any legal move is acceptable.
        for i, p in enumerate(piles):
            if p >= 2:
                return f"{i},1"
        # Fallback (should not happen because we are not all_small)
        return "0,1"

    # Winning position (x != 0)
    # Try a normal Nim move that leaves at least one pile >= 2.
    for i, p in enumerate(piles):
        new_size = p ^ x
        if new_size < p:   # this would reduce the pile
            new_piles = piles.copy()
            new_piles[i] = new_size
            if any(q >= 2 for q in new_piles):
                take = p - new_size
                return f"{i},{take}"

    # Special case: exactly one pile > 1 and all others are 0 or 1.
    # Find that pile and the number of ones.
    big_idx = None
    big_val = 0
    ones = 0
    for i, p in enumerate(piles):
        if p > 1:
            big_idx = i
            big_val = p
        elif p == 1:
            ones += 1

    # Leave an odd number of ones.
    if ones % 2 == 1:
        take = big_val          # take everything, leave 0
    else:
        take = big_val - 1      # leave 1
    return f"{big_idx},{take}"
