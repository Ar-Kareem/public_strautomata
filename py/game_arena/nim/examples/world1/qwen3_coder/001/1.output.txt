I need to create a policy for Nim with 4 piles where the player who takes the last object loses.

This is a misère Nim game. The key insight is to use the concept of Nim-sum (XOR of all pile sizes) and adapt it for misère play. In normal Nim, you want to make the Nim-sum zero, but in misère Nim, the strategy is more complex.

For misère Nim:
1. If all piles have size 1, you want to leave an odd number of piles for your opponent
2. Otherwise, play like normal Nim until all piles are of size 1 or less

Let me think about the strategy:
- Calculate the Nim-sum (XOR of all pile sizes)
- If Nim-sum is 0, we're in a losing position for normal play, so we need a different approach
- If Nim-sum is not 0, we can make a move to set it to 0 (but need to be careful about misère)

Actually, let me be more precise about misère Nim strategy:
- When all piles have size ≤ 1, count the number of non-empty piles
- If odd number of non-empty piles, we want to make it even for opponent
- If even number of non-empty piles, we want to make it odd for opponent
- Otherwise, play like normal Nim (make Nim-sum = 0)

def policy(piles: list[int]) -> str:
    # Calculate nim-sum
    nim_sum = 0
    for pile in piles:
        nim_sum ^= pile
    
    # Check if we're in endgame (all piles <= 1)
    in_endgame = all(pile <= 1 for pile in piles)
    
    if in_endgame:
        # Count non-empty piles
        non_empty_count = sum(1 for pile in piles if pile > 0)
        # In misère game, we want to leave opponent with odd number of piles
        # If we have odd number of piles, take from any pile to make it even
        # If we have even number of piles, take from any pile to make it odd (but prefer to not lose immediately)
        for i in range(4):
            if piles[i] > 0:
                # Taking 1 from this pile
                remaining_piles = non_empty_count - 1
                # If remaining is odd, opponent loses
                return f"{i},1"
    else:
        # Not in endgame, play normal nim strategy but be careful
        # Try to make nim-sum = 0
        for i in range(4):
            if piles[i] > 0:
                # What should this pile be to make nim-sum = 0?
                target = nim_sum ^ piles[i]
                if target < piles[i]:
                    take = piles[i] - target
                    return f"{i},{take}"
        
        # If we can't make nim-sum 0, just take 1 from first available pile
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    
    # Fallback (should never reach here with legal game states)
    for i in range(4):
        if piles[i] > 0:
            return f"{i},1"
