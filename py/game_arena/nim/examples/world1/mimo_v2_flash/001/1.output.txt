### Logic for the Nim Policy

The game described is a variant of Nim known as **Misère Nim**. In standard Nim, the player who takes the last object wins. In Misère Nim, the player who takes the last object loses.

The winning strategy for Misère Nim differs from standard Nim only in the endgame scenario (when all piles have size 1 or 0). Specifically, if every non-empty pile has exactly one object, the strategy flips. Otherwise, the strategy is identical to standard Nim: calculate the **Nim-sum** (XOR sum) of all pile sizes.

**The Strategy:**
1.  **General Case (Not all piles are size 0 or 1):**
    *   Calculate the Nim-sum $S = piles[0] \oplus piles[1] \oplus piles[2] \oplus piles[3]$.
    *   If $S = 0$, the position is losing. We are forced to move, so we must make a move that leaves the opponent with a Nim-sum of 0 if possible, but since we are in a losing position, any move will leave a non-zero sum (in general). To minimize damage, we pick the first non-empty pile and remove 1 object.
    *   If $S \neq 0$, the position is winning. We need to make a move $piles[i] \to new\_size$ such that the new Nim-sum is 0. This is possible by picking a pile $i$ where the most significant bit of $S$ is set in $piles[i]$, and setting the new size to $piles[i] \oplus S$. We take $take = piles[i] - (piles[i] \oplus S)$.

2.  **Endgame Case (All piles are 1 or 0):**
    *   Count the number of piles with size 1. Let this be $k$.
    *   If $k$ is odd (1 or 3 piles of size 1), we are in a winning position. We must take exactly 1 object from a pile of size 1 (changing $k$ to an even number, leaving the opponent with 2 or 0 piles of size 1).
    *   If $k$ is even (0, 2, or 4 piles of size 1), we are in a losing position. We will be forced to leave an odd number of 1s for the opponent. We can just pick the first pile of size 1 and remove 1.

This logic covers all possible states and guarantees optimal play.

def policy(piles: list[int]) -> str:
    # 1. Check for the "All 1s" scenario to distinguish from general Nim
    is_all_ones = True
    for p in piles:
        if p != 0 and p != 1:
            is_all_ones = False
            break

    if is_all_ones:
        # 2. Misère Nim Endgame Strategy
        # Count piles with exactly 1 object
        count_ones = sum(1 for p in piles if p == 1)
        
        # If count_ones is odd, it's a winning position (we leave even count for opponent).
        # If count_ones is even, it's a losing position (we leave odd count for opponent).
        # In both cases, the correct move is to remove 1 from a pile of size 1.
        for i in range(4):
            if piles[i] == 1:
                return f"{i},1"

    # 3. General Nim Strategy (Standard Nim for non-ending positions)
    # Calculate Nim-sum
    nim_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]

    if nim_sum == 0:
        # We are in a losing position.
        # Make a legal move that minimizes impact. Remove 1 from the first non-empty pile.
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
    else:
        # We are in a winning position.
        # Find a move that makes the nim_sum 0.
        for i in range(4):
            # We can modify pile i if the target size is valid (smaller than current)
            target = piles[i] ^ nim_sum
            if target < piles[i]:
                take = piles[i] - target
                return f"{i},{take}"

    # Should not be reached given valid game states
    return "0,1"
