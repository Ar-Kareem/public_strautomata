The strategy for this policy is to implement the optimal Nim strategy using the XOR (nim-sum) of pile sizes to force a win when possible, and to make a legal move when in a losing position. The approach involves:
1. Calculating the nim-sum (XOR of all pile sizes).
2. If the nim-sum is non-zero, find a pile where reducing it to (nim-sum XOR pile) is valid, and take the difference.
3. If the nim-sum is zero, take one object from any non-zero pile to minimize the opponent's advantage.

def policy(piles: list[int]) -> str:
    nim_sum = 0
    for p in piles:
        nim_sum ^= p
    
    if nim_sum != 0:
        for i in range(4):
            target = nim_sum ^ piles[i]
            if target < piles[i]:
                take = piles[i] - target
                return f"{i},{take}"
    else:
        for i in range(4):
            if piles[i] > 0:
                return f"{i},1"
