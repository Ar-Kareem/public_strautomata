

The strategy forthis policy is to use the optimal Nim strategy, which involves calculating the nim-sum (XOR of all pile sizes) to determine the winning move. If the nim-sum is non-zero, reduce a pile to make the nim-sum zero. If the nim-sum is zero, reduce a pile to 1 if possible; otherwise, take one from the first pile. This ensures optimal play in both normal and misère game modes.

def policy(piles: list[int]) -> str:
    nim_sum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    if nim_sum != 0:
        for i in range(4):
            target = piles[i] ^ nim_sum
            if target < piles[i]:
                return f"{i},{target}"
    else:
        for i in range(4):
            if piles[i] > 1:
                return f"{i},1"
        return "0,1"
