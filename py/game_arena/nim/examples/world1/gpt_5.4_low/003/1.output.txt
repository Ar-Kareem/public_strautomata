The strategy for this policy is to play **optimal misère Nim**.

Because the rule is “the player who takes the last object loses,” this is **misère play**, not normal Nim. The optimal policy is:

1. Count how many piles are larger than 1.
2. If **all nonzero piles are size 1**, then:
   - Leave an **odd** number of 1-heaps to the opponent.
   - So if the number of 1-heaps is even, remove 1 from any nonempty pile.
   - If it is odd, you have no winning move, but any legal move is fine.
3. Otherwise, if there is **exactly one pile larger than 1**:
   - Let `ones` be the number of piles equal to 1.
   - Reduce the large pile to:
     - `1` if `ones` is odd
     - `0` if `ones` is even
   This leaves the opponent an odd number of 1-heaps.
4. Otherwise, when **two or more piles are larger than 1**:
   - Play like normal Nim: compute the xor (`nim-sum`) of the pile sizes.
   - If the xor is nonzero, make it zero by reducing one pile appropriately.
   - If the xor is already zero, no winning move exists; return any legal move.

This is the standard perfect-play solution for misère Nim and always returns a legal move.

<code>
def policy(piles: list[int]) -> str:
    # Ensure we always return a legal move, even on unexpected input.
    if not isinstance(piles, list) or len(piles) != 4:
        return "0,1"

    # Gather pile information
    nonzero = [i for i, x in enumerate(piles) if x > 0]
    if not nonzero:
        # Terminal/unexpected; must still return a string.
        return "0,1"

    ones = sum(1 for x in piles if x == 1)
    big_indices = [i for i, x in enumerate(piles) if x > 1]
    big_count = len(big_indices)

    # Case 1: all nonzero piles are 1
    if big_count == 0:
        # Winning if we can leave an odd number of 1-heaps to opponent.
        # If ones is even, remove one whole heap.
        i = nonzero[0]
        return f"{i},1"

    # Case 2: exactly one pile > 1
    if big_count == 1:
        i = big_indices[0]
        x = piles[i]
        # Leave odd number of 1-heaps to opponent:
        # if ones is odd -> reduce big pile to 1
        # if ones is even -> reduce big pile to 0
        target = 1 if (ones % 2 == 1) else 0
        take = x - target
        if 1 <= take <= x:
            return f"{i},{take}"

    # Case 3: at least two piles > 1
    # Play normal Nim: move to xor 0 if possible.
    xor_sum = 0
    for x in piles:
        xor_sum ^= x

    if xor_sum != 0:
        for i, x in enumerate(piles):
            target = x ^ xor_sum
            if target < x:
                take = x - target
                if 1 <= take <= x:
                    return f"{i},{take}"

    # Fallback: no winning move or unexpected issue; return any legal move.
    for i, x in enumerate(piles):
        if x > 0:
            return f"{i},1"

    # Should never happen if game state is valid.
    return "0,1"
