The strategy for this policy is the optimal misère Nim strategy.

Because the player who takes the last object loses, this is **misère Nim**, not normal Nim. The perfect-play policy is:

1. **If all nonempty piles are size 1**
   - This is a pure “take one heap” parity game.
   - If the number of nonempty piles is **odd**, the position is losing.
   - If it is **even**, remove an entire heap of size 1, leaving an odd number to the opponent.

2. **If exactly one pile is larger than 1**
   - Let `ones` be the number of piles equal to 1.
   - Reduce the large pile so that the opponent faces an **odd** number of size-1 piles:
     - if `ones` is even, reduce the large pile to `1`
     - if `ones` is odd, reduce the large pile to `0`

3. **If at least two piles are larger than 1**
   - Play like normal Nim:
   - Compute the xor (`nim-sum`) of pile sizes.
   - If it is nonzero, make a move that turns it into zero.
   - If it is already zero, no winning move exists, so return any legal move.

This is the standard perfect solution for misère Nim and always returns a legal move.

<code>
def policy(piles: list[int]) -> str:
    # Safety: always return a legal move if any exists.
    nonzero_indices = [i for i, p in enumerate(piles) if p > 0]
    if not nonzero_indices:
        # Terminal state should normally not be queried, but stay robust.
        return "0,0"

    ones = sum(1 for p in piles if p == 1)
    bigs = [(i, p) for i, p in enumerate(piles) if p > 1]
    nonzero = len(nonzero_indices)

    # Case 1: all nonempty piles are size 1
    if len(bigs) == 0:
        # Winning iff even number of 1-heaps; remove one whole heap.
        i = nonzero_indices[0]
        return f"{i},1"

    # Case 2: exactly one pile > 1
    if len(bigs) == 1:
        i, p = bigs[0]
        # Leave opponent an odd number of 1-heaps.
        target = 1 if (ones % 2 == 0) else 0
        take = p - target
        if take >= 1:
            return f"{i},{take}"
        # Fallback, though take should always be >= 1 here.
        return f"{i},1"

    # Case 3: at least two piles > 1 -> normal Nim strategy
    x = 0
    for p in piles:
        x ^= p

    if x != 0:
        for i, p in enumerate(piles):
            target = p ^ x
            if target < p:
                take = p - target
                if 1 <= take <= p:
                    return f"{i},{take}"

    # Losing position under perfect play: return any legal move.
    for i, p in enumerate(piles):
        if p > 0:
            return f"{i},1"

    # Unreachable fallback
    return "0,0"
