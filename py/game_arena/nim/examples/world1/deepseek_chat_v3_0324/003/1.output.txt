To solve this problem, we need to implement a smart policy for the game of Nim with four piles where the player who takes the last object loses. This variant is known as Misère Nim. The strategy involves understanding the XOR (Nim-sum) of the pile heights and making moves that either force the opponent into a losing position or avoid giving them such an opportunity.

### Approach
1. **Understand Misère Nim**: In standard Nim, the player who takes the last object wins, and the optimal strategy involves maintaining a Nim-sum (XOR of all pile heights) of zero. In Misère Nim, the player who takes the last object loses. The standard Nim strategy mostly applies, except when all piles have one object remaining; in that case, you should leave an odd number of piles for the opponent.
2. **Check for Winning Moves**: Calculate the Nim-sum of all pile heights. If the Nim-sum is non-zero, there exists a move that can make it zero, leading to a winning position. If the Nim-sum is zero, any move will leave the opponent in a winning position, so the best you can do is minimize the remaining objects to prolong the game.
3. **Special Case Handling**: If all piles are reduced to one object, the optimal move is to take all objects from a pile (or leave an appropriate number of piles) to ensure the opponent is forced to take the last object.

### Solution Code
```python
import sys

def policy(piles: list[int]) -> str:
    if len(piles) != 4:
        return "0,1"  # fallback in case of unexpected input
    
    # Check if all piles are 1 (or zero, with some 1's)
    all_ones = True
    for pile in piles:
        if pile > 1:
            all_ones = False
            break
    total_objects = sum(piles)
    
    if all_ones:
        # In misère Nim, leave odd number of 1's for opponent. So if total number is even, take all from one pile.
        if total_objects % 2 == 1:
            # If total is odd, need to take 1 to make even
            for i in range(4):
                if piles[i] == 1:
                    return f"{i},1"
        else:
            # Take a whole pile (e.g., 1)
            for i in range(4):
                if piles[i] >= 1:
                    return f"{i},{piles[i]}"
    
    # Standard Nim strategy applying XOR sum
    xor_sum = 0
    for pile in piles:
        xor_sum ^= pile
    
    if xor_sum != 0:
        # Find a pile where XOR sum with current pile is less than current pile
        for i in range(4):
            if piles[i] ^ xor_sum < piles[i]:
                take = piles[i] - (piles[i] ^ xor_sum)
                if take >= 1 and take <= piles[i]:
                    return f"{i},{take}"
    else:
        # No winning move possible, take one from the largest pile to prolong the game
        max_pile = max(piles)
        if max_pile == 0:
            return "0,1"  # shouldn't happen as piles may not be empty on our turn
        for i in range(4):
            if piles[i] == max_pile:
                return f"{i},1"
    
    # Fallback: take one from the first non-zero pile
    for i in range(4):
        if piles[i] > 0:
            return f"{i},1"
    
    return "0,1"
```

### Explanation
1. **Initial Checks**: The policy first checks if all piles contain exactly one object. In this scenario, it either takes an entire pile (if the total number of objects is even) or takes one object (if odd), forcing the opponent into a losing position.
2. **Nim-sum Calculation**: If not all piles have one object, the policy calculates the Nim-sum (XOR of all pile heights). If the Nim-sum is non-zero, it finds a move that reduces the chosen pile such that the resulting Nim-sum becomes zero.
3. **Fallback Strategies**: If the Nim-sum is already zero, the policy takes one object from the largest pile to prolong the game. If no other moves are found, it defaults to taking one object from the first available pile.
4. **Edge Handling**: The policy includes checks to ensure it always returns a legal move, even in unexpected situations (e.g., faulty input).

This approach ensures the policy makes optimal moves in most scenarios, leveraging both standard Nim strategy and Misère Nim adjustments to maximize winning chances.
