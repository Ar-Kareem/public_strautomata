# Strategy for Nim (4 piles) with Misère Play

For this Nim variant where the player who takes the last object loses (misère play), we need an optimal strategy that differs slightly from standard Nim. Here's my approach:

1. **Normal Nim Analysis**: First calculate the Nim-sum (bitwise XOR) of all pile sizes.
2. **Winning Position**: If the Nim-sum is non-zero, we're in a winning position and should make a move that brings the Nim-sum to zero.
3. **Misère Adjustment**: For the endgame (when most piles are empty), we need to adjust our strategy:
   - Leave an odd number of piles with one object when possible
   - Force the opponent to take the last object
4. **Special Cases**: Handle small pile counts differently to optimize misère play

The strategy combines standard Nim theory with misère-specific endgame tactics to ensure perfect play.

import numpy as np

def policy(piles: list[int]) -> str:
    # Calculate nim-sum
    nim_sum = np.bitwise_xor.reduce(piles)
    
    # If all piles are size 1, special misère handling
    if all(p == 1 for p in piles):
        # Leave odd number of piles (take from first pile)
        return '0,1'
    
    # Count non-empty piles
    non_empty = sum(1 for p in piles if p > 0)
    
    # When few piles remain, adjust for misère play
    if non_empty <= 2:
        # Find the largest pile (to reduce to 1 or 0)
        max_pile = max((p, i) for i, p in enumerate(piles))[1]
        if piles[max_pile] > 1:
            return f"{max_pile},{piles[max_pile]-1}"
        else:
            # Must take last object from another pile
            for i, p in enumerate(piles):
                if p > 0 and i != max_pile:
                    return f"{i},{p}"
    
    # Standard Nim strategy when not in endgame
    if nim_sum != 0:
        # Find a move that makes nim-sum zero
        for i in range(4):
            if piles[i] > 0:
                target = piles[i] ^ nim_sum
                if target < piles[i]:
                    return f"{i},{piles[i] - target}"
    
    # Default move (take from largest pile when in losing position)
    max_pile = max((p, i) for i, p in enumerate(piles))[1]
    return f"{max_pile},1"
