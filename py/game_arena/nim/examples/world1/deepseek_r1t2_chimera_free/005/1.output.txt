

The strategy for this policy is to employ an optimal Nim strategy incorporating both normal play XOR-based moves and adjustments for misère endgame conditions. The policy first checks for a endgame scenario where exactly one pile has more than one object, handling it by ensuring the number of single-object piles is odd when modified. In other situations, the policy calculates the Nim sum (XOR of all piles) to make non-losing moves. If the Nim sum is zero (a losing position), it takes one stone from the largest pile to prolong the game.

def policy(piles: list[int]) -> str:
    # Count the number of 1-piles and find non-1 piles
    ones = sum(pile == 1 for pile in piles)
    non_ones = [pile for pile in piles if pile > 1]
    
    # Handle the case with exactly one pile greater than 1
    if len(non_ones) == 1:
        i = next(idx for idx, pile in enumerate(piles) if pile > 1)
        k = piles[i]
        if ones % 2 == 1:
            take = k  # Remove entire pile (leaves odd 1-piles)
        else:
            take = k - 1  # Leave one (leaves odd 1-piles)
        return f"{i},{take}"
    
    # Compute Nim sum for normal play strategy
    xorsum = piles[0] ^ piles[1] ^ piles[2] ^ piles[3]
    
    # If losing position, remove 1 from the largest pile
    if xorsum == 0:
        max_val = max(piles)
        for i in range(4):
            if piles[i] == max_val:
                return f"{i},1"
    else:
        # Find a move that zeros the Nim sum
        for i in range(4):
            if piles[i] > 0:
                target = piles[i] ^ xorsum
                if target < piles[i]:
                    take = piles[i] - target
                    return f"{i},{take}"
    
    # Fallback (should not reach here)
    for i in range(4):
        if piles[i] > 0:
            return f"{i},1"
