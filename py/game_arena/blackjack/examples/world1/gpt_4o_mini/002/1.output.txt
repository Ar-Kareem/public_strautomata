To create a policy for the simplified Blackjack-like game, I need to consider the rules of the game, how the card values are calculated, and the conditions for winning or losing based on the target score.

I'll start by defining the value of each card. For cards numbered from "2" to "10", the value is straightforward. For "J", "Q", and "K", the value is set to 10, and "A" can take on two values depending on the current sum of the hand compared to the target score.

The policy for drawing a card or staying must focus on maximizing the chance of winning. Some possible strategies include:
1. If the current hand value is below the target score by a considerable margin, it may be reasonable to "HIT."
2. If the current hand value is close to the target score, it's better to "STAY" to avoid busting.
3. The specific thresholds for hitting or staying will depend on the target score.

A simple strategy would be:
- Always hit if the current value is significantly lower than the target.
- Stay if the current value is close enough to the target (for instance, within 2 to 3 points).
- If the hand value is greater than the target score, then the player has busted.

I'll encode this logic into a function that checks the current hand value and uses simple thresholds to decide whether to hit or stay.

def card_value(card: str) -> int:
    if card in ["J", "Q", "K"]:
        return 10
    elif card == "A":
        return 11  # Initially treat Ace as 11
    else:
        return int(card)

def hand_value(hand: list[str], target: int) -> int:
    total = 0
    aces = 0
    for card in hand:
        total += card_value(card)
        if card == "A":
            aces += 1
    
    # Adjust for Aces if necessary
    while total > target and aces > 0:
        total -= 10  # Consider Ace as 1 instead of 11
        aces -= 1
    
    return total

def policy(hand: list[str], target: int) -> str:
    current_value = hand_value(hand, target)
    
    # Define thresholds for strategy
    if current_value < target - 3:  # If less than target - 3, it's safe to HIT
        return "HIT"
    else:  # Otherwise, we STAY
        return "STAY"
