Since Aces can be 1 or 11, we first calculate the optimal hand value by treating Aces as 11 when beneficial (i.e., when hand value ≤ target). We then implement a risk-aware decision rule:

1. If the current optimal hand value exceeds the target → STAY (already busted, no point in drawing more)
2. If the current optimal hand value is very close to target (within 1-3 points) → STAY (minimize risk of busting)
3. If the current optimal hand value is low (≤ target - 10) → HIT (high reward potential)

For intermediate values (target - 10 < current ≤ target - 4), we use a probabilistic heuristic: calculate the probability of drawing a card that doesn't cause a bust, and HIT only if this probability is above a threshold (e.g., 50%). This threshold is adjusted based on how close we are to the target.

We also account for the fact that cards are drawn without replacement from a fixed deck of 13 unique cards. We track which cards remain to estimate the probability distribution of the next card.

This policy balances aggression when behind and caution when near the target, while explicitly considering card depletion to avoid overestimating safe drawing probabilities.
