To determine the optimal move in a Blackjack-like game, we need to evaluate the best action (HIT or STAY) based on the current hand and the target sum. The decision hinges on the probability of improving the current hand sum without exceeding the target, while also avoiding a bust.

### Approach
1. **Calculate Current Hand Value**: Determine the current hand's value by treating Aces as 1 initially and then converting them to 11 if possible without exceeding the target.
2. **Check for Immediate Bust**: If the current hand value exceeds the target, stay to avoid busting.
3. **Evaluate Remaining Cards**: Identify the remaining cards in the deck (excluding those already in the hand).
4. **Determine Improvement Possibility**: Check if any of the remaining cards, when drawn, would result in a sum closer to the target (without busting). If such a card exists, the optimal move is to HIT; otherwise, STAY.

### Solution Code
```python
def policy(hand, target):
    # Helper function to calculate the hand's value considering Ace adjustments
    def calculate_value(hand, target):
        sum_val = 0
        aces = 0
        for card in hand:
            if card == 'A':
                sum_val += 1
                aces += 1
            elif card in {'J', 'Q', 'K'}:
                sum_val += 10
            else:
                # Convert card value to integer
                sum_val += int(card)
        
        # Adjust for Aces if possible
        while aces > 0 and (sum_val + 10) <= target:
            sum_val += 10
            aces -= 1
        
        return sum_val
    
    current_sum = calculate_value(hand, target)
    
    # If current sum exceeds target, must stay
    if current_sum > target:
        return "STAY"
    
    # Determine available cards in the deck (not in hand)
    all_cards = {'A', '2', '3', '4', '5', '6', '7', '8', '9', 'J', 'Q', 'K', '10'}
    available_cards = list(all_cards - set(hand))
    
    # No cards left to draw
    if not available_cards:
        return "STAY"
    
    # Check if any draw improves our situation without busting
    improved_possible = False
    for card in available_cards:
        new_hand = hand + [card]
        new_sum = calculate_value(new_hand, target)
        if new_sum <= target and new_sum > current_sum:
            improved_possible = True
            break
    
    return "HIT" if improved_possible else "STAY"
```

### Explanation
1. **Hand Value Calculation**: The `calculate_value` function determines the current hand's value by treating Aces as 1 initially and then converting them to 11 if possible without exceeding the target.
2. **Immediate Bust Check**: If the current hand value exceeds the target, the player must stay to avoid busting.
3. **Available Cards Identification**: The remaining cards in the deck are identified by subtracting the current hand from the set of all possible cards.
4. **Improvement Check**: For each available card, compute the new sum. If any of these cards can be drawn without busting and improve the hand's value (i.e., get closer to the target), the player should HIT; otherwise, stay.

This policy ensures that the player maximizes their chances of getting closer to the target sum while avoiding busting, making it a strategic approach to beat other policies in the game arena.
