

Thestrategy for this policy is to calculate the current hand total by considering non-Ace cards and Aces (which can be 1 or 11 based on the target). If the current total is below the target, the policy hits if the difference is 5 or less; otherwise, it stays. This balances risk and reward by targeting close differences while avoiding unnecessary risks when far from the target.

<code>
def policy(hand: list[str], target: int) -> str:
    S = 0
    A = 0
    for card in hand:
        if card == "A":
            A += 1
        else:
            if card in ["J", "Q", "K"]:
                value = 10
            elif card == "10":
                value = 10
            else:
                value = int(card)
            S += value
    
    current_total = S + A + min(10 * A, max(0, target - S - A))
    
    if current_total < target:
        diff = target - current_total
        if diff <= 5:
            return "HIT"
        else:
            return "STAY"
    else:
        return "STAY"
