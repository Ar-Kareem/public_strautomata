The strategy for this policy is to adopt a risk-averse approach that prioritizes avoiding busting while maximizing the chance of reaching or exceeding the target score. The policy calculates the current hand's total value, accounting for Aces (which can be 1 or 11), and uses a threshold-based decision rule to determine whether to hit or stay. The threshold is dynamically adjusted based on the target score to balance aggression and caution.

def policy(hand: list[str], target: int) -> str:
    # Calculate non-Ace card values and count Aces
    non_ace_sum = 0
    ace_count = 0
    for card in hand:
        if card == 'A':
            ace_count += 1
        elif card in ['J', 'Q', 'K']:
            non_ace_sum += 10
        else:
            non_ace_sum += int(card)
    
    # Calculate maximum possible total without busting
    total = non_ace_sum + 11 * ace_count
    if total > target:
        # Adjust Aces from 11 to 1 if total exceeds target
        k = (total - target + 9) // 10
        total = total - 10 * k
    
    # Dynamic threshold: hit if total is below (target - 5)
    if total < target - 5:
        return "HIT"
    else:
        return "STAY"
