Okay, I need to write a Blackjack-like policy. The core idea is to decide when to "HIT" and when to "STAY". I should probably calculate the current hand value and then make a decision based on how close I am to the target. Aces are tricky because of the 11/1 rule.

Here's a basic structure:

1. **`card_value(card)` function:** Converts a card string to its numerical value.
2. **`hand_value(hand, target)` function:** Calculates the total value of a hand, taking into account the Ace rule.
3. **`policy(hand, target)` function:** The main policy function:
   - Calculate current hand value.
   - If hand value > target, return "STAY" (since I will bust if I draw).
   - If hand value is close to target, return "STAY".
   - Otherwise, return "HIT".

Let's start with a simple strategy: stay if the hand value is greater than or equal to some threshold. I can start the threshold with target - 4. I will also try to improve ace handling by counting total aces and calculating the sum with aces being 11. If current sum is greater than target, I convert it to 1s until the sum is under or equal to target.

```python
def card_value(card: str) -> int:
    if card.isdigit():
        return int(card)
    elif card in ("J", "Q", "K"):
        return 10
    elif card == "A":
        return 11
    else:
        raise ValueError(f"Invalid card: {card}")


def hand_value(hand: list[str], target: int) -> int:
    ace_count = hand.count("A")
    total = 0
    for card in hand:
        total += card_value(card)

    while total > target and ace_count > 0:
        total -= 10
        ace_count -= 1

    return total


def policy(hand: list[str], target: int) -> str:
    hand_value_current = hand_value(hand, target)
    if hand_value_current > target:
        return "STAY"

    # Simple strategy: stay if close to target.
    threshold = target - 4
    if hand_value_current >= threshold:
        return "STAY"
    else:
        return "HIT"
```
