The goal is to create a policy function for a simplified Blackjack-like game with a target score T ranging between 10 and 30. The policy must decide whether to "HIT" or "STAY" based on the current hand and target, aiming to approach T as closely as possible without exceeding it.

Key considerations:
1. **Card Values**: Properly handle Ace values (11 or 1) to maximize the hand value without busting.
2. **Decision Logic**: 
   - **HIT** when it's safe to do so (i.e., even the highest possible next card value of 10 won't cause a bust).
   - **STAY** otherwise to avoid the risk of busting with high-value cards.

### Approach
1. **Calculate Current Hand Value**:
   - Initialize total value and count Aces.
   - Add non-Ace cards first (face cards count as 10).
   - Optimally add Aces (use 11 if possible without busting, else 1).

2. **Make Decision**:
   - If current value equals or exceeds T, STAY.
   - If the maximum possible next card (10) can be safely added (current value + 10 ≤ T), HIT.
   - Otherwise, STAY to avoid busting.

### Why This Approach?
- **Safety**: By only hitting when the next card won't cause a bust, the policy minimizes risks.
- **Simplicity**: Efficient calculation with constant-time operations per decision.
- **Optimality**: Strikes a balance between approaching T and avoiding unnecessary risks, especially given no knowledge of remaining cards.

### Edge Cases
- **All Aces**: Handled by adjusting Ace values to avoid busting.
- **T = 10**: Conservative play prevents unnecessary busts when card values are low.
- **Already Busted**: Though rare, functions correctly by staying if the current calculation exceeds T.

This approach ensures the AI makes safe, rational decisions in line with standard Blackjack strategies, adapted for variable targets.

import numpy as np

def policy(hand: list[str], target: int) -> str:
    total = 0
    aces = 0
    
    # Calculate total for non-Ace cards and count Aces
    for card in hand:
        if card == "A":
            aces += 1
        elif card in ["J", "Q", "K"]:
            total += 10
        else:
            total += int(card)
    
    # Optimally add Aces (11 if possible, else 1)
    for _ in range(aces):
        if total + 11 <= target:
            total += 11
        else:
            total += 1
    
    # Decision logic
    if total >= target:
        return "STAY"
    if total + 10 <= target:
        return "HIT"
    else:
        return "STAY"
