To determine whether to `HIT` or `STAY` in this simplified card game, we need to balance the risk of busting (exceeding the target score) against the potential reward of getting closer to the target. The Ace card complicates this decision because it can be counted as **11 or 1**, depending on the current hand value relative to the target.

---

### 🔍 Key Concepts

- **Hand Value Calculation**:
  - The value of a hand is determined as the **maximum possible sum** that does not exceed the target `T`.
  - Aces can be counted as **11** or **1** depending on which value keeps the hand under or equal to `T`.

- **Busting Conditions**:
  - A player **busts** if the hand's value exceeds the target `T`.
  - Buster hands are unable to change the game's outcome, so it's best to `STAY` in such cases.

- **Hand Evaluation Logic**:
  - If the hand's value is **already above T**, you must `STAY` (you're already busted).
  - If the hand's value is **equal to T**, `STAY` is optimal because hitting risks busting.
  - If the hand's value is **below T**, we must evaluate the **expected probability** of busting if we draw another card. If the probability of busting is **below a threshold**, we `HIT` to try to get closer to the target.

---

### 🧠 Strategy Overview

We evaluate the game from two perspectives:

1. **Current Hand Evaluation**:
   - Compute the current hand's value using the Ace-adjusted sum logic.
   - If the hand is busted, we `STAY`.

2. **Expected Bust Probability When Hitting**:
   - For each possible next card (excluding those already in the hand), compute the probability of busting after drawing that card.
   - If the **probability of busting** is **low enough**, we `HIT` in hopes of getting closer to the target.
   - Otherwise, we `STAY` to avoid risking a bust.

```python
def compute_hand_value(hand, T):
    sum_non_ace = 0
    ace_count = 0
    for card in hand:
        if card == 'A':
            ace_count += 1
        else:
            if card in {'J', 'Q', 'K'}:
                sum_non_ace += 10
            else:
                sum_non_ace += int(card)

def compute_probability_of_bust(hand, target):
    current_length = len(hand)
    if current_length >= 13:
        return 1.0  # No cards left to draw, but can't get over target
    
    possible_new_cards = [card for card in "23456789JQKA" if card not in hand]
    total_valid = 0
    
    for new_card in possible_new_cards:
        new_hand = hand + [new_card]
        new_value = compute_hand_value(new_hand, target)
        if new_value <= target:
            total_valid += 1
    
    return 1.0 - (total_valid / len(possible_new_cards))

def policy(hand: list[str], target: int) -> str:
    hand_value = compute_hand_value(hand, target)
    
    if hand_value > target:
        return "STAY"  # Already busted
    if hand_value == target:
        return "STAY"  # Can't improve further
    
    bust_probability_if_hit = compute_probability_of_bust(hand, target)
    
    if bust_probability_if_hit > 0.5:
        return "STAY"
    else:
        return "HIT"
```

### ✅ Summary

This implementation evaluates the current hand value and computes the probability of busting if the player hits. Based on this, it decides whether to `HIT` or `STAY`, aiming to maximize the chance of not busting while getting as close as possible to the target.
