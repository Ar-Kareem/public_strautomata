To solve this problem, we need to create a policy function for the game Amazons on a 6x6 board. The function must return a valid move consisting of moving an Amazon (queen-like) and then shooting an arrow. The move must be legal according to the game's rules, which include moving along a clear path and shooting an arrow through a clear path.

### Approach
1. **Identify the Player's Amazons**: The function starts by identifying all the positions of the player's Amazons (value 1) on the board.
2. **Generate Possible Queen Moves**: For each identified Amazon, generate all possible moves the Amazon can make on the board. The Amazons move like queens in chess, which means they can move any number of squares in a straight line (orthogonal or diagonal) as long as the path is unobstructed.
3. **Simulate Move and Check Arrow Moves**: For each valid queen move, simulate the move on a temporary board. Then, generate all possible arrow moves from the new position of the Amazon. The arrow must be shot through a clear path similar to the queen's movement, ending on an empty square (value 0).
4. **Return the First Valid Move**: Return the first valid move and arrow shot that is found. This approach ensures that the function returns a valid move efficiently.

### Solution Code
```python
import numpy as np

def policy(board):
    for from_row in range(6):
        for from_col in range(6):
            if board[from_row][from_col] == 1:
                # Generate all possible queen moves for this Amazon
                possible_to = []
                directions = [
                    (0, 1),  # right
                    (1, 0),  # down
                    (0, -1), # left
                    (-1, 0), # up
                    (1, 1),  # down-right
                    (1, -1), # down-left
                    (-1, 1), # up-right
                    (-1, -1) # up-left
                ]
                for dx, dy in directions:
                    step = 1
                    while True:
                        new_r = from_row + dx * step
                        new_c = from_col + dy * step
                        if new_r < 0 or new_r >= 6 or new_c < 0 or new_c >= 6:
                            break
                        if board[new_r, new_c] != 0:
                            break
                        possible_to.append((new_r, new_c))
                        step += 1
                # Check each possible move for an arrow shot
                for to_row, to_col in possible_to:
                    temp_board = np.copy(board)
                    temp_board[from_row, from_col] = 0
                    temp_board[to_row, to_col] = 1
                    # Generate arrow moves from temp position
                    arrow_moves = []
                    for dx, dy in directions:
                        step = 1
                        while True:
                            a_r = to_row + dx * step
                            a_c = to_col + dy * step
                            if a_r < 0 or a_r >= 6 or a_c < 0 or a_c >= 6:
                                break
                            val = temp_board[a_r, a_c]
                            if val != 0:
                                break
                            arrow_moves.append((a_r, a_c))
                            step += 1
                    if arrow_moves:
                        arrow = arrow_moves[0]
                        return f"{from_row},{from_col}:{to_row},{to_col}:{arrow[0]},{arrow[1]}"
    # This return is theoretically unreachable due to problem constraints
    return ""
```

### Explanation
1. **Queen Move Generation**: For each Amazon on the board, the function checks all eight possible directions (like a chess queen) to determine all possible moves. Each move must be within the bounds of the board and land on an empty square (0).
2. **Temporary Board Setup**: For each valid queen move, a temporary board is created to simulate the move. This temporary board updates the positions of the Amazon and the original Amazon's position.
3. **Arrow Move Generation**: From the new position of the Amazon, the function checks all possible directions for shooting an arrow. This is done by checking each direction for empty squares (0) in the temporary board, ensuring the path is unobstructed.
4. **Move Selection**: The first valid move (queen move followed by an arrow shot) found is returned. This ensures the function returns a legal move efficiently without needing to compute all possible moves, adhering to the problem constraints and time limits.
