Here’s how the policy works:
1. Identify positions of my amazons on the board.
2. For each amazon, calculate all possible legal moves based on queen-like movement rules.
3. For each possible move, determine all valid positions to shoot an arrow.
4. Evaluate the resulting state of the board to prioritize moves that either capture an opponent's amazon or place arrows in strategic locations to block the opponent's future moves.

def get_amazon_positions(board):

def is_path_clear(board, start, end):
    sr, sc = start
    er, ec = end
    dr = (er - sr) // max(1, abs(er - sr)) if sr != er else 0
    dc = (ec - sc) // max(1, abs(ec - sc)) if sc != ec else 0

        c += dc
    return True
