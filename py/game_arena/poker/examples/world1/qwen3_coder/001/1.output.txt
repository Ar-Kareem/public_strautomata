The strategy for this poker policy is to make decisions based on hand strength and position while considering the pot odds and opponent behavior. Here's the breakdown:

1. **Hand Evaluation**:
   - In preflop (no public cards), evaluate the strength of the private card, prioritizing high ranks (A, K, Q, J) and suited cards.
   - After the flop/turn/river, simulate a basic hand strength calculation considering pairings, flush draws, and straight possibilities.

2. **Action Selection**:
   - **Fold**: Only when the hand is weak and the required call exceeds a threshold relative to potential pot winnings.
   - **Call**: When the hand has moderate strength or good pot odds justify calling.
   - **Raise**: With strong hands (top pairs, better) or strong drawing hands to build the pot or protect your hand.
   - **All-in**: Reserved for very strong hands (two pair or better) when there's confidence in winning.

3. **Aggression Management**:
   - Incorporate a simple bluff factor in certain situations (e.g., when having a high card and no public cards).
   - Adjust based on opponent spending (tight-aggressive approach): Don't get bullied by large bets without strong hands.

This strategy attempts to balance conservative play with calculated aggression based on mathematical odds and hand potential.

def evaluate_preflop_strength(card):
    """Heuristic evaluation of preflop card strength."""
    rank = card['rank']
    suit = card['suit']
    
    # High card value
    high_card_value = 0
    if rank >= 13:  # Ace
        high_card_value = 4
    elif rank >= 12:  # King
        high_card_value = 3
    elif rank >= 11:  # Queen
        high_card_value = 2
    elif rank >= 10:  # Jack
        high_card_value = 1
    
    # Add some value for suited cards
    suited_bonus = 1 if suit is not None else 0
    
    return high_card_value + suited_bonus

def evaluate_hand_strength(private_cards, public_cards):
    """Basic heuristic to estimate hand strength."""
    if not public_cards:
        return evaluate_preflop_strength(private_cards[0])

    ranks = [card['rank'] for card in private_cards + public_cards]
    suits = [card['suit'] for card in private_cards + public_cards]
    
    rank_counts = {}

        rank_counts[r] = rank_counts.get(r, 0) + 1

    unique_suits = len(set(suits))

    high_card_bonus = sum(1 for r in ranks if r >= 12)  # J, Q, K, A
